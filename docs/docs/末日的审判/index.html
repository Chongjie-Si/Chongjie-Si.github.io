末日的审判

Posted on 2021-06-10 In docs Views: Waline:

周日，阴雨的早晨。

早期智人伸了一个懒腰，揉了揉尚未睁开的双眼，抬起还没有完全清醒的脑袋，向洞口摇摇晃晃地走去。迎接他的不是往日的煦日和风，而是滂沱大雨。他慌忙躲进洞中。这下，他完全清醒了。

早期智人逐渐进化成了现代人类。

周日，阴雨的早晨。

现代人类伸了一个懒腰，揉了揉尚未睁开的双眼，抬起还没有完全清醒的脑袋，打开木制的门，摇摇晃晃地走出去。迎接他的不是往日的煦日和风，而是滂沱大雨。他慌忙躲进屋内。这下，他完全清醒了。

雨哗哗的下个不停，天上的窗口好像也打开了。连续几天雨一直下。四十天过去，雨还在下。终于，这场持续很长时间的暴雨停了下来。雨水淹没了平地，淹没了高山，禽鸟亦无踏足之地，一切生灵皆丧命于这场暴雨之中，唯独诺亚一家进了方舟，得以存活。

现代人类逐渐进化成了现代智人。

周日，阴雨的早晨。

“据最新消息，航拍卫星在阿勒山附近再次拍到了类似于诺亚方舟的遗骸，船身……”迈克教授正坐在临时搭建的大本营中避雨。“奇怪的天气，原来说是最近一周晴朗无云，天怎么说变就变。”迈克教授不免发发牢骚。

迈克教授是一名以色列考古学家，也是一名登山爱好者。他曾经征服了许多世界著名高峰，成功登顶珠穆朗玛峰的事更是被考古学同行们大加赞赏。按照人们的惯性思维，考古学家总是那种戴着眼镜身体瘦弱的人，但迈克教授却完全不同。

“这座山里拍摄到了诺亚方舟？笑话。现在的人们，总是喜欢将神话故事与现实混为一谈。”迈克教授不由自主的感叹道。考古多年，这类事情他见得多的去了。

雨终于停了。他带上装备，收拾好东西，又继续向上爬。此时此刻，迈克教授正在阿勒山上。他不知爬了多久，忽然在山上发现了一个洞口。于是，他便选择在这里稍作整顿。可在他进入洞口的那一瞬间，他目瞪口呆：洞中的墙壁上画的都是些什么！考古学家的本性流露出来，他忍不住走近观察，这，是原始罪恶么？难道这两个人是亚当和夏娃？这又是什么？他一幅接着一幅地看，直到最后一幅壁画。

洪水。

150天的洪水淹没了大地淹没了一切有生命的东西。他惊得说不出话。这些壁画，据初步判断，时间应该在几千年之前呐！他不免低下头沉思。忽然，他看见远处似乎有什么建筑物。他小步跑去。洞内越来越宽阔，前方的建筑也越来越明显。没错，是一个祭坛。他不禁想起《圣经》中的话：“诺亚为上帝建了一所祭坛。”

忽然，他大叫一声，应声倒地。

他是被人从身后击昏在地。

再次醒来时，天气是那么的晴朗。他揉了揉尚未睁开的双眼，抬起还没有完全清醒的脑袋坐了起来。迈克教授慢慢的回忆起自己的遭遇，他猛然起身，想去找那个洞口时，才发现自己正坐在阿勒山的山顶上。

“莫非，诺亚方舟的事历史上真的有过？那将是惊天动地的发现啊。”

而此时此刻，比起这些，他更享受站在山顶向下俯视的感觉。他再次出发，准备下山返回。然而当他下山时，他的脚步像是受到了什么力量的牵引，毫不犹豫的朝一个方向走去。

“我这是怎么了？”他自言自语道。“起来，向前走。”他忽然感觉到心里有什么声音在一直提醒他。终于，他停下了脚步。与脚步一起停下的，还有他的眼睛。

眼前的，正是迷失多年的诺亚方舟。

绕着尚可踏足的地方走了一圈，他不禁为这只方舟而震撼。当时的人们怎么会有如此的力量建造这么雄伟的方舟。他最终决定打开木门。歌斐木制的船身有些已经腐蚀。“飞鸟各从其类，禽兽各从其类，一公一母地带入方舟。”望着眼前一个又一个独立而精致的小室，他不由的感叹。

内部构造是多么精美啊！他想起了《圣经》上的话，“分三层”，于是他想，他看到的可能是第一层。一根又一根的歌斐木堆在一起，舟身居然还有透光处。他找到了类似于楼梯的构造，便爬了上去。

船身在缓缓的倾斜。

第二层与第一层结构相仿。唯一不同的是，第二层所分的空间比第一层更细致，也更狭小。

船身在缓缓的倾斜。

第三层也是如此，只不过，空间更大了些，倒像是诺亚的住所。

方舟笔直的坠落。

迈克教授正躺卧在他所进入的那个神秘山洞中，伸了一个懒腰，揉了揉尚未睁开的双眼，抬起还没有完全清醒的脑袋，摇摇晃晃地走出去。突然，他发现似乎有一个人，正在缓缓的走进山洞。那个人面貌模糊不清，但好像在盯着墙上的什么东西。他找了找身边可以用的东西，只发现了背包里的铁棍。那个人越来越近，直到他完全停下脚步。迈克教授悄悄地走到他身后，突然一棒挥去，那个人应声倒地。

迈克教授应声倒地。

迈克教授发现自己正躺卧在一条巨轮上。不，说是巨大的船也许更加合适。歌斐木制的船身摇摇晃晃。“这是在哪？”迈克教授四下里看了看，自己好像正坐在船的顶层。他慢慢的回忆起自己的经历，慢慢地想起自己击昏的那个人惊恐的脸——那不是他自己么！

“这是怎样的遭遇，我居然自己击昏了自己！”他顾不得去解释这些神秘的体验，连忙爬起来，向周围走动。

他找到了类似于楼梯的结构，便爬了下去。下面，都是飞禽走兽，各从其类。他的头突然有一点昏昏沉沉，“飞鸟各从其类，禽兽各从其类，一公一母地带入方舟。”他忽然想起什么，匆匆忙忙地爬上去打开了窗子。

天正在下着瓢泼大雨。

天终于晴了，他再次打开窗户，放出一只乌鸦去。那乌鸦飞来飞去，直到地上的水都干了。他又放出一只鸽子，要看看地上的水退去了没有。但遍地都是水，鸽子找不到落脚之地，就回到船里面。迈克教授等了几天又把鸽子放了出去，到了晚上，鸽子飞回到他那里，嘴里叼着一个新拧下来的橄榄叶子。迈克教授就知道地上的水退了。当他七天后再次放出鸽子时，鸽子就不再回来了。

地上的水好像都退去了。于是，迈克教授打开了船的窗子，船的门。

洪水汹涌着灌进船身。

“你醒了。”

“你还记得发生了什么？”

迈克教授站在一个雾气缭绕的园子里，不知所措。

“我记得我发现了一个神秘的山洞，进去之后看到了许多古老的壁画，然后就被人击昏了。”迈克教授停了下来，若有所思。“但那个人好像是我自己。”

“我醒来时发现自己正在阿勒山的山顶。在下山途中，发现了诺亚方舟……”

“你还记得什么？”那个声音打断了他，继续问道。

“我记得我又回到了山洞中，并且出于防备，击昏了一个人，但，我发现击昏的人好像是我自己啊。”迈克教授好像十分苦恼，他似乎找不到什么合理的理由来解释这一切。

“我好像又回到了方舟上，不过天正在下暴雨。等到雨停后，我打开了门，后来就不知道了。”

“你还记得什么？”那个声音继续问道。

“我说过了我记不清了。”

“你还记得什么？”那个声音继续问道。

“这是哪？”迈克教授没有理会那个声音，因为他的心中有很多的疑问亟待解决。

“这是伊甸园，独居的民最后的归宿。”

“独居的民？我怎么到这里来了？”迈克教授恨不得把所有的问题一连串的说出。此时此刻，他的心中充满了疑惑，他对自己所经历的一切感到十分恐惧。

他并不知道回答他的声音从哪里来，像是从天上，又像是从心里；他也不知道对他说话是谁。但他就是有一种感觉——那个人好像知道一切。

“独居的民是上帝的选民，他们终将万民之中分离开来。”

“你是谁？”迈克教授目瞪口呆而又惊慌失措的看着周围的一切。雾气从地上腾出，滋润遍地。四条河流从那地流出，环绕全地。

“他是谁并不重要，重要的是，你是谁。”

迎面走来一位白衣老者。他慈祥的笑容中又好像多了一份圣洁。“你是上帝的选民，你是那独居的民。万物的结局近了，所以你要警醒自守。”

“您是？”迈克教授忽然想起，他可能是一位先知。先知并没有回答他，他只是笑了笑，若有所思地说道，“到那时，兽的印记会标记在人们的身上，又会有各种的异象……”

“您能告诉我为什么我会在这里么，我现在一头雾水，我经历了到底是些什么？”迈克教授略带恐慌的打断了他。先知并没有生气，依然温和地笑着。

“今天是周日，是上帝的安息日。你闯入了上帝曾与诺亚立约的圣殿。确实是你自己击昏了自己，只不过被击昏的，是你已死去的老我。你现在是一个新造的人，上帝拣选了你。”

“你经历的一切，只不过是上帝的灵临到了你到身上，是你被上帝拯救的全过程。从今以后，你就是一个义人。因为时候已到，上帝必将你们从万民中分离开来。”

迈克教授听得不知所措，“您的意思是，上帝真实存在，而且还要毁灭万民么？”他有点难以置信，但事到如今，他仿佛找不到什么更好的理由去解释这一切。他忽然想起了曾经他的教授说过的一句话：一切问题的根源到最后都要用神学来解释。

“这是个罪恶的时代，人们放纵情欲与肉体，沉溺在罪恶之中。你们的仇敌魔鬼如同吼叫的狮子，遍地寻找可吞吃的人。这个时代不再会有诺亚，也不再会有方舟。”

迈克教授目光呆滞，他看着眼前的先知，这位老者，感到有些意外。他好像是……

他是在山洞中被自己击昏的人！迈克教授满脸惊恐，哆哆嗦嗦的伸出一只手指向先知，“你不是……”

“没错，我是你。我也是诺亚。”先知笑了笑，逐渐转身消失。

七位天使掌管着最后的灾，上帝大怒的七碗倒扣在地。

生命册缓缓打开，世界迎来了末日的审判。

那是一片新天新地。

人们再次发现迈克教授时，是在阿勒山的一个山洞里。令人匪夷所思的是，此时的迈克教授目光空洞。而且在山洞里，人们还发现了另外一具没有头颅的尸体。与此同时，人们还走进了方舟中，并进一步确认了这就是神话故事中的诺亚方舟。

迈克教授仍然目光空洞的待在山洞里。几天没有进食，但他的生命特征仍然明显。

“你们看墙上！”一名科考队员吃惊的喊道。

迈克教授突然间笑了笑，他飞快地奔向洞内，来到祭坛前。

山洞轰然倒塌。

再也没有方舟的出现。

天开始下起了一场永不停息的雨。
