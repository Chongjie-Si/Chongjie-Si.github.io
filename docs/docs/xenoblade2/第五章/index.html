异度之刃2

500年前，反抗军营地。

一个手持长杖的女子正与一名身着白色斗篷的男子战斗。男子手持一把金色的剑，与女子打的有来有回。

“出色的动作，劳拉，即使预测到你的动作，你也能瞬间反击。”男子称赞道。

“因为有霞的力量才能做到。”劳拉甩着长杖再次与男子交手，“你不也一样，没用出光的绝招吗？”

“如果我用了呢？”

“到那时，我也会用出真的。”

两人打的难舍难分，丝毫没有留意到远处走来的另一个异刃。

“阿德尔大人，”真走了过来，“餐点已经准备好了。”

阿德尔停下了手，“都这么晚了吗？”

“这个季节的白天很长，”劳拉说，“让人感觉不到时间的流逝。”

坐在篝火旁，众人吃着真做的晚饭，休息着。一个很像佐田彦的金发的小孩子坐在光的旁边低着头，始终一言不发。

“哎呀，训练已经结束了？真遗憾，我本来还想参加呢。”迦具土笑着走向众人。

“姗姗来迟，就你一个？”光问。

“皇帝陛下政务繁忙，何况我原先就经常单独行动。”

“御刃者和异刃一心同体，这才是正确的方式哦，不然是发挥不出力量的。”光笑着说。

“实际总没有说起来简单，”阿德尔说，“我都还没有彻底发挥出你的力量呢。”

“是吗，我倒是竭尽全力了。”

“我是指你潜力无穷啦。不过话说回来，霞你抑制异刃行动的力量，倒是挺适合用来修行的，”阿德尔转向霞，“实在是帮了大忙。”

“连因果律预测也受到了限制，真的很不好对付。”光说。

“这真是……”霞不好意思的说，“能让阿德尔大人和光小姐如此看重，实在让人不胜惶恐。”

光看着霞和劳拉，笑着说，“两个人在一起，简直就像是双胞胎姐妹嘛。”

“很少能看到这种御刃者的因子体现得如此浓烈的情况吧？”

劳拉看着霞，问，“确实有时会听到这种说法，是这样的吗？”

“谁知道呢……”霞的手指搭在嘴唇边，随即笑着抱着劳拉的胳膊，说，“但对我来说能和劳拉大人拥有相同的外表是件很幸福的事情。”

“这样衣服还能互相换着穿呢。”

“是啊。”

“啊哈哈……”

“说起来——”阿德尔开口道，“我希望下次能和真切磋一下。”众人看向真，

“听到了吧，你的意见呢？”劳拉问。

“我——不喜欢战斗……”真点了点头，随即走远。

“‘伊拉最强的男人’的话，真是有份量啊。”阿德尔说。

“也许这就是真相呢，”劳拉目送着真走远，“他……不，是他们的……”

芳和霞

巨神兽船稳稳的在云海上飞行着，夜幕降临，霞一人站在船头看着茫茫的云海。光站在霞的身后一脸忧愁，想说的话却始终说不出口。光想起在废弃工厂的港口处，击退了伊拉之后……

“你是？”莱克斯问。

“我是芳·勒·诺伦，阿卡狄亚教廷的使者。”芳温柔的回应着。

“教廷来的？”

“您便是莱克斯先生吧？麦佩尼圣座希望能与您见面，可否随我同行？”

“咦，”莱克斯有些惊异，“这是什么意思?为什么教皇大人要见我这种人啊。”

“教廷管理着整个幽界的御刃者及异刃，而您是天之圣杯的御刃者。”

“是想将那份力量化为己用吗？”梅勒芙问。

“并非如此，”芳转身解释着，“圣座他是出于‘同为天之圣杯的御刃者’的理由，想与莱克斯先生见一面，仅此而已。”

“同是——？难道？”

“没错，圣座是灭的御刃者。另外，莱克斯先生，你现在身上带着一把短剑对吧？”

“你为什么会知道短剑的事？”

“我当然知道，因为圣座便是短剑原本的主人。”

“啊！”

“这么说，教皇大人就是考尔爷爷的御刃者？”尼娅问。

“是的——”

“怎么可能——考尔爷爷可是……”

“现任教皇是阿卡狄亚人，”梅勒芙说，“所以也并非不可能。”

“阿卡狄亚人可都是很长寿的。”白虎补充道。

“是吗，”莱克斯说，“所以当时考尔先生才会说短剑的主人知道前往世界树的方法。”

“您意下如何？”芳问莱克斯，“觐见麦佩尼圣座对你来说应该也不是什么坏事。”

“啊，嗯……等，等一下——”

“太可疑了莱克斯，哪有这么好的事。”尼娅不信任的说。

“光你怎么看？”莱克斯问光，“你见到考尔先生那时候起就已经发现了对吧？”

“嗯，”光点头说，“只不过没想到他居然成了‘教皇大人’而己。”

“你想见他吗？”

“我可是有一大堆事情要问，想知道我沉睡之后，到底发生了什么。以及，”光看着霞皱着眉头不解的说，“为什么你现在会出现在这里……”

“明白了，”莱克斯说，“我也想见见他——不，我觉得必须要见他一面才行。”

船上的光犹豫了半天，最终忍不住开口道：

“霞……”

“光小姐？这种时候找我有什么事情吗？”芳回头一看，发现了光，“现在这个季节，云海的夜晚很冷的，小心得感冒哦？”

“我是异刃，不会得感冒的。”

“是这样啊，请原谅我的失礼。”

“霞……你已经不记得了吗。”

“？ 为何要叫我霞呢？我是芳·勒·诺伦，是麦佩尼圣座的异刃。”

“是吗……麦佩尼的……抱歉，忘了我的话吧，因为你和我认识的一个人很像，不小心喊错了。这里的确是挺冷的，我还是听你的忠告，回房间去吧。”

“好，祝你晚安，光小姐。”芳蹲下身向光致意。

“晚安，芳。”

在光的脑海中，焰对光说：

「小光，那个核心水晶的形状……」

「我知道，那可不是简单的调谐。」

「和我们还有莱克斯都不一样。」

「要问的事情又多了一件呢……抱歉，我有点事情要想，和我交换吧。」

「嗯，知道了。」随后，光变成了焰。

独角鲸号内，真按下按钮，一扇门缓缓打开，数以百计的核心水晶露了出来。

“成功了吗？”灭问。

“嗯——异刃们的声音就在这里。”

“来吧，已经找到了有趣的东西。”

“知道了。”

在巨型屏幕前，灭手指着说：

“来看这个。”

“檀佩兰缇亚。”义经说，“斯佩比亚正在那里进行大规模的发掘工作。因为有这样的传闻，就去看了看。”

“虽然变了很多，但看这个外形，是尤迪肯姆的巨神兽兵器吧。”真凝视着屏幕上身披硬甲的大型巨神兽。

“尤迪肯姆？我记得那是？”

“嗯，是500年前我和光击沉的国家。同时，也是食人种的发源地——”

“食人种？”义经诧异道。

“怎么办？”灭问真。

“如果会成为我们的威胁，就将其排除。不过，还有另外一种利用方式——”

“啊！该死！受不了了！”弁庆的声音响起。

“欢迎回来。”义经笑着说，“火气还挺大的嘛。难道是——？”

“交易方面没有问题。”佐田彦回答说，“东西都顺利带回来了。”

“那就好。所以你生气的原因是什么？”义经看向弁庆。

……

“——就是这样了，实在难堪，只能夹着尾巴逃回来了。”佐田说。

“并不是因为以太场被搅乱了吧？”

“教廷的女异刃吗……”灭说。

“真，”佐田彦严肃的看着真，“那是霞，毫无疑问。”

真扭过头没有说话。

“真——？”

“佐田彦，经文的调整做的如何了？”

“哦，那个没问题，虽然比预定要晚一些，但我会想办法的。”

“义经，天之圣杯的位置呢？”真转身看向义经。

“在利贝拉里塔斯岛屿群附近，像是坐着船还是什么的，正在进行移动。”

“目的地应该是阿卡狄亚。”说完，真转身就走。

“咦？你去哪里？”

“檀佩兰缇亚。去争取时间。”

“不管天之圣杯了吗？”弁庆问，“要是和那个男人勾结起来——”

“不可能。”灭打断，自信的说，“光不可能做那种事。”

“她反而会引导我们。”真还是一如既往的冷漠，“通往世界树的道路已被封闭，只有天之圣杯才能打开。灭，从商会那里接收的东西需要调整，你们去给佐田彦帮忙，檀佩兰缇亚就由我一个人去。”

“这倒没关系。”灭回应。

“我知道了，请多加小心。”义经说。

随后，真向着黑暗中走去。

“那家伙……”灭注视着真，渐行渐远。

弥先村

一行人乘着船向阿卡狄亚前进，船下茫茫的云海无穷无尽。

“真棒啊，云海简直就像是山一样。”焰说

“这是利贝拉里塔斯岛屿群，他是季节性的，阿卡狄亚就在那云之山脊的另一头。与其绕路，不如走陆路比较快，所以我们就只坐船到这里。”芳解释道。

“即使是巨神兽船，也无法跨越那条山脊吧。”白虎问。

“看来是不得不走陆路了。”梅勒芙感叹。

“真让人怀念啊——”莱克斯双手叉着腰，笑着说。

“嗯，都已经5年了啊。”青龙爷爷趴在莱克斯的头盔里补充。

焰不解地问：“这里是让你们怀念的地方吗？”

“弥先村——”

“我和爷爷的故乡，就在那条山脊的对面。”

“哦，你是利贝拉里塔斯出生的啊。”尼娅说。莱克斯点了点头。

“原来是这样啊。”焰的眉头舒展开来，露出笑容。

巨神兽船逐渐向港口靠近，远处一座座的小岛屿彼此连接，构成了利贝拉里塔斯岛屿群。

“好了，接下来还是让我来带路吧。”上岸后，莱克斯面向众人说，“去山脊对面就可以了吧？”

“是的，”芳回应道，“到那里应该就会有教廷的人来接我们。”

“了解。”

“大哥果然厉害，真是可靠嗼。”虎扇动着他的大耳朵，跳着夸赞莱克斯。

“嘿嘿嘿……”

“嘿嘿嘿什么嘿”，爷爷忍不住吐槽说，“只是去那些漂浮的群岛而已吧。”

“有些路线可是很危险的啊”，莱克斯解释，“有可能会被厉害的怪物袭击。”

“唔，这倒也是。”

“要怎么去嗼？”

“嗯——”

“弥先村呢？”焰问。

“弥先村？”

“会路过吗？”焰再次追问。

“是有这么条线路。”莱克斯回应道。

“就这个，我想走这条线路！”焰身体前倾望着莱克斯，眼里满是期待。

“你先站直了再说。”尼娅吐槽道。

“怎么突然就变得这么积极了……”白虎说。

“目的改变了嗼。”

“但是，你们不想看看莱克斯的故乡吗？”焰小心翼翼的问。

“啊，这倒确实想看看。”尼娅撇过头，故作清高的回应。

“好吧，那条路线算是比较安全的，倒也还不错，怎么样？”莱克斯看着芳，询问道。

“交给莱克斯先生决定。”芳还是一如既往的温柔的说。

“好！那就定下来吧，做好准备就出发。”

“噢！”

“不过，为什么连你也在这里？”尼娅扭头一脸嫌弃的看着梅勒芙和迦具土。

“我作为斯佩比亚皇帝的钦差，提出了同行的要求，有什么问题吗？”

“这是理所当然的，”芳连忙打圆场，“既然要将天之圣杯请到教廷，那这个情报，就必须要向所有国家公开。”

“说到底，你们的立场还是中立、公正的啊。”青龙爷爷问。

“是的。”

“就是这样啦，请多关照，尼娅。”

“作为一个被你们关进牢房的人，和你们在一起让我感觉很不爽。”尼娅仍然是一脸嫌弃。

“大小姐，请不要这么说，”白虎在尼娅的身旁说，“有梅勒芙大人和迦具土大人同行的话，旅途的安全就可以高枕无忧了。”

“这个我当然知道啊。”

“有什么大不了的嘛尼娅”，莱克斯笑着说，“误会都解除了，之前不是还救了我们吗？”

“真羡慕你能这么心大。”

“乐观——”

“你是想说我乐观对吧？”尼娅打断莱克斯，挥舞着拳头十分无奈，“好好好，之前的事就都一笔勾销吧。”说着，她走向梅勒芙，伸出手，“嗯，握手言和。”

“我这边也要请你多照应了。”梅勒芙笑着也伸出手，和尼娅握在一起。

一行人向着弥先村进发，他们途经了一座座岛屿，也路过了一条条隧道。利贝拉里塔斯是由一座座小型阿鲁斯组成的岛屿群，远远看上去，一座座小岛悬浮在云海上，充满了童话的气息。

在一处山脚下，斯佩比亚的士兵们全副武装的直立着，他们的面前是一只披满了铁甲的阿鲁斯。

“比报告里提到的还多啊。”罗德里希议员看着手中的报告，对身旁的士兵说。

“驱动系统已经换上我国最新的产品了，而控制系统都保养的很好，可直接使用，应该用不了多长时间就能凑齐。”

罗德里希放下手中的报告，仰望着天空，“在古拉一战后衰落的我国，终于能够逆转数量上的劣势了吗……”

“是！”

“稳定性方面没有问题吧？3个月前的爆炸事故——不许有第二次哦？”

“已经在巨神兽的体液中注入了镇静剂，不会妨碍到平时的运用。另外，身体后部已覆盖特殊钢材，即使遭到敌方炮火直击，也可以承受很久。万一被引爆的话，如果是在敌阵之中……”

罗德里希看着士兵，“就和对方同归于尽吗——”

“我们设置了搭乘人员的逃脱装置。”

“你确定那不是——棺材？”

“这个……”

“哼——”

在士兵的陪同下，罗德里希登上了巨神兽的背部。

“将食人种技术的精华登峰造极的亡国——尤迪肯姆，它残留的渣滓，真是让人感到不快……”

“我等很清楚，阁下对将人类细胞与异刃组合的食人种一直感到排斥——”

“并非如此。”

“不是吗，那到底是……”

“大家都说，食人种诞生的目的，是为了制造更强大的异刃，但事实是否真是如此呢？我无法不认为，这只是人类为了活得更久而作出的徒劳挣扎。简直如同在看着镜中的我们，我不喜欢这一点……我去一趟阿卡狄亚，必须先打点一下才行。”

​“真是个不可思议的地方啊，云里居然有隧道。”焰感叹说。

“因为这里是小型巨神兽聚集的群岛，”莱克斯解释道，“巨神兽与巨神兽之间连接的力场创造出了隧道。”

“在云之山脉的中心，也有大型的巨神兽，”爷爷补充说，“不过，那里是云海内部，不适合人类居住啊。”

“大型巨神兽和围绕在其周围的小型巨神兽是这里的特征。”

“嗼嗼，世界真是千奇百怪嗼。”

“主人，偶尔出去走走比较好嗼。”花小声的提醒着虎，“总是窝在房间里摆弄机器，对身体很不好嗼。”

“现，现在不就和大哥他们一起在冒险嗼，我才不是家里蹲的自家警备员嗼。”

“是啊，现在的虎可是个出色的御刃者呢。”莱克斯给予了肯定。

“嗼嗼，大哥就是大哥，好懂嗼！”

一行人嬉笑着，逐渐靠近了弥先村的海峡。

“哦，他们正在玩呢。”莱克斯看到远远的海角处，一群小孩子们正在排队跳入云海。

“那是在做什么？做这种事情就不会害怕吗？”尼娅感到很诧异。

“在这一带是很平常的游戏啦，都是那样学会在云海里游泳的。”

“莱克斯也会这样玩吗？”焰问。

“当然了，在成为打捞员的时候，这个可是帮大忙了。”

“我就不行了，那种被纠缠的感觉和水可不一样。”一想到自己在云海里挣扎的场面，尼娅就忍不住打了个哆嗦。

“而且也不擅长站在高处呢。”白虎撇着嘴。

“只要跳下去很快就会习惯的啦，要不现在就来试试看～？”莱克斯一脸坏笑的看着尼娅。

“等等，你干什么，别过来，别碰我！”

“真是的，莱克斯，不许捉弄尼娅哦。”焰叉着腰歪着头看着莱克斯。

“知道啦。”莱克斯挠着头笑着说。

“真是的，还是那么孩子气。”

尼娅一边吐槽着，一边和大家一起向前走着，不一会，便来到了莱克斯的故乡——弥先村。村庄里的一些小孩子们发现了他们一行，也发现了莱克斯。

“哥哥——哥哥——”

“真……他的过去，到底发生了什么……”
