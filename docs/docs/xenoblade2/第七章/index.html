阿卡狄亚的教廷高台上，麦佩尼站在围栏旁，思绪飘向了远方。曾经，他的手触碰上了紫黑色的核心水晶并成功与之调谐。一团黑雾中，灭缓缓地睁开了眼睛……

“麦佩尼圣座——”卫兵的呼唤打断了他的思绪。

“何事？”

“有关前往洛修利亚的，天之圣杯一事，我有事要禀报——”

“无妨，进来吧。”

“打扰了——”

第三把剑

莱克斯做了一个梦，在梦里焰被真打伤了无数次，而每一次他都只能眼睁睁的看着焰为了救他，选择牺牲自己。苍白的无力感席卷而来，枕头也被不清醒的泪水打湿。他只能眼睁睁的看着焰倒在自己的怀里，直到失去呼吸。

晚霞透过旅馆的窗，照在躺卧在床上的莱克斯身上。莱克斯睁开了眼，颓废地坐在床上。手臂上缠绕着绷带，裹住被真划出的伤口，也裹住自己内心的恐惧。窗外的鸟啼叫着盘旋着，像是那么的自由。莱克斯低下了头，手中原先锋利的剑如今只剩下了破碎的剑柄，仿佛一切与焰和光的经历都未曾发生过，只有胸前的核心水晶证明这一切并不是幻梦。眼泪啪嗒的掉下，他拭去眼角的泪水，沉寂于暖金色中。

屋外，众人坐在旅馆的桌子前沉默不语。一路走来，个中艰辛只有经历的人才知晓。好不容易走到这里，眼看着就已经拥有前往的世界树的方法，但却不得已在此止步。真，太强了，这是不可争辩的事实。对于现在的大家而言，真就是一道不可逾越的山。不甘、愤怒和无奈的情绪萦绕着大家，门外的风吹的更猛烈了。

“——大哥还没醒嗼，都已经2天了嗼……”

“毕竟他身受重伤，”白虎趴在地上说着，“即使没有生命危险，他的身体也需要休息。”

尼娅手托着腮撑在桌子上，眼里满是担忧。“他没事吧……”

这时，零碎的脚步声从屋内传来。莱克斯背着一个布袋，低着头，不敢看着众人。

“——莱克斯——那是什么……”尼娅疑惑地问。

“回去吧……”

“啊？回哪里去？”

“怎么办好呢……爷爷也变得这么小了，只能回阿伐利缇亚了吗……”

爷爷坐在椅子上，一脸不可思议的望向莱克斯：“莱克斯——你……”

“只能回阿伐利缇亚啊——算什么啊！”尼娅着急的问，“焰怎么办？你理解现在是什么情況吗，你应该不是失忆了吧？”

忧郁的情绪从莱克斯的眼睛里满溢而出，他轻声的回应着尼娅：“我有记忆啊——记得清清楚楚……”

“那——”

“算了……”

“什么算了——”

“都说我是天之圣杯的御刃者，我也当自己是个人物，头脑过热，以为自己无所不能——”莱克斯痛苦的摇了摇头，“可其实，我什么也做不到。我做到的，就只有让焰受伤……”

梅勒芙低着头，不知在想些什么；齐格背靠在窗边坐在椅子上，静静地听莱克斯继续说道：

“所以，还是算了吧……”

“怎么可能就这么算了！”尼娅抓着莱克斯的肩膀，“她可是为了保护你才被抓的。”

“我知道……可我做不到！”

“什么做不到？”

“我怎么可能打得过那么厉害的人，凭我根本赢不过他——”莱克斯甩开尼娅的手，低沉着头无力的说，“我现在终于明白了，和他们在一起，焰也不会再受伤了，所以，我要回去了！”

“你——”

“本来就是不可能的事，乐园——还有御刃者——我还是更适合当打捞员——”

“你这——混蛋！”尼娅一拳将莱克斯打倒在地上，摆放整齐的木桶瞬间被打乱，在地上滚动着。莱克斯捂着脸，疼痛感似乎没有想象中那么强烈。尼娅自责的看着莱克斯，他不过是一个不成熟的少年，却要独自承担着压力与悲伤，甚至是拯救幽界的使命。尼娅的心里渗透出了后悔，但是她狠下心，继续说道：

“疼吗，焰心里的疼痛可不只是这种程度——”尼娅抓着莱克斯的衣领生气的看着莱克斯，“她忍耐着数倍的疼痛去了他们那边，就是为了救我们——”

“这些我都知道！”莱克斯吼着打断了尼娅，“可我还能怎么办呢！我的武技对他们完全没用，仆从的攻击也被弹开，对那家伙完全无计可施啊……”莱克斯背过身冷静了下来，悲伤的说，“我越是战斗，越是让焰和光受伤——”迦具土走上前，想安慰莱克斯。可是，莱克斯的话传入她的耳朵：

“那我还是……不在她们身边比较好……”

迦具土愣了一下，随即坚定的向莱克斯走来，一个响亮的巴掌声在屋内响起。莱克斯捂着脸看着迦具土，满脸不知所措。

“你真的这么想吗？你知道她是抱着怎样的心情离开了你的身边吗！”在弥先村，迦具土清清楚楚的听见了焰说的那句：“但是，我担心如果留在莱克斯身边，会不会害他失去更多的东西。那么，我选择离开的话，无论是莱克斯，还是世界，都一定会更幸福……”

“她甚至说出自己还是消失比较好，你却没有任何想法吗！”迦具土生气的指责着莱克斯，而莱克斯静静地低着头，一言不发。“她是打算……和他们同归于尽啊！让她做到这个地步，你却——你想和她一起去乐园的意志，就只有这种程度吗！”

“好久不见啦，爸爸——妈妈——我成为了御刃者哦，很厉害吧？”莱克斯猛的回想起在自己父母的墓碑前，他向父母介绍着焰，“然后，这个人是焰，是我的搭档，也是我非常重要的人。我己经决定了，要和焰一起去乐园，今后不知道会发生什么事情，请你们保佑焰和我吧。”

莱克斯痛苦的低着头，紧咬着牙关。

“够了，我们自己去救她吧。”尼娅走上前，“这种人，才不是我们认识的莱克斯！他愿意回哪里就回哪里去吧——”

白虎起身，迈着步子边走边说，“各位，也不必说到这个份上——”他还没有走多远，坐在椅子上的梅勒芙伸手拦住了他，说，“这里就交给女士们吧。——嗯？”随即，她看向窗边的齐格。齐格没有说话，只是摊了摊手。

莱克斯一蹶不振的喃喃道：“我……”

“就是嗼，”花生气的向莱克斯靠近，“现在的莱克斯，一点也不像莱克斯嗼！只不过输了一次就放弃，实在是太没出息了嗼！”

“可是——”

“不要再东拉西扯了嗼！”花突然收紧了拳头，钢铁制成的手有着千斤的力气。

“花！”虎惊吓的跳了起来，梅勒芙也连忙从椅子上站起。莱克斯闭上了眼睛，然而想象中的痛苦并没有袭来。

“花……”

花轻轻的戳着莱克斯的脑袋，随后温柔的抚摸着莱克斯的脸，说：

“花不想看到这样的莱克斯嗼……”花略带哭音的说，“花和主人，都不是真正的嗼，所以一直都很憧憬莱克斯你们嗼。莱克斯至今为止一直非～常努力嗼，因为莱克斯这么努力，花和主人才能一起努力嗼。莱克斯是花和主人的目标嗼，希望你能一直一直向前跑嗼……”花抽泣着， 闭上了眼睛，“所以，不要停步在这种地方嗼……”

“花……”

彩歌从门外小跑着进来，趴在齐格的耳边，“王子……”听闻彩歌的话，齐格站了起来。

“小子，老爸在叫你呢，去王宫吧。”

皇宫的王座前，塞里希抱歉的看着莱克斯。

“你的伤势没问题吧？”

“是的，已经没事了。”

“抱歉，若非我妄图抹除天之圣杯，那些人便不会……”

“即使放任不管，那些家伙也会出现在焰面前，”莱克斯摇了摇头难过的说，“这不是国王的错，是我——的错……”

塞里希望着眼前的少年，缓缓开口：

“500年前，国家曾经险些一分为二。一派是以与巨神兽及异刃自然共存为目标的阿德尔派，另一派是坚持人类统治，意图利用巨神兽的反阿德尔派。阿德尔亡故后，反阿德尔派利用偶然得到的海蛇控制核心——三圣颂锁链，兴起了洛修利亚王族。”

“并且自诩英雄阿德尔的后裔——”

​“正是。祖先为掌握人心，利用了阿德尔的名号……”国王闭上了眼睛，历史浮现在眼前。“最终，洛修利亚为躲避他国的干涉，隐遁至云海深处，一直保守着自身并非阿德尔血脉的秘密。而教廷也对此默许，你明白其中原因吗？”

“不。”

“洛修利亚被迫约定每年供奉一定数量从云海得来的财富——核心晶片。有一样东西想给你们看看。”

“给我们？”

皇宫内一扇石墙缓缓落下，五根管柱内，淡蓝色的能量穿梭跳动着。

“这是——”

“是玄武体内流动的以太。巨神兽从云海摄取供给活动所需的能量，活动能量化为以太流，流淌在全身之中，经过精炼与结晶化，形成核心晶片。此处下层即为核心晶片的生产设施。”

“在这里取出一部分巨神兽活动必须的能量吗？”梅勒芙敏锐的察觉到了中间的关联，问。

“洛修利亚的低温，以及作物难以生长，皆是因此而起。”

“原来是这样。”

“核心晶片对于国家能源政策及军事政策不可或缺，因此教廷想要它们。”

“目的是什么？”尼娅问。

“当时的幽界极其凋敝，然同时也是扩大自国影响力的绝佳机遇。”

“所以才——”

“最终，洛修利亚陷入慢性核心晶片不足，之后就如你们所见……严寒的气候导致作物无法茁壮生长，洛修利亚的国民不得不过着饥寒交迫的生活。”国王愧疚的表情浮现在脸上。

“你们为什么不浮出云海？”尼娅接着问道，“既然可以自由移动，明明可以去更暖和的地方啊。”

“是因为畏惧与他国交流吧，”梅勒芙说，“和他国交流就有可能被发现阿德尔传承的虚假。”

“因此才闭关锁国吗。”

“正是如此。而后过了500年，天之圣杯觉醒，教廷逼迫我们供出神圣锁链，还说若是拒绝，便将洛修利亚的历史公之于众。”

“阿卡狄亚竟然——”莱克斯惊讶的看着塞里希，“怎么会——”

“那封亲笔信的内容，原来是这个吗？”齐格也理解了为什么塞里希看到文书后会那么生气。

“我已经受够了，我无法忍受洛修利亚和国民再为贫穷所困，因此我做出了决断。”

“所以才会将焰……”

“正是。天之圣杯有可能化为教廷之剑，然只要封印那力量，我们尚存抵抗教廷之机。只要宣扬能脱离如今的生活，过去王家犯下的罪孽必定也能获得宽恕。”

尼娅问，“们没想过利用圣杯的力量吗？”

“那力量不是我们能承受的，至少我明白，自己没有那样的资格。”塞里希说，随后看着莱克斯，“然而对你而言，是否也是如此呢？那时——看到你的眼神时，我便确信了。若是你，或许可以托付天之圣杯的力量。”

“老爸……”

“但就结果而言，只是令事态进一步恶化。都怪我的想法太肤浅……”塞里希陷入深深的自责中，随即，他从身后拿出一本书，说：

“这本书记录着英雄阿德尔相关之事。据其记载，阿德尔使用白之剑拯救世界免于灭亡，并与赤之剑一同消失了踪影。”

“白与赤，是指光和焰的剑吗？”

“写下这本书的，估计是阿德尔身边的人吧。作者出于好奇，在阿德尔消失踪影之后，踏上了寻找某物的旅途。”

“某物是指？”

“天之圣杯第三之剑——”

“第三之剑？”莱克斯瞬间打起了精神。

“据说那柄剑如金刚石般通透，绽放着清澄光辉。阿德尔却毕生没有使用这柄剑——不，是他无法使用。”

​“为什么不用呢？”

“据传，即使是英雄阿德尔，也无法承受那惊人的力量激流。阿德尔因畏惧剑力，将其封印在了某处。作者还如此写道，那柄剑才是真正的天之圣杯之剑。莱克斯，你败给了真，那个男人的力量确实强大。然而，真是如此吗，天之圣杯的力量，难道还不如一介异刃吗？是否可以这么认为呢，败北是由于你没能完全引出天之圣杯的力量。”

“我没有引出她们的力量……”莱克斯低头陷入沉思。半晌，他抬起头，眼神变得清亮而坚定。

“国王，那把剑在哪里？我想得到那把剑，得到那把剑，用它拯救焰和光！”

塞里希看着莱克斯，犹豫不决，但最后开口道，“很遗憾，书上并未记载剑的所在之处。”

“怎么会……”

“不过上面写着剑或被封印在利贝拉里塔斯岛屿群的某处，似乎是作者的推测。尽管困难，但若能找到它——”

“利贝拉里塔斯——”

“你有觉悟吗？“爷爷从莱克斯的头盔里飞了出来。

“咦？”

“我问你有没有觉悟成为那孩子真正的御刃者？”

​“爷爷？”

“究竟有没有，莱克斯！”

“有！当然有，我想成为焰和光真正的御刃者，然后，带她们去乐园！”

“那就跟我来吧，我来给你带路。”

众人纷纷离开了皇宫，塞里希的声音从齐格身后传来：

“齐格。”

“什么事？”

“我以王的身份请托于你——”

“真稀奇，该不会下雪吧——哦，早就已经下起来了啊。”

“我任命你护卫天之圣杯——不，护卫那个少年。”

“莱克斯吗？”

“嗯，我相信那少年的眼神，我们不能失去那光辉。”

“嘿，你真够看重他的——好吧，包在我身上。”齐格说着，便转身离开。

“拜托了……”

“反正我最开始就是这么打算的。”齐格摆了摆手。

独角鲸号里，两只人造异刃抓着焰的手臂，将其牢牢锁住。

“说起来，女性真是可怕，”义经说，“看起来这么文静，却说什么消灭自己。若这么做，会令那个少年同时丧命。”

“这全都在她的计算之中。”灭说。

“原本就只是虚张声势？”

“不是。她将核心水晶分给那小子，是为了替代被真破坏的心脏。天之圣杯的核心，记录着所有生命的信息，她就是利用了这一点。”

“好厉害，竟然能做到这种事——果然和我们不一样。”

“核心随时记录着外来的情报。他们伤口的关联，说白了也是介于同一个核心的情报交换，某种意义上，确实算是生命联系在一起。但到了关键时刻，也可以切断其间的联系。”

“你的意思是？”义经转身看向灭。

“她打算在对仆从下达攻击命令之前，将剩余的核心也交给那小子，这样那小子的心脏便能完全再生。天之圣杯即使失去核心，也能继续存在一阵子。尽管不知道原因，但我们的构造就是如此。”

“真也是明知如此，才接受了她的提议。”

“那小子是死是活都无所谓，可是我们不能失去这家伙，仅此而已。”

齐格的遭遇

众人回到弥先村，这里还是一如既往的安静祥和。

“来村子里做什么？”莱克斯问。

“你知道弥先村的别名吗？”爷爷说。

“诶？”

“英雄之村——”

“英雄——咦?”在此生活了快十年的莱克斯第一次听到这个名字。

“没错，弥先村是流浪至此的阿德尔建立的村庄。大战之后，阿德尔将守护村子的任务交付于我，我也在此住下。”

“这村子还有这样的秘密？”

“你认为我是在守护什么？”爷爷问。

“难道是——”

“走这边。”

爷爷带着众人走到一个石像面前。

“这不是守护神吗——”莱克斯看着眼前的石像。从有记忆开始，这个石像就一直存在于此。
