异度之刃2

在一团漆黑的迷雾中，独角鲸号的船门发出剧烈的响声，缓缓打开。

“欢迎回来，真。”

“义经吗？”一个带着面具的白发男子和身旁黑色盔甲的肌肉男走了过来，与此同时，另一个身背长剑的男子继续说道，“天之圣杯的觉醒，失算了？或者是——也罢，反正在我的剧本里，一早就有这样的剧情。”这个男子推了推自己的红色眼镜，一副早就预料的样子说。

“事后就马后炮个不停，话说，义经你这人真是小人风范啊～”义经的异刃从二层舷窗飞下来，在义经的周围飞着。

“你好烦啊，神威。”

“哎呀，我好怕怕呢。”义经推开神威，将其推到身后。

“那两个人呢？”灭问。

“佐田彦去斯佩比亚的‘工厂’视察，我可爱的妹妹和往常一样在狩猎，肯定又有无数御刃者成了她的手下亡魂。”

“狩猎千刃的事吗，”灭抱着双臂，“又不是收集到越多核心就越好。”

沉默的真开口问，“知道圣杯现在的位置吗？”

“你以为你在问谁，当然尽在掌握。神威——”

“好啦好啦。”神威伸手托起一块光粒地图，地图中赫然可见几只巨神兽。

“现在位于古拉领地，似乎正在离开城镇。”

“离开城镇？那个方向什么都没吧。”

“也许打算直接离开那里吧。啊，对了。”

“怎么？”真问。

“那艘定期船，好像出航了。”义经看着地图中一个闪烁的蓝色光粒说。

“在教廷为核心水晶洗礼吗？真是群一板一眼的家伙。”灭骂道，“先解决他们吗？”

“这件事我来处理，”真毫无感情的说，“你们去追圣杯。”

“收到，我还挺想亲眼看一次天之圣杯呢。”义经触碰着代表焰位置的光粒，“说起来，尼娅要怎么办？”

“知道她的位置吗？”

“嗯，看这个光点。从波长来看，应该是白虎。看来在和天之圣杯一起行动呢。”

“圣杯……？”真低头思考了会，“嗯，随你处置吧。”

“原来如此，”义经转身对真说道，“那我就不客气了。”

“独角鲸号，我们直接开走了哦。”灭看着转身离开的真，而真头也不回的说着可以。

“真。”灭犹豫了一下还是叫住了真，“可别太过逞强。”

真只是停下了脚步，但也没有丝毫犹豫，继续走入黑暗中。

坐船去世界树

“这是什么嗼？”

画面一转回到古拉，漫漫长夜露出了黎明，古拉大地沐浴在朝阳的光辉下。

“云海罗盘。”莱克斯看着手中的罗盘继续说，“打捞员人手一个，就像是云海的地图吧。这里要这样，再这样……”说着，莱克斯用手指拨动着罗盘的指针。“今天是哪年哪月哪日啊？”

“我想想，”焰说，“是神历4058年9月5日。”

“谢啦。输入年月日之后，看！”

“这个还真方便嗼。”

“距离世界树最近的就是古拉这里了。问题是船啊。”

“从城镇出发的船都归军队管理呢。”虎提醒道。

“嗯，没戏了。”

“我也同感。”焰略微失望的说。莱克斯回头盯着头盔里的爷爷看着，好像在比划着什么。

“别想骑我头上哦。”

“知道了啦，”莱克斯叹了口气，“不过这下就麻烦了。”

“只要能上船就行嗼？虎知道一个好地方嗼。”

“真的？”

“在古拉的屁股位置有能造船的人嗼，是千造爷爷的熟人，一个叫乌摩恩的大叔嗼。跟他说明情况，也许会借给我们嗼。”

“好吧，死马当成活马医，去那里看看也好。”

说罢，一行人便赶往古拉巨神兽的尾部，乌摩恩的造船厂。

“大叔，乌摩恩大叔，你在哪里嗼？”虎的声音在空荡的造船厂响起。

“谁嗼？我还在工作呢，真吵闹嗼——什么嗼，”一个中年诺彭族一蹦一蹦的走了出来，“这不是虎嗼。”

“好久不见嗼，大叔。”

“嗯，好久不见嗼，大概有1年了嗼？”

“嗯，差不多嗼。”

“一年不见，你长大了这么多嗼。”乌摩恩大叔打量着眼前的诺彭族少年，早已不是记忆中的模样了。

“这就是所谓的成长期嗼。”虎也高兴的回答着。

“还带了这么多随从来——”

“大叔，莱克斯大哥他们不是随从，是朋友嗼。”虎转身对莱克斯介绍说，“大哥，这个人就是乌摩恩大叔嗼，他以前是千造爷爷的研究同伴，也很擅长造船嗼。”

“您好，给您添麻烦了。”

“是吗，是虎的朋友嗼？朋友越多越好嗼。”乌摩恩大叔扑腾着大耳朵向大家打着招呼，“虎，你也长大了嗼。”

“嘿嘿嘿。”

“说起来，您们今天是来干什么的嗼？”

“其实，我们是来找乌摩恩大叔借船的嗼。”

“船？要船的话，托里格镇的港口里就有不少嗼。”乌摩恩大叔不解的看着虎，“为什么要特地来我这里嗼？”

“嗯——这，这个嗼……”

“难道是没钱嗼？”

“对，对啊！其实，虎为了造出花这个人造异刃，”虎疯狂的扇动着自己的大耳朵，“把所有钱都花光了，现在身无分文嗼。好想坐船出去旅行换换心情啊——想归想，但是没船费……”

尼娅在虎的背后小小声的和莱克斯说，“真亏他能这样滔滔不绝地说出这种不打草稿的谎话。”

“有什么办法呢？”莱克斯也悄悄地回应道，“总不能说我们是从托里格镇逃出来的。”

“好吧，这倒也是。”

“这也是虎在为我们着想。”

“人造异刃，你是说千造先生他们研究的那个？”乌摩恩问道。

“我叫花嗼，请多多指教嗼。”

“噢！”乌摩恩大叔惊讶的两耳直立，“她就是你说的！哎呀，真是太厉害了嗼，不愧是千造先生的孙子嗼。好，我知道了嗼。既然如此，随时可以借你们船嗼。”

“那我们就借走了嗼，大叔！”

“谢谢你，乌摩恩先生！”

“嗯噢，好好享受驾船之旅嗼！”

船缓缓驶出港口，向世界树进发。几天之后，他们终于逐渐靠近世界树。粗壮的树干上延伸着数不清的枝丫。

“这就是世界树……”莱克斯握着船的方向盘感慨。

“我还是第一次靠这么近来看呢。”尼娅也被震撼到。

“我也从没离它这么近过，果然壮观啊。”

然而世界树并没有想象中的那么好接近，世界树的周围围绕着一个空洞，那里并没有云海。空洞周围的云海笔直的坠入空洞中，场景十分壮观。

“好湍急的海流啊，掉下去的话根本活不了。”

“来归来了，但到底怎么到那边去啊。你说呢，焰？”莱克斯回头问焰。但是，焰正在沉思，她的表情逐渐变得十分的恐惧。

​“快——快逃，莱克斯，不能待在这里！”焰大声的叫道。

“出，出什么事了？为什么突然叫我们逃跑？”

话音未落，一只巨大的生物从云海中探出身来，粗壮的身躯与长长的身体让人感到莫名的恐惧。

“海蛇！”

海蛇张开血盆大口向莱克斯乘坐的小船咬下去。

“快！”

“知，知道了！”莱克斯迅速回转着方向盘。

焰望着海蛇痛苦不堪，“住手，海蛇。怎么了，听不到我的声音吗……难道……？ ！”

海蛇翻涌起巨大的海浪，莱克斯等人的船也被海浪冲的更远。这时，海蛇突然停止了进攻，向世界树扭去。

“那家伙，突然间怎么啦？”莱克斯回头看着远去的海蛇。突然，前方再次涌起巨大的海浪，一只大型阿鲁斯张开了嘴，很快就要把他们吞没。

“那是！”

“英维迪亚巨神兽！”

“糟，糟糕！要被吃掉了！”莱克斯努力回转着方向盘，但抵不住巨大的引力，船直直的坠入巨神兽的口中……

佣兵村

“好痛啊……”莱克斯从地上爬起，周围漆黑一片，什么也看不清。他拿出自己随身携带的手电筒插在腰间，寻找着众人。

“焰！大家！”

“在，在这里……”焰微弱的声音响起。

“没事吧？”

焰站起身，“嗯，我没事。”

“好痛……”

“嗼……”

“看来大家都没事。”莱克斯舒了一口气。

“话说，这又是被什么庞然大物给吞进肚里了吗。”爷爷飞了过来。

“这里是哪里嗼？”

“英维迪亚巨神兽的内部。”

“嗼嗼！”

“看那里。”爷爷指着远处的光亮，好像是巨神兽的嘴巴。再靠近一些，是一个水池，里面漂浮着许多不知道是什么的东西。周围也四散着一些尸体。

“总感觉这地方有些恶心嗼。”

“我们也会变成那样吗？”莱克斯问。

“别说啦，真是不吉利！”尼娅连忙打断这个话题。

“各位，去找出口吧？我听说英维迪亚的城镇是在它背上的。”白虎提议，“希望能尽快走到那里。”

“是啊，赶快离开这里吧。”

“那里？”

“刚才有什么闪了一下嗼，有人在那里嗼。”

“怎么可能？你看错了吧？”尼娅又变得紧张起来。

“没有看错嗼，在忽悠～忽悠地动嗼。”

莱克斯念叨，“忽悠～忽悠？难道是鬼火吗～”他一脸坏笑的看向尼娅，毫无意外，尼娅果然非常害怕。

“别，别说这种话题啊！”

“怎么了尼娅，你居然会害怕。”

“我，我的意思是这太无聊了，又不是小孩子！”尼娅辩解道。

“真的在害怕啊，”莱克斯得意的笑着，“这样啊，尼娅害怕鬼魂啊。”

“好烦呐——咦，焰呢？焰不见了。”

“咦，刚才还在那里的……焰，焰？”莱克斯向周围看去，这时他的背后出现了一个声音：

“叫我吗？”

莱克斯回头一看瞬间被吓了一跳，焰正托着一团火静静地看着莱克斯。

“你，你干什么啊，别干这种事啊。”

“哪有干什么，就是去那里检查了一下。怎么了？”焰歪着头看着莱克斯。

“呵呵呵。”报复性的笑容从尼娅的嘴巴里被制造出，刚被捉弄的她在这一刻感受到了快意。

“你笑什么……”

“没有，没什么。”

“可恶。！”

焰抬起手将手中的火焰放得更大，而花通过自己的眼睛发光，进一步照亮了这幽暗的空间。

“焰和花真能干啊，相比之下，我家的白虎就……”

“实在抱歉。”白虎低下了脑袋。

“好了，这么亮就不用害怕了，去找出口吧！”

一行人顺着巨神兽的气管小心谨慎的向内部走去。

“这里隐约有光点呢。”

爷爷看着天花板，说，“因为英维迪亚的巨神兽背部的体表是半透明的啊，应该是外面的光透到了这里来。”

“就是说，现在巨神兽在云海上吗？”莱克斯问。

“多半是的。”

“沿着光的方向走，也许就能离开这里了——”

“站住！”

不和谐的声音打乱了莱克斯等人的交谈，前方的石桥上，站着一个身材魁梧的中年人壮汉，他的身边还站着两个像是手下的御刃者。三人从桥上一跃而下，稳稳落在地面上。

“我怎么从没在这一带见过你们？看样子多半是在漂流的途中被巨神兽给吞进来的？”

走近之后，莱克斯发现为首的人一头飘逸的长发，最明显的是脸是有一道长长的刀疤。他穿着士兵背心，手里持着两把镰刀。

“嗯……？这个异刃？翠玉色的核心水晶？原来如此，传闻是真的啊。 ”

“什么传闻？”

“凡是御刃者，都曾经听说过的传说中的异刃——天之圣杯，据说时隔500年，再度觉醒了。不过话说回来，没想到你这样的小子会是御刃者啊。”

“我就不能是御刃者吗？”莱克斯握紧拳头问道。

“没说不能，但前提她得是个普通的异刃。可她太不普通了，给你是暴殄天物。小子，把天之圣杯和那把剑交出来！”

“难道你也想抓走焰？”莱克斯立刻进入战斗状态，“才不会交给你！”

“气势不错啊，小子。勇，周，你们解决其他两个，这小子让我来。”

“嘿，包在我们身上，老大。”

“我会好好疼爱你的，小姑娘。”

莱克斯没有想到，他面对的敌人战斗的方式十分老练，很快就他就被打的节节败退。

“怎么，怎么！不是天之圣杯吗？”一脸凶相的中年人拿着双镰刀有节奏的向莱克斯砍来，莱克斯奋力抵抗，可终有疏漏。这时，焰跑到莱克斯的面前，撑起双手形成蜂窝状的防护罩，抵抗着男人压下来的镰刀。

“诶？这感觉，不对劲……这个人，莫非……”

“怎么了，就这点能耐？”

“那就让你见识一下，我和焰的力量！” 说罢，莱克斯举起剑，斜劈出一个火球。壮汉身体微微一斜便躲了过去。

“躲开了！”

“这不是废话吗，慢到我都打哈欠了。”

“那这招又如何？”莱克斯再次送出无数个火球。 壮汉虽然在不断的后退，但总能轻松躲过这些攻击。终于，无路可退了，壮汉盯着眼前的莱克斯，同时也注意到了焰疲惫的表情。

“莱克斯，等等，力量的传送跟不上了。”

“你已经无处可逃了，接招吧！”莱克斯高高的举起了剑，可是，剑上的能量突然消失，剑也变成了一把没用的废铁。

“咦？！”

“结束了！”中年人立刻抄起自己的镰刀，“小子！”

莱克斯刚缓过神，壮汉的镰刀就已经飞到了眼前……

“小子，你刚当上御刃者没多久吧？所谓的御刃者，是将异刃传输的能量暂时积蓄在武器里，然后通过武技释放。”中年壮汉收起了牢牢抵在莱克斯脖子上的镰刀，继续说，“积蓄量自然也有个极限，如果没头没脑的施展武技，就算是天之圣杯，也难免后继无力。”说罢，莱克斯才注意到，倒在地上毫无力气的焰正大口喘着气。

“你，你……”

与此同时，另两名手下也停了手。

“怎，怎么——”

“我叫凡达姆，”中年人拍了拍自己厚实的胸肌，“在前面村里靠当佣兵为生。”

“佣兵？”

“跟我来。难得一睹天之圣杯的本事，作为谢礼，请你们痛痛快快吃上一顿。哈哈哈。”

“刚才，受到攻击时就察觉到了，”焰走了过来，“那个人好像自始至终，就没有使出全力战斗。”莱克斯和焰望着前面带路的凡达姆，和他背后的异刃。

“搞，搞什么啊，那个大叔……”

在路上，莱克斯向凡达姆解释了他的遭遇。

“原来如此啊，为前往乐园接近世界树的时候，被一口吞下肚了，真是一场灾难啊。”

“凡达姆先生，为什么你会在那种地方？”莱克斯问。

“这里的阿鲁斯啊，会定期地吞噬外界的东西。我本来是想去瞧瞧有没有村里能用的物资漂过来，谁想遇到了你们。”

“原来是这样啊。”

“到了，这里就是我们的村子弗雷斯贝古。”

莱克斯顺着凡达姆指的方向看过去，映入眼前的是一个祥和的村子。村子并不是很大，但却给人一种安心感。不远处的地上，还有一群人正做着俯卧撑锻炼身体；也有许多孩子正在热闹的玩着。

“这座村子里有好多孩子啊。”焰微笑着说。

“御刃者和异刃也有很多嗼。”

“这里可向世界各地输出了不少御刃者和异刃。这些孩子都是我们救下的战争孤儿。”

“如此说来，”白虎望着那些孩子们，“是你们收养了这些无依无靠的孩子。”

“也就到成年为止。”凡达姆的手下勇说，周也补充道，“独立之后，有的成为了御刃者，也有的人去开了裁缝铺。”

“我们让孩子们以自由意志决定他们想走怎样的人生道路。”

“人称的‘侠义佣兵团’啦。”勇自豪的说。

“真是慈善家啊。”尼娅看着凡达姆。

“说得好，这个词我不讨厌。”凡达姆大笑着，“不过我的肚子也饿了，赶紧去吃饭吧。”

莱克斯看着村庄里的居民们。这里，虽然不是乐园，但是每个人都像是在过着最好的生活。或许这里也可以是乐园吧？他想着，情不自禁的笑了起来。

对于新来的客人，凡达姆盛情款待了他们。

“焰？”莱克斯惊讶的看着焰的变化，说不出话。随即，他发现自己手中的剑也变得金光闪闪，剑尖的金色火焰比之前更加浓烈。

“——就，就算变成这种样子，那又怎么样！”义经怒吼着掩盖内心的恐惧，他向焰跑去，试图进攻。焰的眼神变得凶狠，身体上翠玉色的核心水晶忽然发光。

“蠢货！上面！”灭看着天空连忙后退。

“咦！”一道猛烈的光柱从天而降，直射在义经的身上，地面被光柱的力量破坏的不堪。

“——终于醒了吗，光！”灭狞笑着提着刀向莱克斯飞来。

“光？焰，你这模样是？”莱克斯持剑抵抗者灭落下的刀，在强而有力的防护罩下，莱克斯的压力减小了不少。

“我不是焰。”

“咦？”

“我是光，焰是我创造出的另一个人格。”光伸出一只手向莱克斯传递着力量，冷冷的说。

“人格……？”

“别东张西望！”光在莱克斯的身后大声说。

“啊……嗯。”

“集中精神，你不集中注意，我无法将力量传输给你。”莱克斯的进攻变得游刃有余，逐渐开始主导战斗的走势。

“怎么可能，我的力量完全不起作用！”义经痛苦的后退着，“这，这么强大的力量是从哪里来的！”

“废话，她的力量不是来自以太！”灭说。

“不是以太！”义经震怖的望着灭。

“这姿态——很好，光！这才是我要的！”说罢，灭和义经再度发起进攻。

“要上了哦！”光站在莱克斯的身后提醒莱克斯。她的双手交叉放在胸前，又迅速的展开，瞬间一个巨大的光场包围了整个卡拉姆遗迹。

“这是——！”莱克斯惊异的看着眼前的两人，他们的动作变得异常的缓慢。

“因果律预测。”

“因果律？”

“我把未来会发生的事情可视化了。”光解释说，“预判他们的行动轨迹，料敌在先，进行反击。”莱克斯看着两人的进攻路线，点了点头。

“明白了！”

随即光场消失，义经和灭一前一后从左右方向向莱克斯砍过来，却被莱克斯巧妙的化解。

“不，不可能，那招不可能被躲开！”义经的战斗心已经破碎。

“别分心，还有后手！”灭的话音未落，无数光柱从天而降射向两人，义经慌忙躲避，却没注意到持剑上前的莱克斯。

“破绽百出！”

“呜，你这小角色！”义经被掀翻在地，“什么——他们的连携是怎么回事，这样根本无从应对！”

“这就是天之圣杯真正的力量。”灭狞笑着，“真让人怀念啊，长久的等待，就为这一刻！是吧，光！”

莱克斯和光举起手中的剑，呐喊道：

“神圣之剑！”

无数的光柱从天而降，神威制造的防护罩瞬间被破坏，她的核心水晶也被击中。

“为，为什么——义，义经……”神威的身体逐渐消散，变成核心水晶坠落在地。

“神威！”义经瘫坐在地上，无法接受这样的打击，“怎么会……”

“啧，先撤退，义经！”

“神威……”

“别发呆了！”说着，灭举起拳头砸向地面，在浓烟中两人消失不见。

浓烟散去，尼娅发现莱克斯拿着剑，直直的看向远处——倒下的凡达姆。
