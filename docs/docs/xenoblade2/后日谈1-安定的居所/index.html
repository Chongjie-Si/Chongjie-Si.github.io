安定的居所

Posted on 2023-06-09 In docs Views: Waline: Word count in article: 13k Reading time ≈ 24 mins.
异度之刃2

水晶突然开始闪烁，翠玉色的光芒再一次包围了莱克斯，就如同一开始的相遇。他用手遮挡着眼，一道光束冲上天际，随后，光芒消失。莱克斯放下手臂，怔怔地看着不远处的两人。

光和焰。

她们站在不远处，笑着看着莱克斯。花一边哭着，一边高兴的跑上前去抱住了两人。莱克斯看着花从他的身边窜过，此时此刻他多么也想像花这样，冲上去抱住两人。但一路走来，他已不是曾经的他，此时的他变得更加成熟，更加稳健。他就这么静静地驻足，看着光和焰，心里已是满满的感激与幸福。

尼娅在身后轻轻的推了莱克斯一把，莱克斯转过身，看着一路并肩走来的众人，坚定地点了点头。随即，他转身看着光和焰，正如第一次见到她们那样，迈开了脚步，毫不迟疑的走上前去。光和焰看着逐渐走来的莱克斯，脸上的笑意更浓，直到莱克斯走到她们面前，伸出了手，对着两人微笑着说：

“欢迎回来。”

“你是？”

焰一脸疑惑的看着莱克斯，问道。莱克斯的心里咯噔了一下，他好像忘记了，当异刃变回核心水晶之后，曾经的记忆都将消失不见。是啊……莱克斯低下了头，沉默不语。对于她们来说，一路的经历好像已经变成了泡沫，和记忆一样不见了。他摇了摇头，随即笑着看着焰和光。光与焰相视一笑，随即接过焰的话，也是迷惘的看着莱克斯，问道：

“你就是我们的御刃者嘛？”

莱克斯笑着回应说：

“是的。”

随后，光和焰看着挠着头不知所措的莱克斯，哧哧地笑了起来。焰拉起莱克斯的手，说：

“我知道，你叫莱克斯。”

莱克斯的心突然被击中，他和焰在乐园第一次相遇时，焰也是这么对他说的。

“我还知道，你为了救我们……”光笑着看着莱克斯，语气有一些哽咽。

“诶？”

光也拉起了莱克斯的手，望着焰，和她笑了起来。然后，她们紧紧的拥抱着莱克斯，在他的耳边轻轻说：

“我们没有失去记忆……即使如此，我们也永远不会忘记你。”

光和焰的发香从鼻尖传来，莱克斯沉默的抱住两人，把脸埋进她们的肩膀上，小声的啜泣起来。泪水打湿了光和焰的肩，也同样浸润了莱克斯的打捞员服。听着光和焰的哭声越来越大，莱克斯擦干了眼泪，笑着抬起头，看着光和焰。此时此刻，已经不需要有任何的语言，她们回来便已足够。莱克斯笑着，哭着，抱紧着两人，不忍放手。他害怕只要一放手，她们就真的离开了自己。

焰拭去眼角的泪，对莱克斯说：

“好啦，你也是个男孩子，要坚强一些哦。”

“是呀，我们不是已经回来了嘛。”光温柔的摸着莱克斯的脸，“不要这么难过啦。”

“我真的以为……”莱克斯抽泣着，“你们真的回不来了……”

“在那里，为什么……你们要丢下我一个人……”莱克斯哭的更加大声，“你们离开了我，我该怎么办才好……”

焰温柔的看着莱克斯，对光眨了眨眼，光愧疚的笑着，说：

“我们是天之圣杯，不仅储存着所有异刃的信息，我们自己的记忆也会始终保留的。现在的我们还是之前的我们，我们没有离开你呀。”说着，光俯身拍了拍莱克斯的头，带着歉意的看着莱克斯。

离开的那一瞬间，光焰知道自己的决定是自私的。她选择牺牲自己，留下莱克斯一个人孤单的活着，甚至不允许他和自己一同踏往死路。那一瞬间，她甚至担心莱克斯是不是会像真一样，重新走上他的路。所以，光焰在做出选择之后，便和爷爷说了自己的想法，并请他劝阻莱克斯，解开他的心结。

莱克斯抽泣的声音逐渐小了下去。哭过之后红红的眼望着身前的两人，他笑了。牵着两人的手，他把她们牵向了同伴们。齐格笑着看着光和焰，说：“你们能回来，真的是太好了，不然我都对小子的精神状况很是担心呢。”

“是啊，看他回来之后一言不发，我也很担心。”彩歌拉着齐格说，“心爱的人和自己就这么告别了，想一想我都觉得……难以接受……”彩歌望着齐格，手却是拉的更紧了。梅勒芙脱去了礼帽，一头长发披在肩上。她和迦具土一起，再次俯身对光和焰致意：

“感谢你们拯救了幽界……话虽如此，不过，你们能够回来，我真的很开心。”

“各位，我知道你们有很多话想说，”爷爷扑腾着翅膀，回头看着众人，“不过我们要不还是先落脚，找一个旅馆休息一会吧？”

“老头，你这是急着去和新大陆合并，生怕没有自己的位置嘛？”尼娅笑着问。

“老、老头！！尼娅，你也太不尊重老人家了！”爷爷嗔怪道。

“好，好，知道啦，老人家～”

众人嬉笑着，随着爷爷一起缓缓降落。站在爷爷的身上，莱克斯牢牢的牵着焰和光。左手传来了焰的温度，而右手手臂被光紧紧的贴着。站在他们身后，尼娅羡慕的说道：

“诶，我就说，光更粘人吧。”

听到这话，光羞红着脸，却是更加紧紧贴近了莱克斯。莱克斯看着左右两人，焰红红的头发散落在肩上，遮盖着额头与她的眼，光的长发披肩，洒在他的手上，他微笑着对两人说：

“乐园，就在这里啊。”

“我们，终于来到了。”

英维迪亚

克劳乌斯在消失之前，将幽界的阿鲁斯全部合并，形成了一块新的大陆。爷爷找到了英维迪亚的巨神兽，逐渐靠了过去。

“莱克斯，就到这里吧。”爷爷靠岸，对众人说。

“爷爷？”

“我就不和你们一起了，更何况我现在的体型，也没法跟着你们啊哈哈。”

“那爷爷，你之后去哪啊？我还没地方住呢！”莱克斯着急了。

“你这家伙，你现在好歹也算是幽界最富有的打捞员了，难道租房子的钱都没有吗？我先回弥先村，看看村里的人们都怎么样了。等你们安顿好之后，再来弥先村找我吧。”

“那我们也告辞了。”齐格和彩歌对莱克斯说道。

“诶，你们……”莱克斯的眼里流露出不舍。好不容易大家一路走来到了这里，结果却不得不分离。

“幽界已经没事了，我也得回洛修利亚了，小子。我还得回去帮助老爸，看看能不能出一份力气呢，哈哈哈哈。”齐格猖狂的大笑起来。

“齐格……”

“小子，我们又不是见不到了。人生在世，总有别离，欢迎你们来洛修利亚来旅游哈！”

“那我们也去了，斯佩比亚王室这边应该很有很多的事情等着我们回去解决。”梅勒芙和迦具土向众人解释道。

“尼娅呢？”莱克斯问道。

“我……要不还是先去古拉吧。毕竟那里是我的故乡。”

“尼娅，我可以推荐你去担任古拉的领事官，”梅勒芙笑着对尼娅说，“以此感谢你对幽界的贡献。更重要的是，感谢你救了我的弟弟。”

“诶，这样好吗？”

“嗯，相信陛下也会同意的。”

莱克斯望着远去的众人，情绪又低落了下去。光看到莱克斯的样子，于是走过来笑着哄着他：

“那我们是不是也该动身去寻找住的地方啦？”

“嗯……话虽如此，可我们该去哪呢？”

“要不去弥先村吧。”焰笑着说。弥先村远离世俗的安详让焰流连忘返。

“嗯……光你呢？”

“啊，我，我都可以的，只要……你……”光低下着头，声音越来越小。

“什么？”

“没，没什么。”

“那好，”莱克斯拍板决定，“不过在此之前，我们先去考尔先生那里报个平安吧。”

夜幕降临，莱克斯他们住进了英维迪亚的旅馆。曾经在这里，焰为了救倚音，选择独自一人去面对灭……不过都结束了，她们都在，莱克斯心想着，陷入了梦乡。

半夜，莱克斯做了一个梦，他梦到站在天空桥的一边，看着光焰再次切断天空桥，苦笑的看着他，然后消失在他的眼睛里。他努力的追去，却发现自己无论如何努力，光焰最终都是孤独的走上那条路。泪水打湿了枕头，莱克斯蜷缩在床上，喃喃自语：“光……焰……”

“光焰！”

莱克斯突然惊醒，强烈的痛苦袭击着他的神经，瞬间睡意全无。当他醒来时，他惊讶地发现光和焰就坐在他的床边担忧的看着他。

“你们……我……怎么了？”莱克斯的眼角还挂着泪水，他一脸疑惑的看着身边的两人。

“诶……”焰关切的看着莱克斯，握着他的手，“从刚刚开始就一直在喊着我们。”

“莱克斯，怎么了呀，是做噩梦了吗？”光问。

“没，没事……抱歉，让你们担心了。”

“莱克斯……”焰担忧的神色更浓了，“是不是……没事的，我们都已经回来了，我们不会离开你的。”

“焰……”说着，莱克斯的眼泪便流了下来。光抱住了莱克斯，摸着他有些戳手的头发，安抚着他。他不过是一个还没成年的少年，却要承受这样的痛苦……

“莱克斯，我们在这陪着你，你安心的睡觉吧。”光看向焰，提议道。

“嗯，睡吧，莱克斯，经历了这么多，好好休息吧。”焰拉着莱克斯的手，说，“我也会在这陪着你的。”

“诶，这，这不好吧……？”

“没关系的，听话。”焰温柔的语气中带有不容拒绝的坚定。

看着逐渐睡熟的莱克斯，光和焰对视了一眼，满满的自责感涌上心头。之前就听说过，有的人会在经历战争之后，留下一种创伤综合征。我们一定是给莱克斯留下了很伤痛的心理阴影……焰温柔的抚摸着莱克斯熟睡的脸，和光一起悄悄地离开了房间。

清晨，温暖和煦的阳光透过巨神兽的背部，落到了冯斯麦姆的各地，幽界的人们再也不用担心巨神兽的生命终结而失去家园。阳光透过窗，照在了莱克斯的脸上。他迷迷糊糊的擦了擦眼，从睡意中醒来。推开门，光和焰早早的就坐在桌子前有说有笑的喝着茶。看到莱克斯出来之后，焰笑着起身，拉着莱克斯说：

“今天我们一起去看剧哦？我和小光啊已经去买了票了。”说着，焰扬了扬手中的票。

“诶，你们……？”

“这可是从小倚音那里拿的，”光悄悄地说，“位置很好哦。”

“考尔先生已经写好了关于凡达姆先生的剧本，今天正好是首映，他特地给我们留好了位置。”焰从桌子上拿起她早晨去冯斯麦姆买的饼干，递给莱克斯，说：“咱们等看完剧之后，要不要去看看凡达姆先生？”

“啊……嗯。”莱克斯点了点头，“我也很想去和凡达姆先生聊聊天，聊聊这一路走来的经历。”

故人

莱克斯带着光和焰走进了剧场。经过了战后的重建，新的世界迎来了比往日更盛的繁荣。剧场外，人们早早的就排起了队。而墙上的海报上，凡达姆放荡不羁的脸被印在最显眼的位置，下面写道：

“佣兵团团长——凡达姆的一生。”

莱克斯他们坐在了剧场的中间。演出即将开始，光说：

“真让人怀恋啊，凡达姆先生。”

“是的。”莱克斯点了点头，“他是我们的救命恩人。”

坐在两人中间，光和焰靠在莱克斯的肩上，抱着他的手臂。莱克斯红着脸，看着考尔先生走上舞台，对众人鞠躬。随后，考尔先生对观众们说：

“感谢各位前来观看今天的首映。”待潮水般的掌声停息之后，他接着说，“在演出开始前，我想先简单的介绍一下凡达姆先生。凡达姆先生是我曾经的团员，也是我的战友，他曾经跟着我走南闯北，救了很多人，帮助了很多人，也收养了很多孩子。也许在坐的各位有很多人都认识凡达姆先生，或是曾经受过他的恩惠。遗憾的是，凡达姆先生最后牺牲于卡拉姆遗迹。佣兵的一生随时随地都充满着危险，凡达姆先生也深知这一点。但是我觉得他的牺牲是他所希望的，他并不后悔。因为，他为的是拯救我们幽界的英雄，那个将我们从幽界大陆垂死挣扎中解脱出来的英雄——莱克斯。”

随后，聚光灯聚焦到了莱克斯的身上。此时的莱克斯还正红着脸，光和焰像是挂件一般黏在他的身上。突然被灯打在脸上，莱克斯连忙从光和焰的手里挣脱开来，一本正经的危坐着。潮水般的掌声再一次爆发，新世界诞生后，人们或多或少的都听说过这么一位救世的英雄。然而当英雄就这样出现在眼前时，惊讶与激动瞬间弥散在人群当中。掌声久久不停息，像是在感谢着莱克斯对这个世界作出的贡献。莱克斯挠了挠头，起身对众人鞠躬。在掌声中，考尔先生继续说道：

“莱克斯，不知可否有请你上台，为今天的首映式做最后的结语？”

“诶……我？”莱克斯有些不知所措，他并没有提前收到消息，也没有提前做好准备。但是，当他看到焰和光肯定的目光时，他便坚定的走上台，面向着台下无数的人们，缓缓开口道：

“我是莱克斯……”话音未落，掌声便更加热烈。“也不是什么英雄啦，我只是希望大家的脸上都有笑脸……很开心能有这样的机会来看一部关于凡达姆先生的戏剧。凡达姆先生是我一生中最重要的人中的一员，他看着很凶，但内心十分善良，是他教会了我如何成为一名合格的御刃者，是他教会了我不要恐惧力量，也是他为了救我，选择牺牲了自己……”莱克斯的语气有些哽咽，“我很想念你，凡达姆先生。我也希望，能让更多的人了解你，了解你的事迹与独属于你的伟大……”

在掌声中，莱克斯走下台，戏剧缓缓拉开帷幕。

“我是凡达姆，是一名自由佣兵。”台上，一名扮演者凡达姆的魁梧大汉开口道，“我曾经跟着考尔老爷子大杀四方，创下无数战果，战功显赫——”

“但也总是因为慷慨解囊落的喝西北风的下场……”另一名演员身着严实的斗篷，像是在扮演考尔先生。

“……后来啊，我遇到了一个少年，他叫莱克斯。当我看到他的时候，我突然就回想起了自己死去的儿子。要是他现在还活着，一定和莱克斯差不多大吧。不过让我更加留意的是，莱克斯身边的那个异刃，居然是传说中的天之圣杯。我顿时觉得，这小子不一般。”

“后来，我发现莱克斯是一个御刃者新手，没有什么实战的经验，我便开始手把手的教会他如何更有效率的战斗。不得不说的是，这小子的领悟力还是很强的，有的技巧我花了很久才学得会，而他只是简单的看了看便能很快掌握。若是如此，作为天之圣杯的御刃者，他一定是合格的，我是这么想的。”

“然而，他总是恐惧着力量，害怕因为力量而导致身边的人受伤。我告诉他，力量本身并无对错，它取决于使用者的心。为了守护身边的人，我们必须拥有足以守护的力量。我成立弗雷斯贝古佣兵团时，我便做好了这样的觉悟。我要有着强大的力量，保护好我的异刃的同时，也能保护好身边的人。”

“……灭和义经抓走了焰，莱克斯等人面对着他们的攻击，显得手足无措，只能被动的挨打。眼看着莱克斯就要命丧于此，像我死去的儿子那样，我看着敌人顿时怒火攻心。于是，我做了一个决定，牺牲我自己，将我的生命延续到莱克斯的身上。我并不后悔这么做，只是，我对朱雀，我的战友，十分的愧疚……我并没有能保护好他，而是替他做出了决定……不过，即便是朱雀，他也一定会这么做的。”

“……此去之后，已是多日。不知道他们如今可好，是否已经进入了乐园，是否已经拯救了世界呢？”

“凡达姆先生……”莱克斯呢喃着，握紧了光和焰的手。

演出结束之后，莱克斯走进了考尔的房间，向他问好。

“你终于平安回来了，莱克斯。”考尔咳嗽着笑着说。

“考尔先生，您的身体……”

“无妨。我已经习惯了，500多年的生命使我早已对它不再苛求。莱克斯，我很感谢你为大家做的这一切，也很感谢你，替我斩断了和那个男人的联系。”

莱克斯笑着挠着头，“这些都是我应该做的。”他随后看着光和焰问道：“你们有办法解决他的身体问题吗？”

光摇了摇头，惋惜的说：“抱歉，要是之前是我们原来的模样的话，还可以使用力量，就可以修复核心水晶。但现在门消失了，恐怕我们很难再变回成那样了。”

来到父母的坟前，莱克斯笑着拉着光和焰，对他的父母说：

“好久不见了，爸爸，妈妈……我已经成为了非常厉害的御刃者了，嘿嘿，厉害吧？”莱克斯笑着挠着头，接着说，“之前向你们祈求，保佑我能够顺利前往乐园。然后，爸爸，妈妈，我找到了乐园，并且，现在大家都幸福的住在乐园里。我很厉害吧，现在他们都叫我幽界的英雄呢，嘿嘿……不过，我的压力有点大呢……”

“哦对了，上次和你们介绍过，这位是焰。然后，这位是光，也是我重要的搭档，是我非常重要的人。新的世界诞生了，我和焰还有光未来还不知道会发生什么，会去往何处，不过，无论发生了什么，都请你们保佑我们吧……”

莱克斯默默的跪在墓前，低头祈祷着。焰和光站在莱克斯身后，也默默的低下了头。

“走吧，我们回去吧。”

“嗯，好。”

回到柯蕾尔的家，莱克斯品尝着阿姨的手艺，笑着说道：

“嘿嘿，吃惯了焰做的菜，感觉阿姨你做的菜居然有些开始吃不惯了……”

“你这孩子！吃了快10年，居然还说这种话。”柯蕾尔笑着嗔怪道。

“毕竟焰的厨艺太绝了呀。”

“好可惜，莱克斯是不是还没有尝过我做的饭？”光问道。

“嗯嗯，下次可以尝一尝光的手艺呢。”突然，莱克斯无意中启动了因果预测率，看到了品尝完光的料理之后的自己，瞬间黑了脸。光像是感受到了什么，生气的看着莱克斯。

“不行，我也想做饭给莱克斯吃！”

莱克斯想起了自己未来的惨状，咽了咽口水，点了点头。

“我听说村子里有一间房子一直在闲置，听他们说，他们是要留给……”柯蕾尔阿姨笑着看着莱克斯。

“诶？留给……我吗？”

“对啊。毕竟莱克斯，你可是我们的骄傲哦。这样一来，你们便可以长期在这里住下了。等会吃完饭，我可以带你们去转一转呢。”

饭后，跟随着柯蕾尔阿姨，莱克斯看到了那个房子。一进弥先村的拱门，右手边的那间便是了。稍近一些的地方是莱克斯为村子里筹建的教室，而不远处便是海峡。推开门，里面一共有几间屋子，莱克斯和光焰各一间，再加上一个小小的餐桌和沙发，家的温馨感扑面而来。

“真不错呢……”焰笑着看着莱克斯。

“是呀，这样我们也有家了。”光说。

“哦对了，你们呀，”柯蕾尔阿姨补充道，“今晚会有流星雨哦。”

“诶，柯蕾尔阿姨，真的吗？我还没看过流星雨呢。”莱克斯兴奋的问道。

“嗯，听说出现在后半夜，要是不困的话可以去守一守哦。”

夜色正浓，月色正浓。三人坐在屋顶上，吹着晚风，等待着流星雨的到来。

“听说，当流星落下的时候，许下的愿望都会成真呢。”光说。

“诶……我还是第一次知道。”

“你们看！”

这时，黑色的夜幕里一抹翠玉色彩带映在空中，像是突然拉开的帷幕，曼妙多姿。

“那是极光诶……好美……”焰沉浸在这美景中。

莱克斯回想起焰曾经在古拉点起的火焰，那一幕也还是和现在一样的美，她们也还是和现在一样的美。这时，天空像是拉开了序幕，无数的流星从夜空中划过，留下闪烁着的尾巴。黑暗的夜晚，因流星雨的到来变得光明。

“快，莱克斯，快许愿。”

看着紧闭着眼默默许着心愿的光和焰，莱克斯再一次沉默了。从世界树上回来时，炸毁的世界树也是如同流星一般，划过天空。那时的晚霞如同此刻的极光一般美丽，那时的碎片雨也像此刻的流星雨一般划过夜空。只是，那时的她们不在身边，而此时此刻，她们，就在我的左手，以及右手。莱克斯看着身边的两人，轻轻的握住了两人的手。光和焰的手明显的颤抖了一下，但随即更加坚定的握紧了莱克斯。莱克斯闭上了眼睛，想象着流星划过的场景，回忆着古拉的星空下，焰轻轻吹起的火苗，回忆着莫尔斯之地，光静静地枕在自己的腿上。随即，他口中呢喃着，许下了心愿。

“莱克斯，你许下的心愿是什么呢？”焰笑着问道，感受着指尖莱克斯的温度。

“我们一起说出来吧，怎么样？”莱克斯笑着看着身边的两人，她们同时点了点头。

“那么我说一二三，我们一起说出来吧。”

“一……”

在那片星空之下，在莫尔斯断崖之前，莱克斯手中握着剑，拼命的抵抗着真，但却从未想过放弃，从未想过丢下她们。

“二……”

在那片星空下，在世界树的顶端，光焰选择牺牲自己，不仅仅是为了世界，更是为了莱克斯。

“三……”

即使自己无法陪伴在对方的身边，也希望对方能够幸福的活下去。生命是短暂的，然而在这短暂的生命中，拥有一刻的美好便是幸福。神啊，我是个贪心的孩子，我想向您祈求，我希望，我的一生，都是幸福快乐的，最重要的是……

“我们永远在一起。”

The End
