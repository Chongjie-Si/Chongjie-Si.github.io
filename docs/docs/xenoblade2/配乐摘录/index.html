异度之刃2

主题曲

整个游戏的主题曲。像是在云海里浮游，像是在古拉看着星空，像是在灵洞里穿梭，像是在世界树攀爬。只不过，更像是莱克斯站在梦中的乐园里，眺望着荧光闪闪的世界树，紧紧牵着光和焰的手，约定永不分离。

反击神曲 Counterattack

反击神曲，不解释。

与你同行的路 君と歩く道

真是幸运的，也是不幸的。他幸运的遇见劳拉，与她度过十几年快乐的时光；不幸的是，恰是这种幸福让他背负了500年的枷锁与桎梏，活在黑暗之中。他多么想消灭自己，却因为劳拉临死之前的话而不能消灭自己。他茕茕孑立，孑孓而行了500年，并陷入思想的死胡同，直到莱克斯将其救赎，他才彻底解脱。

梦中的乐园 夢の中の楽園

这首曲子最突出的地方是它的名字。取名为梦中的乐园，看上去是那种梦幻的、让人憧憬的、童话般的地方，然而曲调却十分忧伤，与之形成强烈反差。在前半段，吉他作为旋律乐器，配合着钢琴与弦乐，简单的勾勒出梦中的乐园的模样。而在后半段，弦乐和人声涌入，伤感的情绪瞬间便弥漫开来。光焰记忆中的乐园不存于世，所以曲调变得悲伤，而名称也只能是“梦中”的乐园。

有你的未来 君との未来

这首曲子在整个故事中只出现过两次。第一次是在莫尔斯断崖，莱克斯对焰和光的告白；而第二次是在莱克斯最后与光焰分别时，光焰对莱克斯告白。一次是莱克斯对光焰，而另一次恰好相反。曲子里的柔情见证了莱克斯成长和蜕变。撕心裂肺的分离与喜极而泣的重逢，再到最后的诀别，这一切让莱克斯变得成熟。一开始，曲子是温柔的。舒缓的弦乐把情感表现的淋漓尽致，如同莱克斯和光焰的情感一样细腻而温柔；随着曲子的进行，曲风变得充满了力量，管弦乐团与架子鼓组的使用让其听感变得更有活力，更有希望。整首曲子正如莱克斯与光和焰的未来，充满了爱与温柔，充满了希望与光明。

流浪的灵魂 Drifting Soul

这首歌在整个故事的情感高潮部分会出现。虽然第一次出现是在凡达姆牺牲的时候，但实际上它是尼娅的一首专属配乐，歌词与尼娅的心路十分贴切。

片尾曲 One Last You

看完整个故事之后，再听这首曲子，极易致郁，每每听到总是十分压抑喘不过气。这首歌的歌词从某种层面上来说是光和焰对莱克斯的告白，歌词感人肺腑，充满柔情。

光田谱的曲一如既往的稳健，和声设计不落俗套，搭配上高桥填写的词，浓浓的爱意与不得不分开的结局让人心碎，配合着副歌极具穿透力与感染力的女高音，把这种纠结与挣扎诠释的淋漓尽致，直击内心最柔软的地方。编曲部分，结合mv的画面，水晶闪烁时风铃声划过，低音的贝斯走动，吉他的独奏旋律从左右声道流出，与画面中满天的光焰融为一体。结尾重复两遍的so la me与歌词want you to know对应，想说却说不出口的话让旋律也变得纠结与重复。最后的xi do la又回到小调的难过，虽是高音，却因为零碎显得伤感而压抑，更有一种戛然而止的感觉。

“当满天的星火划过云霄，眼前的巨神兽围绕着世界树聚成大陆的时候。一行人在青龙的背上看着眼前一切，这才是光和焰一直向往的地方。水晶默默的恢复了光芒，两个熟悉的身影面带微笑的伫立在远方，莱克斯向前迈开了第一步，就像最初的相遇一样。于是，少年遇到了少女。”
