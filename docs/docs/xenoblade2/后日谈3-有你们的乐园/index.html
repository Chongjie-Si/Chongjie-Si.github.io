异度之刃2

夜深人静，弗雷斯贝古的老树下月影斑驳。

“水分，星织，我需要请你们和佣兵团的各位同伴一起办一件事。”坐在佣兵团团长的办公室内，莱克斯对桌子前的姐妹两说。随后，他的视线转向另一边，对朱雀说：“朱雀，我也想请你参与其中，作为统筹监管。”

“什么事情，团长？”星织疑惑的问道。

“事情是这样的……”

“……”

“诶……？感觉超级大的工作量……”

“是的，”莱克斯双手合十，拜托道：“这件事只能麻烦你们……而且，还请大家务必保密。”

“……”朱雀对莱克斯疯狂的想法颇有些无语，“你啊……”朱雀扇了扇翅膀，转身离开，边走边说：

“知道了，我会帮你安排好的。”

“团长，你真的，太……”水分的眼里流露出羡慕的神色，她摇了摇头，感慨着。

“非常感谢你们，”莱克斯挠了挠头，“只有你们才能办到这些事情，我也只想请求你们帮忙。再次感谢……”莱克斯笑了笑，不好意思的说。

“没事团长，包在我们身上。”

这时，一阵敲门声响起，穿着居家服的焰手里托着几只茶杯走进屋内。暖黄的灯光温暖着整个屋子，如同散发着温暖的焰一般让人感到安心。焰笑着看着屋内的水分和星织，充满爱意的眼神随后看向工作了一天的莱克斯。每次回到弗雷斯贝古，莱克斯便忙的团团转，佣兵团的大小事务他都悉数过问。不知不觉，弗雷斯贝古佣兵团已成了幽界的一个非常庞大的势力，每天收到的委托更是数不胜数。焰把茶具轻轻放在桌子上，取出两杯茶递给水分和星织，再取出她最爱的塞利奥茶。这杯茶我可是精心调制了很久呢，焰一边想着，一边微笑着把茶递给莱克斯，问道：

“莱克斯，你们在聊什么呀，这么开心。”

“啊……没什么，焰。”莱克斯接过茶，茗了一口，接着说，“就是些团内的事务。谢谢焰的茶，我很喜欢。”

听到这话，焰轻轻的捂着嘴笑着，“只要莱克斯喜欢就好。时间不早了哦莱克斯，今天的工作是不是可以结束了？”随后，焰转身看向水分和星织，像是在询问她们的意见，又像是给她们下达逐客令。水分姐妹俩看了看彼此一眼，随即向焰点了点头，说道：

“嗯嗯，时间不早了，那我们先出去了，你们也早点休息哦。”

“好的，谢谢你们啦，水分，星织。”莱克斯挥了挥手，说。

“包在我们身上。”水分和星织对莱克斯点了点头，转身离开了办公室，轻轻的关上了门。

“姐姐，莱克斯真的……这真的是一个大手笔啊……”出了门，星织依旧沉浸在莱克斯疯狂的安排中。

“是的……虽然我们都没去过那里，但可以想象的到……团长……真的太宠她们了。”

“诶，姐姐你是不是羡慕了？”星织打趣道。

“才，才没有！”

姐妹两嬉笑的声音逐渐远去，宁静的村庄内仿佛只能听到几声蝉鸣。

“你也真是的，莱克斯，”在屋内，焰轻轻的拉着莱克斯，说，“这么晚了还在工作，我们都等了你半天了……”

“抱歉抱歉，”莱克斯一脸歉意的说，“才回来没多久，事情比较多，就工作的比较迟了……”

听了这话，焰默默低下了头不再说话，似乎有些难过。

“焰……？”莱克斯疑惑的看着焰。

“莱克斯，要是我们能帮你分担些就好了……”焰难过的抬起头，看着莱克斯，“我们回来后就再也没有战斗过，就连佣兵团的任务我们也一次没有参与过……莱克斯，其实我们也想为你做些什么……”

莱克斯摇了摇头，把焰的手合在手心中。“你们能一直陪着我，就是我最大的心愿了。好不容易可以远离战争，不用像以前那样每天都是伤，这难道不好嘛？”

“可是，我们也想做些什么……哪怕是成为佣兵团的秘书，也可以为你做些事……”焰的声音越来越小。

“我说过了，要保护你们的，对吧？”莱克斯摸了摸焰的头，说，“所以我不想让你们受伤害。我希望焰，还有光，你们都能过上幸福的，自己想要的生活，而不用去战斗。”

“可我们是异刃——”

“你们是女孩子。”莱克斯打断焰的话，温柔的看着焰，“是我最重要的人。”

“莱克斯……”

“不说这个了，这么晚了焰一定也累了吧？对了，光呢？”

“小光还在生闷气呢，说你这么晚都没回去陪陪我们，所以我就过来看一看了。”

“哈哈……”莱克斯愧疚的摸了摸鼻子，“那我们快回去吧。”

从办公室走出来，莱克斯才发现今晚的月亮是那么皎洁。弗雷斯贝古的水田里作物轻轻的摇晃着，老树的桃花的影子也跟着微微摇摆。穿过早已关门的商铺，走到村庄的另一边，莱克斯轻轻推开房门，一抹金色印入眼帘。

“莱克斯！”光嘟着嘴不满的说，“这么晚不回来，你要怎么补偿我？”

“对不起，光，那个……”莱克斯想了想，“明天我们去吃云海蟹奶油土豆饼？”

“不行，不同意！”

莱克斯笑了笑，随即上前抱着光，在她的耳边轻轻问道：

“这样可以了嘛？”

光红着脸趴在莱克斯的肩上，轻轻的锤了锤莱克斯的胸膛。夜更深了。

美好的午餐

“早上好，莱克斯。”平常而幸福的一天，从焰的早安开始。莱克斯睡眼惺忪的起床洗漱，坐在餐桌前，看着要好的两人，笑着拿起勺子，吃着早餐。

“真美味……”

“那是当然，这可是我做的。”焰笑着说。

“要不下次我来做早餐？”光提议道。

“诶，说起来我都没有吃过光做的料理呢。”

“莱克斯想尝尝嘛？”光的眼神里充满了期待的色彩，“我最擅长做各种料理了！”

“那个……”焰为难的看着光，“这样真的好吗……”

“好，就这么决定了！明天的早餐就包在我身上了！”

“哈哈哈，好呀，我也有些迫不及待了。”莱克斯看着叉着腰自信的光，笑着说。

“原来莱克斯这么想品尝我的料理！”光的语气有些激动，“那么今天的午餐也请交给我吧！”

“那个……小光，莱克斯会吃坏肚子的……”焰有些担心的对光说。

“好过分，我做的料理才没有这么差劲……虽然是比不过焰啦。”光不满的说，随即看向莱克斯，“莱克斯，可以吗？”

“可以呀，你们谁做饭给我吃我都很开心。”莱克斯微笑着点了点头，“焰每天做饭也很辛苦呢，偶尔休息一天换换心情吧？对了，昨天火花完成委托从斯佩比亚回来了，说不定你们可以一起去冯斯麦姆买点东西回来哦？”

“诶？莱克斯……有什么想要买的吗？”

“唔，我想想……”莱克斯摇了摇手中的餐具，他想起曾经焰一直想要的书，说道：“我想要那个料理书籍……哦对了，我还想吃云海蟹奶油土豆饼。”说完，莱克斯哈哈笑着问焰，“可以麻烦你嘛？”

“莱克斯……”当焰听到了莱克斯口中的书和食物时，她便知道莱克斯又在为自己和光考虑，感动与爱意涌上心间。

“放心吧，焰，交给我吧。”光笑着对焰说。

“对了，光，不如让大家也尝尝你的料理？”莱克斯提议道。

“咦？可以吗？”光有些不可置信，又有些后悔……早知道不该说这么多的……

“当然可以了。中午我们请朱雀，水分和星织一起来品尝一下吧。”

“那个，焰……”光咽了咽口水，求助的看着焰。而此时焰正捂着嘴偷笑着，说，“好啦，我会帮你的。那上午我们准备午餐，下午一起去逛街吧～”

“太好了……”光长舒了一口气。

莱克斯笑着点了点头，收拾好餐具，离开了屋子，投入到佣兵团的工作中。

今天是其他佣兵团的负责人前来交流的一天，同样也是弗雷斯贝古佣兵团结算薪水的一天。

“做的不错哦，力男，这是最近你完成的委托的薪水。”莱克斯看着水分将金币递给力男，力男的脸上浮现的笑意遮掩了他三分的恐怖。

“韦驮天，又可以去买吃的啦。”星织笑着对吃货韦驮天说。

“嗼嗼，谢谢你，也谢谢团长。”韦驮天扭动着胖胖的身体，像是孩子一般让人觉得可爱。

“真是稀奇……”斯佩比亚佣兵团的团长不可思议的看着眼前这一幕，“我头一回看见还有这么对待异刃的佣兵团。”

“啊……？”莱克斯不解的问。

“在佣兵团里，异刃是作为战力存在的。实际上没有御刃者也就没有异刃，因此委托的薪水我们都是交给御刃者，从来没有听闻说异刃还能拿薪水。真是稀奇啊……”

“哈哈，我们这里御刃者比较少，很多委托都是靠他们独立完成的。”莱克斯笑着看着大家，说，“而且，御刃者和异刃本身就是伙伴，也没有地位的高低，大家是应该和平相处的。”

“你是受了天之圣杯的影响吗？”斯佩比亚佣兵团团长问。

“他可不是。”默默站在莱克斯身后的朱雀开口说道，“想必这应该是莱克斯自己的追求。”

莱克斯笑着看着他的搭档，说，“我想，这也是凡达姆先生所希望的吧。”

“说来惭愧……今天来这里学习，我真的受益匪浅……也许我们是时候要重新审视我们和异刃的关系了。”

莱克斯握着他的手，用力的摇了摇，说，“您客气了，我们也存在着一些不足。以后说不定还要和你们多多合作。”

目送着别的佣兵团的人们远去，莱克斯小声的问着朱雀：

“对了，那件事现在进展如何？”

“别着急。人手已经安排好了，前期的清理工作基本上已经完成。目前来看，物资都很丰富，主要是运输需要耗费时间。”

“嗯嗯，辛苦你了，朱雀。”

“不用客气，莱克斯。能够见证者你一步一步成长也是我的荣幸。诶，那不是光吗？”

朱雀抬起翅膀指着不远处的身影。顺着朱雀的视线看去，莱克斯看到光正一蹦一跳的向他们挥着手。莱克斯情不自禁的笑了起来，午餐已经完成了吗，居然有些小期待。随即，他和朱雀向不远处的星织和水分走去，莱克斯对她们说：

“已经是中午了，我们一起去吃午餐吧？”

“好啊……”星织伸了一个懒腰，问道，“团长，今天午餐吃什么？”

“今天我们去吃光为我们准备的料理哦。”莱克斯笑着回答。

“咦咦咦！！”

“你们两做秘书这么久了，非常的辛苦，一直想和你们一起吃一顿饭来着。正好今天朱雀也在，要不……”

“说起来，今天品尝不到焰的料理，还真是有点可惜。”一想到焰做的食物，水分的口水就要流下来了。

“说不定光做的也很好吃呢？”星织对姐姐说。

“原来如此，”朱雀点了点头，“那我们一起去尝尝吧。”

坐在餐厅的桌子前，光穿着围裙，双手合在一起微笑着看着莱克斯，“今天我可是做了莱克斯爱吃的烤肉哦。”说着，光端起一盘奇奇怪怪的菜，放到桌面上。马赛克一般的食物散发着奇奇怪怪的味道，莱克斯盯着这盘菜，有些手足无措。

“光，这是……”

“这是莱克斯最爱的烤肉。”光忐忑的说。在端上桌子之前，她已经提前品尝了一口……看着这盘紫黑色的烤肉，莱克斯愣了愣神，随后看着身边的众人，说：

“我们一起尝尝吧。”

水分和星织对视了一眼，像是做好了慷慨就义的准备。她们夹起一点点散发着特殊香味的菜，艰难的咽了咽口水，努力的张开嘴小心翼翼的将菜送入口中。几秒之后，水分的脸以肉眼可见的速度变成紫黑色，而星织则像是被噎住了一般，连忙抄起桌上的水杯狠狠的灌了几口水。姐妹两根本不敢抬起头，她们害怕面对光的目光，于是索性低着头，一言不发。看到这一幕，光的表情变得有些僵硬。

“原来如此。”朱雀的语气还是一如既往的冷静。盯着眼前看不出原材料的食物，朱雀盯了半晌，历经百战的他终于也慌了神。一想起星织和水分的表现，朱雀沉默了。

“朱雀……？”莱克斯小心翼翼的望着朱雀。

“原来如此……”朱雀喃喃自语道。随后，他夹起自己碟中的料理，迅速的咀嚼着。一道闪电划过他的脑袋，朱雀感觉到一股能量从自己的胃里不断的涌现，白色的吐沫就要喷涌而出。他急忙捂住嘴，努力的咽了回去。

“朱雀，你觉得怎么样？”光一脸期待的看着朱雀，询问道。

“唔……”朱雀努力的平静了下来，他低头看着盘中的食物，寻找着赞美的话语。“比我预想的好很多，称不上难以下咽。不过，这个味道不应存在于幽界。”说完，朱雀向光点了点头，随即对莱克斯说，“我还有事，先走了……”

“朱雀……”莱克斯看着朱雀推开了门，还没走多远，就昏倒在地。看到这一幕，光的眼神里流露出失望与自责。

“莱克斯，要不算了吧……我……真没用……”

“每个人喜欢的食物和口感不一样呢，光。我还没有品尝你精心制作的料理哦？”莱克斯笑着安慰道。

“可是——”

莱克斯晃了晃手中的餐具，笑着打断了光。随即，他夹起料理送入口中，耐心的品尝着。莱克斯的表情变没有出现任何的变化，相反，好像还很满意的样子。身旁的水分和星织不免看呆了眼，“不愧是团长……”莱克斯又夹起一小块，送入嘴中，笑着对光说，“光真厉害，做的料理也很有新意，我很喜欢。”随后，他牵起光，说，“谢谢光，今天辛苦你了。”

“莱克斯……”泪水浸润了光的眼角。她知道自己做的东西并不很可口，但莱克斯依然鼓励着她，夸奖着她。她不由得抱紧了莱克斯，把脸埋在莱克斯的胸前。“谢谢你……”

莱克斯抚摸着怀中的光，对星织努努嘴，星织立刻会意，悄悄的出现在光的身后为莱克斯递来一个空盘子。莱克斯终于忍不住将刚刚的食物吐了出来，瞬间，莱克斯感觉空气是那么的清新，阳光是那么的灿烂，生活是那么的美好。星织迅速的撤走了盘子，一切都在光毫不知情的情况下默契的配合完成了。这就是我们佣兵团的秘书啊，莱克斯心里感慨着。这时，一阵诱人的香味传来，焰端着料理走了过来。

“各位，开饭了哦。”

“哇，是焰的料理！”水分瞪大了眼睛满心欢喜。

焰一边把盘子放在桌子上，一边对莱克斯说，“小光今天也很努力了呢。”

“是的呀，”莱克斯微笑着对光说，“看到为了我这么努力的光，我也要更加努力呢。不过，我们还是先开饭吧，我也有些饿了。”

“莱克斯……”光捂着嘴，感动的看着莱克斯。

“莱克斯，我回来了。诶，这么巧，正好开饭了哈，那我就不客气了。”这时，门推开了，朱雀仿佛又活了过来，冷静的坐在莱克斯的旁边。

“朱雀……”莱克斯回想起光的料理，仿佛已经猜到朱雀毫无表情的脸下是多么的暗潮汹涌。他不由得担心起朱雀的身体，又不能伤害到光的感情，于是，他谨慎的问道：

“没事吗？”

朱雀立刻会意，“没事，刚刚去巡查了一下，没什么事。”

“那就好。”莱克斯向战友点了点头。随后，在焰的料理的陪伴下，大家愉快的度过了中午的美好时光。

下午茶

饭后，莱克斯一刻也没有停歇，向光和焰告别后，和众人一起投入到佣兵团的工作中。

“莱克斯也太拼命了……”光坐在桌旁，托着腮看着远去的莱克斯的背影喃喃说道。

“他也是为了我们呢。”焰一边收拾着餐具一边说，“小光，我们下午去逛街吧～”

“好呀。”

两人手挽着手走在冯斯卖姆热闹的集市中。光很快被一阵熟悉的香气所吸引，于是，她顺着香味走去，对焰指了指不远处的甜点店，说，“焰，我们去买这个！”

“好啦，小光每次都要来买呢。”焰笑着拿出钱包，给小光买了5个云海蟹奶油土豆饼。店老板微笑着接过金币，将打包好的食物递给了光。光迫不及待的接了过来，闻着土豆饼的香气，露出了心满意足的微笑。

“对了焰，我们去前面的书店吧。”

“诶？”

“焰不是一直想去买料理书嘛？”光笑着拉着焰向书店走了过去。

“可是……”

“没事的，莱克斯和我都想尝尝焰做的新的料理呢。”

看出了焰的犹豫，光立刻把莱克斯搬了出来。果然，焰点了点头。走进店铺里，找到了那本书，正当付钱时，焰才发现，今天带的金币不太够。老板似乎认出了焰，笑着说：

“你是——弗雷斯贝古的团员吧！”

“咦……嗯。”焰点了点头。

“喜欢的话就拿去吧。”老板笑了笑，将料理书打包好递给了焰。

“诶？”焰有些不知所措。

“没事的，这个店铺都是你们团长的。不仅如此，这里绝大部分的商店也都是他的。那个叫莱克斯的男人，真是厉害啊。”

“嗯嗯……”焰和光泛红的脸颊浮现出笑意。

“莱克斯什么时候买下的这些店铺啊？”走在回去的路上，光问着焰。

“我记得，我和莱克斯第一次来冯斯麦姆的时候，莱克斯就买下了蔬菜店和艺术品店……”焰的食指搭在嘴唇上，回忆着说。

“好厉害呀，莱克斯……那时候年纪还不大吧？”

“不过已经能独当一面了呢。”焰笑着说，光也微笑着点了点头。

“你看，前面是西装店诶，印象中好像之前这里是乐器店？”光问。

“嗯……应该是的。”

“诶，我们去给莱克斯买几件衣服吧！”光立刻提议道。

“可是我们的金币不够了呀……”焰有些为难的看着手里的书，早知道应该多带些金币的……

光低头看着手里的土豆饼，像是下定了决心一般，弱弱的问道：“这个还能退嘛……”

“要不这样，我们直接去买，然后下次来的时候再补上欠款？”焰想了想说。

果然，一听到莱克斯与弗雷斯贝古佣兵团的大名，老板立刻点头同意。

傍晚时分，莱克斯结束了和新人们下午的训练。光和焰从弗雷斯贝古的后门进入村庄，看着莱克斯露出了微笑。莱克斯也看见了她们，向她们挥了挥手笑着小跑了过来。

“莱克斯，我们给你买了新衣服哦。”

“哇……谢谢你们，”莱克斯接过她们手中的大包小包，“今天下午玩的开心吗？”莱克斯一边向家中走去，一边听着光和焰不断的说着下午的趣事，心里也感到很是快乐。

“莱克斯，今天我和小光无意中得知你购买了很多店铺呢。”

“嗯……好像是有这么一回事。”莱克斯努力的回忆着。几年下来，虽然没有去追求过财富，但是自己通过打捞和佣兵团已经十分的富裕，在养活佣兵团的同时自己的生活也能十分富足。几年时间里，自己的足迹又遍布了整个幽界大陆，很多店铺自己也都是有产权的。

“先不说这个，莱克斯，快来试试看！”光开心的拿出崭新的西服。莱克斯脱下战斗服，伸出手穿上西装。光仔细的打理着衣服上的褶皱，而焰拿出配套的领带，轻轻的为莱克斯穿戴上。记忆中的莱克斯始终都是打捞服，很少有穿过其他的衣服。而如今穿上了西装的莱克斯，显得更加成熟。看着换上西装的莱克斯，光和焰晃了神。

“莱克斯……真帅气……”焰的眼神变得柔软，变得温和。

“是，是呀……”光的脸微微泛红，“以后再去参加会议什么的，就可以打扮成这样了。”

“光，焰，谢谢你们。”莱克斯牵起两人的手，紧紧的握在手中。这时，敲门声响了起来。

“莱克斯？”朱雀的声音响起，“抱歉打扰到你们了。”

“啊，朱雀啊。”莱克斯转身看去，“有什么事情嘛？”

“是的。有一件事我需要和你商讨一下。”

莱克斯看着光和焰，向她们点了点头，随即和朱雀一起走向佣兵团。

“怎么了，朱雀？”

“是这样的，莱克斯，我们在那里发现了一个奇怪的东西，需要你去一趟现场。”

“诶？奇怪的东西？”

“我不知道那是什么，”朱雀摇了摇头，“但是，我隐隐约约能感受到它的内部涌现着强烈的能量冲动，甚至有一种危险的感觉。我已经让他们暂时先封锁住那里了，我建议你最好尽快去确认一下。”

“好的，那我们明天就动身吧。”

“嗯。”
