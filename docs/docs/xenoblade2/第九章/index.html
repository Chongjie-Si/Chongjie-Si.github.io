异度之刃2

站在战壕里，麦佩尼手横在胸前，低头为死去的战士们默哀。远处的世界树闪着光辉，麦佩尼行走在难民营中，看着孩子们嬉笑打闹着。他带领着难民们跋山涉水，寻找新的家园，并向他们传输着和平的理想信念。他也是最温柔的医生，他给受伤的战士打上绷带，默默的为他祝福。

一天，他正在山里修行。途径一间木屋时，他发现木屋的门口悲惨的躺着一具尸体，死不瞑目。听见了屋内孩子的哭声与玻璃被打碎的声音，他愤怒的走了进去，却看见一个男子拿着刀，就要向一个婴孩刺去。 就在这时，麦佩尼一把抓住男子的手。男子惊恐地回头，然而麦佩尼却更加意想不到，眼前的强盗就是自己曾亲手救助的战士……一阵荒唐感向他袭来，不久后，屋内传来了男人的惨叫……

麦佩尼抱着孩子，听闻着孩子的哭声，手指放在唇间，随后，他把手盖在孩子的脸上，孩子的哭声逐渐消失了。

“神啊，这就是你所期望的世界吗……”

混战

“危，危险嗼！要是再往前一点就会被卷进去了嗼！”

世界树上，众人的前方不远处刚刚被一阵极其强大的炮火击中，世界树的墙壁上被燃烧成了一个洞。

“从哪里来的？！”

这时，梅勒芙惊恐的发现，斯佩比亚巨神兽再次从云海里抬起头，向众人一步一步走来。

“斯佩比亚？不可能！”

斯佩比亚的王都如同地震一般剧烈的晃动着，岩石震得满地都是。皇宫内，一名士兵连忙跑进来问道：

“陛下，您没事吧！”

“发生了什么？”

“刚才从巨神兽胸部放出了巨大的能量，想必这就是电力降低的原因。陛下！偏离回游轨道的巨神兽的现在位置已查明！”

“是哪里？”

“请看！”

奈费尔透过露天窗，惊惧的后退了几步：

“那、那是——”

斯佩比亚的巨神兽抬起了头，准备开始下一次的进攻。

“又开始了！不行，再这样释放能量，斯佩比亚就……”

“在发光……”世界树上的莱克斯看着斯佩比亚巨神兽的胸口又开始凝聚能量。

“还想再发射一次吗，糟糕了——”爷爷着急的说，“斯佩比亚的巨神兽的寿命据说已到了末期，如果再继续释放体内的以太能量……”

“会怎么样？！”

“关节将无法支撑自身的重量分崩离析……”

话音未落，一阵激光再次射向世界树，射向义经等人的重战舰。独角鲸号内电流呲呲的吼叫着，操作盘的表面跳出电火花。

“那家伙，竟然连巨神兽都叫出来了！”

“义经，看那里！”

弁庆惊恐的看着英维迪亚的巨神兽从云海中探出头，浮在云海之上。

“英维迪亚！麦佩尼那家伙，连各国的巨神兽都控制住了吗！”

“女王陛下！这，这是！”英维迪亚的王宫内，众人盯着眼前的世界树不知所措。

“世界树！斯佩比亚和阿卡狄亚！外面究竟发生了什么？”话音未落，强烈的激光炮从英维迪亚巨神兽的眼睛里射出，无数的激光瞬间包围起世界树，将世界树的长廊炸的支离破碎。

“彩歌！”齐格一把抓住掉落下去的彩歌。

“这家伙，打算把所有巨神兽都叫到这里来吗！”义经震怖的看着这一幕，却想不清楚麦佩尼的用意。

“义经，弁庆！”

“佐田——”

“快去，去保护真他们！”

“你怎么突然说这个。”

“就怕有个万一。”

“那阿卡狄亚怎么办，他们要把全世界的巨神兽都——”

“我来想办法。”

“想什么办法——就你一个人吗，办不到的！”

“没事的，这东西原本就是我组装的哦。我可以控制一切，而且——”佐田回头看了看冻在冰块里的劳拉说，“我还有可靠的同伴啊。”

“佐田……”

“拜托了，代替我……”

“——明白了。弁庆！”义经随即对弁庆示意道。

“啧！”

“弁庆！”佐田的声音响起，“我爱你……”

“恶心死了，佐田！待会儿再来揍扁你，你可别给我死了！”

看着弁庆终于消失在自己的眼里，佐田彦喃喃自语：“那——倒也不坏呢。”佐田彦在操作盘上输入一串指令，独角鲸号再次发生巨大的变化。随着机械交错杂合的声音响起，独角鲸号的外貌变得如同巨神兽一般。

“看他们的船！”莱克斯指着独角鲸号，对众人说。

“那模样，简直就是阿鲁斯！”

独角鲸号内，佐田彦推动着旋钮大声吼道：“要上了，臭和尚！”巨神兽般的船向阿卡狄亚巨神兽挥去了拳头，狠狠的打在巨神兽的额头上，瞬间碎石迸出，阿卡狄亚巨神兽发出惨烈的叫声。

“打起来了，”齐格急忙对众人说，“好机会，我们快去追真他们！”

“但是阿鲁斯——”

“莱克斯，现在我们必须做力所能及的事。”梅勒芙打断莱克斯道。

“梅勒芙——”

“没事的，陛下是我的弟弟。”

莱克斯放下了执念，“——我知道了。”

此时，独角鲸号里警报连连，斯佩比亚的炮火猛烈的席卷着小小的船。佐田愤怒的吼着：

“碍事，给我闭嘴！”随即，独角鲸号的拳头里射出强烈的激光炮，扫射着斯佩比亚巨神兽，瞬间产生的爆炸波席卷着斯佩比亚大大小小的地方，王宫也剧烈的晃动着。

“陛下！”

“没事的，先别管我，向整个王都发布避难通告，允许使用所有的军船，包括我私有的在内！”

“最先避难的应该是陛下！”

“不用管我，快去！很遗憾，斯佩比亚已经不行了……”

“陛下……”

“快！”

“是！”

奈费尔失神而有无助的看向世界树，喃喃说道，“姐姐……如果你在那里，就拜托你了。请拯救斯佩比亚，还有这个世界的未来……”

“圣座，敌人出现了！”阿卡狄亚教廷内，僧兵向麦佩尼汇报着最新的情况。

“那是伊拉失落的技术吗，真是有趣。”麦佩尼身前的屏幕上出现了佐田彦的脸，佐田不屑的盯着麦佩尼，开口道：

“吓到了吗，我们可是很顽强的！”

“伊拉的残党吗，没想到你和那个男人一样，作为食人种活下来了。既然如此——”麦佩尼额头的核心水晶再次亮起，然而并没有向他预想的那样，佐田居然毫无反应。

“真遗憾，这对我无效！毕竟我和你是一样的嘛！”

“什么！”

“我是食刃种！”

“怎么可能！食刃种的技术只有我们阿卡狄亚才——”

“没错，可是，应该不止你一个人吧，你在为自己移植核心水晶之前做过些什么！”

麦佩尼猛然回想起自己曾经抓了一堆人作为试验品，将核心水晶移植到他们身上。“难道你是那时的难民的幸存者？可是，那些人已经全部——”

“你是想说处理掉了？可是有的人死不了，也不能死！无论如何也不能啊！”

独角鲸号伸手死死的抓住阿卡狄亚，透过显示器，佐田清清楚楚的看着人们在战火中惊恐的躲避着逃跑着。

“你都没让人避难吗！还是老样子啊，麦佩尼！”佐田怒吼着，控制着船一拳挥了上去，看着四散的百姓，佐田喃喃说道，“抱歉，我决不能在这里退却！”船身猛烈的喷射着能量，这时，有一道激光射向了独角鲸号，洛修利亚的玄武也浮出云海。

“老爸！”

“连洛修利亚也……”

“为什么——彩歌！”

“不行，我的话已经传达不到了……”

“可恶！那个教皇到底在想什么！他想抢在灭之前毁灭幽界吗！”

“他在焦急。”梅勒芙对怒吼着的齐格说。

“焦急？”

“为未能得到天之圣杯而焦急。那个男人，得到天之圣杯究竟是想做什么……”

“我们必须想办法阻止巨神兽之间的战斗。”莱克斯说。

“可是那种空战要怎么才能阻止啊！”

“对了，就是那个光柱！”莱克斯眼睛亮了起来，“刚才其中一根瓦解时，麦佩尼的力量地瞬间削弱了！”

“也就是说，打烂那些柱子就能解放巨神兽们？”

“应该吧，光？能行吗？”莱克斯转身向光询问道。

“只需要瞄准柱子对吗？”

“嗯！”

“它们动得那么激烈，我无法校准，绝对做不到。”光摇了摇头。

“怎么会——”

「对了，小光！」

「嗯？」

焰变了出来，看着对决的巨神兽们，说，“只要能更加靠近它们，也许就能办到了。”

“真的吗，焰！”莱克斯一脸惊喜。

“是的，请回想一下和真战斗时的情况，当时我们改写了周围的物理法则，得以追上真的速度。同样，只要以那两只的战斗为中心改写法则……”光出来后接着说，“在我们眼中，它们和停止没什么区别，是吗。”

“可我们要怎么去那里呢？”尼娅问道，这里距离阿卡狄亚中间只有下方遥远的云海。

“这件事虎有个好主意嗼！花！你能拖着大哥和小光把他们送到那上面嗼？”

“可以！只是两个人应该可以做到嗼！”

“她说可以嗼。”

“那就拜托你了，”莱克斯说，“大家继续追赶真！”

“可是——”

“现在没时间商量了，”莱克斯对齐格说，“没事的，我们会想办法解决的，对吧？”

“大哥，花的推进器使用时间有限嗼。”

“知道了，尽可能的接近他们吧！”

莱克斯等人飞速的奔向一个露台，看着眼前正在拼命进攻的伊拉战舰和阿卡狄亚。

“从这里应该可以，花！”

“明白了嗼。”花飞到两人面前，伸手拉起两人，“两位请抓紧了嗼！”

“拜托了！”

教廷内，麦佩尼看着花拉着两人，向他飞来，陷入沉思：

“是莱克斯和圣杯？ 他们打算做什么？难道是……”

突然，他意识到大事不好，

“击落他们，别让他们靠近！”

花拉着莱克斯和光飞在空中，灵活的躲避着阿卡狄亚的子弹，离阿卡狄亚越来越近。

“莱克斯！”花空中旋转着将莱克斯和光甩了出去。随后，光立刻撑起防护罩，两人直奔阿卡狄亚巨神兽而去。

“莱克斯！”光喊道。

“就这样发动，可以吗！”

“交给我吧！”

“位置不错，要上了！”

光的核心水晶闪烁着光，变成了光焰，莱克斯高高举起剑，营造出一个能量场向阿卡狄亚卷携而去。光焰紧紧盯着增幅塔，终于找到合适的机会：

“成功锁定，塞壬！”

瞬间，天上降下来无数的光柱，射向阿卡狄亚背部的增幅塔。强烈的爆炸气流将两人冲击着装向世界树，两人向云海坠落。莱克斯在空中向下方的光焰伸出手，“光焰！”

“莱克斯！”光焰张开了双手和莱克斯抱紧在一起，随即莱克斯伸出左手发射钩锚，“一定要够到啊！”钩锚牢牢的插在世界树的铁管里，莱克斯抱着光焰顺势摆到了露台上。

“成功了吗！”

远处的阿鲁斯们逐渐停下了进攻。

“停止了……”

“那小子，竟然干成了！”齐格高兴的跳了起来。

“花，干得好嗼！”

“那是当然嗼，主人。”

“圣座，增幅塔……”

“何其愚蠢！”麦佩尼狰狞地看着众人，“你们知道自己在做什么吗！”

阿卡狄亚巨神兽瞬间向正在高兴的众人发射出能量炮，众人甚至还没来得及做出反应，就……

佐田的选择

尼娅害怕的捂住了眼，意想中身体的融化并没有出现，她迷茫的睁开了眼，突然发现，伊拉的战舰挡在他们的身前扛下了这必死的一击。

“为，为什么……”

“你在这种地方磨蹭些什么，尼娅！”

“这声音——是佐田？是你把那个……”

佐田看着眼前的一群人，开口道：

“你们不是要追上真和灭吗，那就快走啊！”

“你为什么要……”

“不知道，我也搞不明白了。”

“佐田……”

“我最讨厌这个世界了，尤其是人类！”

“你原本也是——”

“可是啊，”佐田的声音变得柔软，“我也有一群最喜欢的人。”佐田的目光变得柔和，500年前的伙伴们，阿德尔、劳拉、真，好像一一出现在眼前。

“看到你们，我就想起了他们。如果那时候，他们能够驰骋这个世界，现在又会怎么样呢？我想到这个，就完全不明白了。”

“佐田彦……”

“你叫莱克斯吧？”

“嗯——”莱克斯回应道。

“那么你找到答案了吗？”

“找到了，最棒的答案！”

“是吗……”佐田彦笑着看着莱克斯，“那么……真，就拜托你了……去吧，去让他们看看，看看你找到的答案！”

独角鲸号转身向着阿卡狄亚奔去，佐田彦幸福的笑着。500年前，无数的光柱从天而降，本应该丧生的佐田彦却被他的同伴——一名古拉少年所救，挡下了致命的进攻。500年前，佐田彦是多么的难过，他看着自己的伙伴在自己的怀里，逐渐失去了呼吸。而今天，他却救赎了一名古拉少女。

“米尔托，我没让你失望吧。”佐田缅怀着他的同伴，随即坚定的看向阿卡狄亚。

“已经够了吧，麦佩尼！”

独角鲸号抵住阿卡狄亚的下巴，将其狠狠的砸在世界树的枝干上。

“你——也太小看我！”麦佩尼一声令下，阿卡狄亚巨神兽的嘴里涌现出清冽的能量波，向着正在靠近的独角鲸号射去。在剧烈的光波中，佐田看到了500年前，儿时的自己站在废墟前无力的哭着。这时，一个天使般的姐姐向自己走来：

“怎么了？就你一个人？”

“好像没有其他幸存者了，”一个白头发的异刃说，“看这烧毀的模样，不像是灭下的手，多半是人类……趁着世界混乱进行掠夺吧。”

劳拉随即看向佐田，问，“如果你愿意的话，要不要来姐姐这里呢？没事的，不用担心，大家都是好人哦～”

真蹲在佐田的面前，轻声问道：“走的动吗？”佐田点了点头，真轻轻的拍着他的头，夸奖道，“是吗，真了不起。”随后拉着他，和劳拉一起，向远处走去……

“抱歉……真……”佐田看着最后的一抹光亮，“我先和劳拉一起……”

独角鲸号瞬间被炮火撕裂成碎片……

齐格和已是教皇的麦佩尼站在教廷的露台上，看着下方的难民们为生计而奔波。

“——看起来怎么样？”麦佩尼问。

“还能怎样，都是拼命求生的人的模样。在如今的幽界，也不稀奇。”

“你杀过人吗？”

“没有，我没必要这么做。”

“哦？”

“所谓的杀人呢，是因为嫌对方碍事，或是憎恨对方，又或是因为自己软弱。我不软弱，所以没必要这么做。我可不是在说腕力哦？我指的是这里。”齐格拍了拍自己的胸口，回应着麦佩尼。

“那我——就是软弱的人类。由于软弱，才站在这里。”

“你讨厌人类吗？”

“不，我讨厌世界。”

齐格站在世界树的过道里，从回忆中醒来。随即，他问莱克斯：

“小子，你喜欢这个世界吗？”

“怎么突然问这个？”

“你老实告诉我。”齐格的表情变得严肃。

“喜欢啊！虽然有很多令人不安的事，可也有更多美好的事物。”

“那你喜欢我吗？”齐格问。

“啊？”

“哦？难道是爱的告白？”尼娅调侃道。

“咦！？”焰捂着嘴一脸的不可思议。

“笨蛋，我是问你喜欢人类吗？”

“这样啊，吓我一跳。”莱克斯长舒了一口气。

“你有什么好吓一跳的，”齐格笑着说，“所以，你怎么想？”

莱克斯抱着胸认真的思考着，“这很难回答啊。有开心的事情，可是也会吵架，也有像会长那样可恨的家伙……”

“确实。”

“可是，如果这一切加起来才是人生，那我还是喜欢。因为如果不是这样，也没有现在的我了吧？”

“你也太自恋了吧？”尼娅忍不住吐槽道。

“哈哈哈，是吗，”齐格笑道，“很有小子的风格，我放心了。”

“什么嘛？”

“真……”

真站在原地，仰望着天空，露出了笑容。他的身体逐渐破碎，变成圣洁的粒子，消散于风中。

“做得真漂亮啊，真！”爷爷感慨万千。500年前，爷爷亲眼目睹着真烧毁了自己的日记，连带着那间曾经装满回忆的木屋，走上了一条不归的道路。500年后，虽然走了弯路，可是，这时的真，才是那时的真。

雨淅淅沥沥的下着，这座寂寞的城市里，少了一个天使般的笑容，多了一个悲伤的人。

“好臭啊，这城镇都不管乞丐吗？”从雨中漫步而来，灭看着远处的地方，缩卷着一个身披斗篷的人。那个人把自己牢牢的包裹起来，靠在一旁的木箱上。

“你——是异刃吧？你的御刃者呢？难道是在这附近忙着喝酒——看着倒也不像。”

灭并没有得到回复。自调谐以来，灭是孤单的，也是孤独的。他凝视着眼前颓废的异刃，看着他的额头微微抬起，空洞的眼神从斗篷里露了出来——那是一种想要消失的眼神。灭伫立在他的身旁，雨下得不停歇，两人就这样在雨中，站着，坐着。

“我是灭。我和你一样。”灭向坐在地上的异刃伸出了手。异刃缓缓地抬起头，万念俱灰的眼神中透露出一丝幻想和希望。他额头的核心水晶闪着鲜血般的光，一头白发下，憔悴的面容显得更加令人悲伤。

真缓缓地抬起了手。

雨淅淅沥沥的下着。

The End
