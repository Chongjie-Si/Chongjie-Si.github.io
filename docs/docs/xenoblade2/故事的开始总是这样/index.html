故事的开始总是这样

Posted on 2023-05-23 In docs Views: Waline: Word count in article: 471 Reading time ≈ 1 mins.
异度之刃2

在一片云海中，一个带着头盔的人正向着海底的岩石发射自己左臂上的钩锚。他慢慢的向海底靠近，在海底的箱子上安装一个气球。随着气球逐渐膨胀，箱子也慢慢的上浮，直至露出云海。

终于，他把箱子搬到了一块空地之上。这块空地看起来毛茸茸的，甚至还有些青绿。他脱掉头盔，长叹了一口气：

“唉——还是应该换一台探测器的。竟然往东偏了150之多，简直难以置信。”

“宝物情况如何，和你的预测一样吗，莱克斯？”一个苍老的声音响起。

“看起来还挺结实的，也没什么加固的必要，扣掉劳务费，应该还能赚上一笔。”

“计算打捞物的构造花了整整两天，算起损益倒是挺快。”说着，不远处的“陆地”动了起来，一只巨大的生物回头看了看莱克斯，他的脖子毛绒绒的，头顶还有一只长角。

“好烦啊，爷爷，我是有生意头脑的打捞员啦！”莱克斯一边说着，一边卖力的拿着铁锹努力的想撬开刚刚打捞上来的箱子，“你就不能夸我一句聪明能干——吗！”话音刚落，箱子也随之被撬开，出现在眼前的是一只巨型螃蟹。

“呜哇，吓到我了，原来是这家伙的窝。”

“莱克斯！”

“别担心！”说着，莱克斯回屋拿了一把刀，“今天的午饭就是它了！”

The End
