后记

Posted on 2023-06-06 In docs Views: Waline: Word count in article: 1.3k Reading time ≈ 1 mins.
异度之刃2

这是一次报复性的写作，目的是为了宣泄心中的情感。接连写了十几天，写到第七章时虽然也想过放弃，但是，最终我还是坚持下来。我眼睛里没有疲惫，只有泪水和感动。

当结局结束，女主全部回来的时候，游戏也跟着戛然而止，我的心情也随之空荡荡的，总感觉缺少了点什么。或许就像网上有人说的，这就是一部优秀的JRPG作品给人的感受：游玩的过程中感受不到时间的流逝，结局的那一瞬间突然戛然而止，甚至还会期待有后续的故事发展。

然而我知道游戏也好剧情也罢，结束了就是结束了，之后不会再有他们真正的发展，只能自己进行脑补。故事虽然是一个好的结局，但我的心里始终十分悲伤，就仿佛结局并不圆满，最重要的人离我而去的感觉始终缠绕着我。或许这就是入戏太深？

感触最深的是，当我打完最终的boss，看完结局时，如果想再进入游戏，就会回到最终战争之前的存档。和塞尔达传说不同，我知道最终一定是王子拯救了公主，在异度神剑中，我是知道女主光焰会为了男主莱克斯，为了大家的安危选择牺牲自己。因此即使我逃离了那个画面那个地方，去了别的场景里，我还是会忍不住去想到最终女主还是要和男主分开。兴许是逐渐成熟年岁逐渐增长，我对这种悲伤的故事也逐渐丧失了抵抗力。

片尾曲响起，整个故事到此告一段落，我随着歌声在画面中回忆着现实世界中短短的几十个小时而游戏中也许是若干岁月的时间发生的故事。从莱克斯与焰的初识，他说他一定会带着焰去向乐园，找到理想的国度；她毫不犹豫地给出自己一半的生命，将牵绊深深的种在彼此的心中；到古拉的夜晚满天繁星，莱克斯与焰坐在篝火旁，由于水晶的牵绊两人受伤的位置都一模一样。焰的手指间掐出一团火苗，她轻轻的吹走火焰，化为满天的光；再到莱克斯真正理解了光焰的痛苦，她们畏惧着自己的力量，害怕再一次发生500年前的事情，害怕伤害到自己身边的人；她们甚至认为离开莱克斯才是最好的选择。而当莱克斯真正明白并接受光焰的恐惧时，莱克斯才真正的成熟；再到经历了一切，游历了诸国的主角团们站在神的面前，毫不犹豫地说出“感谢你创造了我们”，而神的回应是“我还能再次直面你吗”，我感慨万千；直到最后光焰和莱克斯隔着遥远的距离相望，“莱克斯，你一个人也应该可以了吧”。看到这，我的鼻头酸了起来，眼泪便开始止不住浸润眼眶，直到现在，写下这段文字时依旧如此。

套上了老久滤镜的画面如同回忆一般将过往的经历徐徐放映，我看着剧情的片段，仿佛早已置身于剧情之中，就像真说的，“你早已置身其中”一般，无法从剧情里苏醒。画面一转，又回到了古拉的夜晚，坐在篝火旁的焰和莱克斯抬头仰望着满天的星光，高大的世界树周围荧光点点，焰吹起的火苗化成光线飘上天空，仿佛就是光焰拯救世界之后天空中滑落的流星。从空间站回到幽界的主角团们成双成对，彩歌紧握着齐格的手臂，梅勒芙和迦具土在一边微笑的看着，花抱着虎，而尼娅也靠在白虎的身上。只有莱克斯一人，望着暖金的晚霞，不知是不是在回忆着一路的曲折，是不是在回忆着记忆中的光焰。满天的荧光逐渐落下，终于有一粒落到了核心水晶上，水晶再次调谐，终于，光焰站在另一端，她们并没有离开。莱克斯静静地看着她们，花毫不犹豫的拥抱了上去，而经历了这么多的莱克斯，最终坚定的向她们走去。

于是，少年遇见了少女。

The End
