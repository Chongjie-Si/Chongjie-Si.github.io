“什么就是这样嗼！你是笨蛋嗼？如果不按计划死在那里嗼，之后对方让我退钱怎么办嗼？”巴恩会长狠狠的拍着桌子，对着桌上的通讯镜说。

“咦，死——退钱是什么意思？”镜子里的普宁问道。

“哼，和你无关嗼。之后莱克斯和他的异刃去哪里了嗼？”

“这个，驮着莱克斯的巨神兽从特尔奈亚海向南边逃走，之后就不知道它的下落了嗼。当时风暴实在太厉害……”

“不知去向！嗼！”

“是的嗼……”

“然后你逃回来了！嗼？”

“是，是的嗼。”

“呜嗼嗼，我不想听你的借口嗼！”巴恩会长愤怒的拍着桌子紧盯镜子中的普宁，“赶紧滚回来嗼！接下来的工作堆积成山了嗼！”还没等普宁回话，巴恩就挂断了电话。嗼嗼嗼——我还给涡潮号买了保险，这下可血本无归了嗼。话说回来，特尔奈亚海往南，也就是说，以这个季节来看，他们可能去了古拉嗼。

“帮我呼叫古拉的莫夫领事官嗼！”巴恩对着身边的人说。

“遵命，请您稍等片刻。”

员工对着镜子指指点点比划了一会，镜子中浮现出另一个中年男人的面孔，随后一个谄媚的声音响起。

“也罢，比起这些事，我有一个好消息嗼。”

变小的爷爷

“嗯——”

终于，莱克斯从睡梦中睁开了眼，可他的神智还是混沌不清。隐隐约约，他看见焰正在低头看着自己。焰的表情变得明亮，脸上也露出了笑容。

“——焰？”

“太好了，身体没有哪里不舒服吧？”

当焰醒过来时，她发现自己正处于一片密林之中。幸运的是，莱克斯并没有和她分散，但是也许是过于疲劳，莱克斯一直在昏睡着。终于醒了啊，真让人担心，焰心想着。

“身体？嗯，应该没事。这里是？”莱克斯枕靠在焰的腿上，逐渐清醒过来。

“我不知道，我们好像漂流到了某处的阿鲁斯身上。”

“阿鲁斯？”莱克斯迟疑了会，突然想起之前与灭的战斗，瞬间坐了起来。

“爷爷，尼娅他们呢？”

焰摇了摇头。莱克斯四处张望，发现了不远处地上他的打捞员头盔。

“找找看吧，应该也漂流到这一带了。”

“好。”

莱克斯和焰顺着丛林深处走去，小路的两旁郁郁青青。这里像是刚下过一场雨，空气中弥漫着青草的气息。很快，他们看到了青龙爷爷巨大的身影。青龙爷爷的身上插满了枪，他整个身体瘫倒在地上，显得苍白无力。

“爷爷！”

“你平安无事吗，莱克斯？”爷爷努力的睁开眼。

“好过分……”焰看着青龙身上的伤口，难过的说。

“等等，我这就拿伤药。”莱克斯慌张的翻着自己的背包。

“你的药啊，就算有一千人份，也不够我用。”像是行将就木的老人，爷爷的声音没有一点的力气。

“可是……”

“这也是命运。不要哭，莱克斯。”

“别提这种……没法做到的要求……”莱克斯低下了头，泪水在眼眶打转。

“离别只是短暂的片刻，在以太的指引下， 终将重逢……”说着，青龙爷爷的身体开始变成闪烁的粒子，变得逐渐虚无。原本巨大的身躯，此时此刻也慢慢消失。莱克斯痛苦的看着这一幕，焰难受的也捂住了嘴。

“和你度过的每一天都很开心……再见了，莱克斯。”

“爷爷！”莱克斯伸手想去抚摸爷爷的身体，可是，他的身体最终变成了光粒，随风而逝。莱克斯努力的用手去抓住这些光粒，却也无济于事。

“爷爷！啊！”他瘫坐在地上，泪水大滴大滴的往下掉。

“不是说了别哭吗，莱克斯。”

“莱克斯？”

“莱克斯！”

“爷爷！”莱克斯闭着眼仰天吼道。

“莱——克斯！”

莱克斯睁开眼睛，刚刚好像有一个声音在呼唤他，甚至还破音了。他低头看去，顿时被震惊的说不出话。在他的面前，站着一只小小的阿鲁斯，还拖着长长的尾巴。他擦了擦眼睛，再三确认。

“咦咦咦！！你——你是爷爷？？！”

“怎么会？”焰吃惊的看着这只小小的阿鲁斯。

“那是当然了，认不出来吗？”

“呃——认不出来。”焰略带抱歉的说。

“谁认得出来……”

爷爷哈哈大笑，“总而言之，将全身代谢提升至极限，以维持身体机能，其结果直接退化成了幼体。”说着，爷爷扑腾着翅膀飞了起来。莱克斯翻着白眼，“巨神兽也太方便了吧。”

“可不是所有阿鲁斯都能做到哦，也就是我才能做到。”爷爷叉着腰一脸的骄傲，“咦？莱克斯你生气了？”

“也没有，只是觉得哇哇大哭的自己简直像个笨蛋一样……”莱克斯扭过头没好气的说。

“怎么，还是生气了啊。”

“说吧，什么时候能变回去？”

“这个嘛，有300年就能恢复原状了吧。”

“3，300年！ 到时候我都死了吧！”

“嗯～估计十有八九。”

“那我该住哪儿啊？我难道是无家可归了？”

“以此为契机开始独立生活吧？”爷爷提议道。

“那得付房租的嘛。”

“你真小气～”

“你应该说我现实，不过，你没事就好。”莱克斯长舒了一口气。闻言，爷爷给莱克斯竖起了大拇指。

“还没到放心的时候，得找到尼娅他们才行。”

“尼娅？就是那个和你在一起的御刃者和她的异刃吗？”

“嗯，是她救了我们。”

“希望她平安无事。”焰默默的祈祷。

“爷爷，你还记得什么嘛？”

“来这里的一路上，我好几次撞上了树木，也许那时掉下去了。”

“好，那我们去找找看吧！”

“说起来莱克斯，你胸前发亮的那个是核心水晶吗？你身上到底发生了什么？”

“这件事稍后再说，”莱克斯将爷爷抱起，放在自己背后的头盔里，“现在搜寻尼娅他们优先。”

“哦！！这样还真轻松！”爷爷兴奋的坐在头盔里，感慨道。

一行人朝着树林深处前进，突然一阵刀剑交锋的声音传入耳中。

“这声音是——”

焰看着前方，“大气中的以太也在振动，是御刃者和异刃在与什么战斗。一定没错。”

“我们去看看！”莱克斯小跑着向前，不远处，尼娅正与密林中的怪物战斗。莱克斯立刻拿出武器帮助尼娅战胜了怪物。尼娅感到庆幸不已，幸亏莱克斯来得及时。

“你们怎么会……不，好像多此一问了。”

“是啊。”白虎说。

“嗯，不过幸好你们平安无事。”

“你们也是。说起来，当时救了我们的那个大家伙，那个巨神兽——”

“你在找我吗？”爷爷从头盔中探出头，向尼娅挥手打着招呼。

“咦！骗人吧！为什么？”

“我们这里也说来话长。先找个能安顿下来的地方吧？到时候再细说。”

“我也认为那样比较妥当，大小姐。”白虎看着尼娅，询问着尼娅的意见。

“嗯，就这么办。”


一行人辛苦跋涉，终于找到了一块可以落脚的地方。夜幕降临，焰为大家点燃了篝火，他们围坐在一起，讲述彼此发生的事情。

“是吗，你要和她一起去乐园啊。”尼娅一边说着，一边看着莱克斯和焰。“我还没有好好道谢呢，谢谢你救了我。白虎告诉我了，是你们把我带到了这里。”

“巨神兽大人，非常感谢。”白虎弯腰向青龙爷爷表达谢意，青龙爷爷摆了摆手，说，“不必道谢，你们也救了莱克斯。”

“不用谢，这点事。不过，阿鲁斯真方便啊……”

“不是所有巨神兽都能做到。”爷爷又骄傲地叉着腰说。

“ 麻烦你不要重复那些话了。”莱克斯吐槽道。

“什么‘不要再重复’，你这是什么态度。这一切还不都是因为你接了那种莫名其妙的工作，说什么爷爷就在这里好好放松吧，转眼跑得不见人影！”

“啊好吧好吧，明白了，都是我不好，对不起！我错了！”

“道歉也太敷衍了事了，完全看不出你的反省之意。”青龙爷爷摊着手，十分无奈。

“换谁都不会反省吧。”

“为什么？”

“我要是没去那里，焰就会任他们摆布了。”莱克斯低头说着。

“莱克斯……”焰看着莱克斯小声的念着。

“决不能让那种事发生，不能把焰交给那种家伙……”

“嗯～哼。”爷爷接受了这一现实。

夜越来越黑了，尼娅靠着白虎陷入梦乡，莱克斯也侧身睡着，缓解一天的疲劳。焰一个人站在河边，忧心忡忡的，不知道在想些什么。

“怎么，你还没睡呢。”

“不知为什么睡不着。好久不见了呢，青龙爷爷。”

“唔——和以前相比，你变化很大啊。”

“这期间发生了很多事。”

“感谢你把生命分给莱克斯。即便如此，我仍然想问你，你对莱克斯说的，是真心话吗？”

“是的。那是我真实的想法。”焰逐渐低下头，声音也越来越小。

“是吗——那我选择相信你，毕竟那不是出自他人，而是你的嘴里。”

“但是，我又多了一个目的。”

“真，和灭嘛？”

“不能放任他们，继续为所欲为。”

“天之圣杯，永恒的宿命啊。”

“是的。”

“要把莱克斯也卷入进来？”

听了这话，焰把头低得更厉害，沉默不语。

“我并非在责备你，无论你的本意如何，他都不会视而不见，这是那孩子的性格。”

焰默默的把手搭在胸前，不知如何回答。

“胸前的核心变成一半了。看来你也背负起了责任啊。”青龙说。

“青龙爷爷……”

“莱克斯，就拜托了。”

“好的。”焰轻轻的点了点头。

虎

第二天一早起床，莱克斯感觉神清气爽，整个人变得轻松无比。

“好了，总之先去附近的村庄或城镇……想归想，这里到底是哪里呀？”

“这里是古拉，斯佩比亚帝国古拉领。你见过古拉的巨神兽吧。”尼娅说。

“嗯，从远处看到过。”

“就是它的肚子一带。”

“是吗，这里就是古拉啊……嗯？说起来看你的耳朵，尼娅该不会是古拉人吧？”莱克斯看着尼娅的耳朵，两只耳朵就像小猫咪一样。

“太迟钝了吧，怎么才发现？”

“古拉是大小姐的故乡。”白虎说。

“哦？那太让人放心了。”

“你要想去城镇，首先要穿过这片森林，沿路向上，就能到平原了。城镇在平原前方。”

“好，先朝那里出发吧。”

顺着小路，他们走出了森林，站在了一块高地之上，眼前的景色瞬间变得开阔。多么大的一块土地啊！山脉，溪流，瀑布，多元化的地理特征着实让众人感到无比的感动。各种说不清的动物遍布在大陆上，如此广袤的空间，莱克斯还是第一次看到。

“哇，好辽阔的平原！”焰感慨道。

“真壮观啊！”

“是啊，太让人感动了，和爷爷狭窄的背部大不相同。”

“唔，后面那句是多余的！”爷爷从头盔中跳出来。

“那边看到的就是古拉最大的城镇，托里格镇。”尼娅接着说，“我先送你们到城镇去，到了城镇，我们的任务就完成了。”

“咦？为什么？”莱克斯问尼娅。

“还问为什么，当然是因为我是不能和你们在一起的。”

“是因为那些家伙的缘故吗？”莱克斯想起了之前和尼娅同行的灭和真。

“我和他们认识还没多久，不过好歹——算是同伴。”

“他们是你的同伴？他们可是打算杀死你啊？”

“即使如此，”尼娅无奈的说，“那里是我唯一的容身之地。”

“尼娅……”

“好吧，走了！”

托里格镇是古拉最大的一个城镇，一进门，便是一个喷泉广场。广场的周围四散着许多店铺，城镇里各种商品琳琅满目，人们在此安居乐业，总的来说，生活还算是比较幸福。

“这里就是托里格镇吗——”

“一点也没变。”尼娅轻声说。

“尼娅？”

“没什么。好啦，我带你们去旅店，就在那里道别吧。”

他们一行人路过托里格镇公告板时，尼娅停下了脚步。公告板上赫然是一张通缉令，上面画着一张老虎的脸，再配上两只猫的耳朵。在画像的下方，写着“尼娅”二字。

“这是什么！难道这画的是我？！”

白虎走上去看了一眼，“这张通缉令对特征的把握简直恰到好处。”

“啊？！你说什么？”尼娅的眼睛中冒出想要杀人的光瞪着老虎。

“啊啊不，没什么。”白虎低下了头，“看起来我和大小姐的情报被混为一谈了，实在令人遗憾。”

“好——气——人！”尼娅瞬间将通缉令撕得粉碎。


在不远处，几名士兵的周围围了一圈市民。

“……下一个，没有其他有勇气的人了吗？以你的勇气，扛起未来的斯佩比亚吧！”一名士兵说，“不只是每个月的俸禄，还可以拿到抚恤金，多多立功，甚至还有机会被授予爵位。为了伟大的斯佩比亚皇帝奈费尔陛下，展现你的勇气吧！上前报名吧，未来的英雄啊！”

“嗯，那边的人群是什么？”莱克斯问。

“招募御刃者吗。”尼娅回应。

“招募御刃者？”

“最近啊，他们在城镇里募集御刃者。”

“能够调谐的人日益减少，大概军人中没有合适人选吧。”白虎说。

“募集御刃者和调谐？你们在说什么？”

“百闻不如一见。”尼娅不再理会莱克斯，静静地看着上前报名的人。

“不要去，哥哥，很危险的。”

“要是哥哥有什么三长两短，我们该怎么办呢？”三个古拉孩子在阻挡着另一个向前去报名的古拉青年。

“哥，哥哥……”一个孩子拉着青年的手，眼里充满了不情愿。

“我，我都明白。可是，如，如果我能成为御刃者……”

“让开，黄毛小子！”这时，一个壮汉一把把青年推倒在地。

“来吧，适合我的异刃啊，将你的力量借给我吧，喝！”说着，他握住桌面上的核心水晶。可是，他突然痛苦的叫出声，身体也跟着颤抖着。

“看那个样子是没戏了。”尼娅说。

“——是啊。”青龙附和着。

男子还在痛苦的嘶吼着，最终倒在地上，身体出血。

“噢，真是中看不中用，遗憾。”士兵说。

“喂，刚才发生了什么？他全身喷血啊？”莱克斯惊恐的看着躺在地上的男子，问尼娅。

“他没能承受住核心的负荷。”

“很遗憾，没有资格的人触碰核心水晶就会成为那样。”白虎看着被众人架走的男子，摇了摇头。

“成为御刃者需要资格吗？”莱克斯扭头不解的询问。头盔里的爷爷回答说，“就跟适应性有点像。”

“适应性……”

“下一个，想得到力量与荣耀，想要为帝国做出贡献的人没有了吗？嗯？是你吗，下一个是你吗？上前来吧，鼓起勇气上前来吧！”在士兵的刺激下，古拉青年下定决心走上前。

“哥哥……”

“没，没事的，哥哥一定要成为御刃者，让你们过上好日子。”说着，青年把手放在了核心水晶上。一道金光从他的胸口涌现，瞬间他被光芒笼罩。光芒逐渐消散，他的面前出现了一把长枪，身后也诞生了一个黑色的异刃。

“我，我成功了！”

“核，核心水晶，变成了武器——”第一次见到这幅场景的莱克斯甚是吃惊。

“异刃就是那样诞生的。”爷爷说。

“未免太过肤浅了，少年！”梅勒芙再次将火焰斩成两半。

“了解嗼。”花的脚下冒出火光，飞了过去。

“爆燃之剑！”

随着剑的落下，在一片烟雾中，莱克斯等人迅速逃脱。蓄水池里的水仍在流淌着，滋补着土壤。

“什么意思？”

“我梅勒芙，居然会被人在战斗中放水。”

“难道不是吗？他本可以击中我，却故意打偏。”

迦具土望着倒下的水塔，说，“看来也很擅长审时度势。”

“终有一天还会重逢，到时将再次……”梅勒芙仰望着古拉的夜空，喃喃自语。

古拉的夜空

“我想去乐园！”
