人物分析

Posted on 2023-06-07 In docs Views: Waline: Word count in article: 2.6k Reading time ≈ 2 mins.
异度之刃2

写完了这部小说之后，我还想简单的谈一谈书中的一些人物。

灭的真名叫理性，但是驱使他的却是冲动，这本身而言就已经非常荒唐。灭自从和麦佩尼调谐以来，其思想深受麦佩尼的影响，因此，他一心想要灭世。但是，随着时间的流逝，他开始逐渐怀疑这种思想是不是他自己的思想。换言之，他踏上了寻找自我的道路。他一心想要得到答案，但可惜的是直到最后才明白。灭逐渐从把麦佩尼的意志当作自己的意志，转变到了为了真而灭世上。最终临死之前，灭才明白，原来自己想要灭世的冲动来源于自己。朝闻道夕死可矣，他找到了自我，实现了自己的意义。

灭和真的友情是复杂的。500年前，灭就认识了真，然而那时真是站在灭的对立面。直到劳拉死后，真消极避世，灭把他拉了出来，并为了他而选择灭世。他们的友情是滑稽的，却又是真实的。

真是一个非常复杂的人。500年前的他，心地善良，不愿与人争斗，擅长做饭，并且一心守护着劳拉。可是真实这种守护的欲望让他走进了死胡同里。在真曾经居住过的木屋里，当劳拉看到真收起了日记时，一种对真的占有欲便开始萌生，直到临死前，劳拉的话让真彻底戴上了思想的桎梏。正如真日记里写的，他会理所当然的随御刃者而去，但劳拉的话给他戴上了枷锁，言语皆为桎梏。为了不忘记劳拉，他之后的500年度过的是如此的痛苦。他一心想要消失，但又因劳拉的话不能消失。于是，他把自己的这种无奈与仇恨转移到了世界的人们。他觉得是人类的邪恶造成了劳拉的死。人类为了掌握异刃，不惜发动战争，在他看来是可耻的。

实际上，500年来真一直试图向麦佩尼进攻，但总是搁浅而归。劳拉的死对真的人生无疑是一个具有强烈冲击性的拐点。灭发现真没有愿望，不知道自己生存的意义，所以才会说“我和你一样”。而后灭给真指出了灭世的道路——我们一起去问神，自己和异刃的存在意义；而既然这个幽界的运行方式是把异刃当做人类的工具，是错误的，那就破坏掉这个不合理的世界吧。没有活下去的意愿的真于是接受了灭指出的道路，并将其作为自己活着的一个意义。但是真和劳拉、以前在伊拉的记忆，使他内心对这条道路隐隐地有一种抗拒，他渴望听到一个更好的答案。因此，当他听到莱克斯所言，“御刃者和异刃的牵绊的真正意义”时，他慌了神，因为，这也是他500年来一直想要得到的答案。所以，在莫尔斯断崖前，他并没有全力的对莱克斯攻击而去，而是等待着莱克斯的答案，直到耐心耗尽。最终，在世界树上，通过莱克斯的回答，他得到了自己最想要的答案，也释怀了一切，又回到了500年前的那样。然而劳拉已经不在，自己也没有活下去的欲望，他最终和麦佩尼一起消失于世界树上，完成了自己最后的使命。

麦佩尼从小便目睹着社会的黑暗，儿时的母亲为了救自己，选择牺牲自己吸引强盗的注意，最终被强盗吃了。童年的阴影让麦佩尼产生了一种厌世态度，不仅是人类，也包括异刃。但是，他仍然心存希望，相信人们是善良的。他向人们讲述着自己的理想，传播爱与和平。直到有一天，他发现自己救助的士兵杀死了无辜的一家人，甚至还要对婴儿动手时，他才觉得自己做的一切是如此的可笑。悲观灭世的思想彻底爆发，他决定自己去问神。可以想象得到，他也经历了与莱克斯相同的幻境，而在环境里他把自己悲观灭世的思想当成了神的思想，所以说这是神的意志。

伊拉一行人里比较有特点的是佐田彦。500年前，他被真和劳拉收留，一直跟随着他们。实际上佐田彦是认识光的。但是，在500年前的圣杯大战中，暴走的光和灭疯狂的扫射着阿鲁斯。佐田彦的好友米尔托为了救他被塞壬的激光射中而死，这在佐田彦心里留下了疤痕。由于是混战，他并不清楚具体是谁杀死了米尔托，但由于光的辐射范围较大，所以他认为是光杀害了他的朋友，因此对光心里有的芥蒂。曾经一个沉默寡言的人，为了纪念米尔托，硬是活成了他的模样。他虽然跟着灭和真，但却并不清楚他们想要灭世的真正原因。直到在莫尔斯之地，他听灭说了之后，才发现其实自己的想法和他们是有出入的。他可以算是伊拉五人团中最为清醒的那一个。

在和光调谐之前，阿德尔便明确的知道自己的目的是为了达到灭，因此光对于他来说是一种对抗灭的武器。在他的心里，光和灭一样，都是天之圣杯，都拥有着能够毁天灭地的力量，因此他从调谐之初便抗拒着这种力量。受他的影响，光从诞生时便种下了对自己的力量感到畏惧的种子。和莱克斯不同，阿德尔最大的心愿就是回家种田，过上安稳的生活。这也决定了他无法完全发挥光的力量。

光曾经和阿德尔同行时，就经常会出现轻微的暴走——即阿德尔控制不住她的力量，不过幸亏有霞能够抑制住他的她的力量，所以没有出过问题。圣杯大战之后，她看着巨神兽沉入云海，曾经的朋友全部死在自己的暴走之下，痛苦袭击着她的思想，因此创造了一个更柔和的人格，即焰，来代替自己。500年后，莱克斯与他在古沉船相遇。此时她的想法非常简单——回到乐园，让克劳乌斯消灭自己。然而在与莱克斯的相处过程中，她逐渐的喜欢上了莱克斯，并犹豫要不要告诉莱克斯。她害怕莱克斯知道她的力量之后，也会和阿德尔一样畏惧她的力量。她并不是不信任莱克斯，而是缺少了一个契机。直到莫尔斯断崖莱克斯真心告白之后，她毫无保留的向莱克斯告知了自己的一切，这时她的目的已从去乐园消灭自己转变为想要知道自己为什么存在。对于光来说，莱克斯是她最重要的人。正如她最后所言，莱克斯给了她光明，照亮了她昏暗的道路。

作为故事的主人公，我们能够很明显的看出莱克斯再不断的成长和进步。从一开始对异刃的无知，到最终成为天之圣杯的御刃者，从不了解光焰到了解并接受她们，从只顾向前冲到最终也会留意身边人，对于一个少年来说难能可贵。在他人生的最重要的阶段，他遇到了凡达姆，他的人生导师。凡达姆教会了他战斗的技巧，更为重要的是，教会了他不要去畏惧力量，力量取决于使用者的心。因此，和阿德尔不同，他能够全盘接受光焰。从另一个角度出发，阿德尔是为了守护世界而选择力量，但莱克斯只是为了守护光和焰。最后的分别时刻，莱克斯一心想和光焰一起牺牲，不愿自己都留在世界上。但是，众人们纷纷劝阻他，爷爷也站了出来。这是为了避免他走上真的老路，产生悲观厌世的思想。

其实无论光焰选择还是不选择牺牲，她最终都是会消失的。因为她和莱克斯使用的是门的力量，而当门消失时，光焰也会跟着消失。虽然最终她们都回来了，但站在天空桥另一边的光焰，那个为了众人牺牲的光焰却再也回不来了。她也深知如此，才会那么难过。至于最终的回归，我想，那是神的馈赠。

The End
