地铁

Posted on 2021-11-13 In docs Views: Waline:

坐在地铁上的座位上，百无聊赖的看着手机闪光的屏幕打发时间。还有多久才能到站呢？或许很久吧。

“九龙湖站到了。”地铁传来沉闷的报站声。我抬头看了看地铁屏幕上的字，又低下头来。

“最近过得怎么样呢？”

“每天都是那样，索然无味。生活的意义到底是什么呢？”

“活着便是意义吧。”

“请要下车的乘客做好准备。”

生活是一趟永不停息的列车，大家在不同的地方上车，却在同一个终点站下车。命运总是如此戏谑，你永远不知道身边的乘客是谁，谁会半路下车，而又有谁会中途上车坐在你的对面或身旁。

生活是一趟不可回头的列车，一旦搭乘便向终点义无反顾地开去，永不回头。

“诚信大道站到了。”

生活和地铁又有多大的区别呢？我靠着椅子，关上手机。车厢里的人逐渐散去，我突然发现，整节车厢只有我一个人。或许生命到了最后也是这般？一个人孤独的到终点，一个人下车离去。

生活总是充满了烦恼。就像她，仿佛近在眼前，却也悄悄的离去。但就算重来，我还会有所改变么？

地铁缓缓的停了下来，我也缓缓的站了起来。耳边的风静悄悄的吹过，指间的水也悄悄的滴下。

“九龙湖站到了。”屏幕上滚动着显示出这么几个字。

我看了看笑了笑。

突然一愣，仿佛在那么一瞬间，世界都静止了。

这趟永不回头的地铁，这趟脚步不停的地铁，是回头了吗，是停住脚步往回走了吗？

我站在打开的门前，犹豫着要不要下车。时间静静地流逝，却也仿佛能听见嘀嗒的水流声。

门就这么开着，也没有急促的关门的提示音。

终于，我迈出了第一步。

当我的脚终于踏上站台，我感觉指尖多了些什么。低头一看，指尖的水也正在看着我，缓缓上升，慢慢爬到我的脸上，爬进眼睛里。

我走出站台。似乎与平常无异，似乎又有那么些区别。指示牌还是在哪里告诉着人们前进或是后退的方向，检票口还是吞咽或是吐出一枚枚硬币。我站在那里，好像与这一幕融为一体，成为画中人；又好像与这个世界格格不入。可是这终归不是我的终点站，就算下站又有什么意义？

转身回去。地铁还是在那里停着，仿佛预料到我会回头。门依旧没关，依旧没有急促的滴滴声。

我突然意识到，我踏上的是一趟特殊的地铁。

再次进入车厢里，还坐在原来的座位上。车厢里依旧空无一人，一排排扶手像不会说话的风铃一样闭着眼。

门关了。我看着窗外的广告牌开始慢慢向身后溜走，最终成为一道光影。隧道里黑漆漆一片，只有车内亮着微弱的灯。抬头看了看，车顶仿佛映着天，那么广阔，什么也看不清，什么也看不全，什么重点也把握不住。那干脆闭上眼睛吧，什么都不去想，什么都不去做，什么事情都不管不顾，任由地铁载着我，去往不知名的远方。

车停了，鸡鸣寺，40分钟前。我看着我自己站在地铁站的出口，站在十字路口的中央，徘徊在梧桐树下，徘徊在古老校区的门口。我看见我发着消息，却记不清发的是什么。

“是在最大的那个门等你么？”

“嗯，就是有四个大字的那个门。”

她来了，和别人一起。

她好像看见我了，却又好像没看见。因为我面对着她，她却与我擦肩而过。

她看了我一眼，也只看了我一眼。

有些人原来一辈子都不会与你有交集，可总在阴差阳错之下与你相遇相识相知，又在不知不觉中离你而去。

她走了，和别人一起。

我走了，孑然一身。

车停了，排练室，1天前。我看见我自己拿着琴和别的朋友一起讨论着和弦的走向，只为她来的时候能听到最动人的旋律。弦乐器唱着歌，月亮也变成湖水，爱也变成了舟。

她进来了，紫色的衣裳甚是好看。她看着我露出甜美的笑容，而我却微微失神。手中的琴弦停了下来，朋友的黑白键依旧在不停跳动。

她开心的看着排练室多种多样的乐器，忍不住想敲打架子鼓，便径自坐在椅子上，正如现在坐在车内看着这一幕幕的我一样。手中的棒槌比划着敲打下去，连鼓也闷哼出微微的叹息。

当时的我笑的多快乐，笑的多开心。

车内的我也不免嘴角上扬，仿佛我只是个看客，这些故事都与我无关。

“下一站。”

车并不会停留，好像一直也不会停留。我始终坐在车上，时间缓缓地倒退，曾经的一幕幕慢慢呈现在我眼前。

车又停了。门开了。

是到站了吗？我起身踏上站台，走出车站。

月色朦胧，出发的时候是下午，不知不觉已经是晚上了？

周围都是陌生的人，陌生的街道。几辆归途的车驶过，昏暗的灯光也阻挡不了回家的心。

不，好像也有点熟悉。

一年前。那时的我好像就这样拥有了她，拥有了一切。

一年前，我记得我就在这里向她告白，也是一个月色朦胧的夜晚。

一年前，那时的我还怀揣炙热的心，那时的她还拥有羞涩的表情。

一年前，一年的时光飞速溜走。

我站在这里，深有感触。这一切的一切逐渐熟悉逐渐清晰，却又让人无可奈何。

“在想什么呐？”

我回头发现，她向我走来。恍恍惚惚，真实而又不真实。我向她挥了挥手，却没收到任何的回应。

她双眼直视前方，面带微笑。她好像看不见我。

她从我身边经过。

就这样，再一次擦肩而过。

如果明知道结局，那么还会开始吗？我坐在地铁上，低头沉思。留在时光中的都是那些闪亮的回忆，好像所有的难过与心酸都随风而逝，在风中飘荡。

车又一次停下来，仿佛也是最后一次停下来。

“终点站到了。”

刺耳的女声响起时，也标志着旅途到达终点。

这次会回到哪里呢？我想会是第一次相遇的地方吧。记忆中的梧桐树下是那么的美好。她拿着手中的课本静静地坐在树下的长椅上。

如果不相遇，又会是什么样呢？现在坐在我身边的乘客会是她么？

她好像能看得见我，向我微微点头。

我点头致意，沉默，坐在她的身旁。

如果选择开始，便不会回头，一去不复返；如果知道了结局，还会继续么？

或许生活的乐章就是这样不确定才变得跌宕起伏。

“你好，你是？”

如果已经知道了最后还是在这里，还是在梧桐树下，我们转身离去，渐行渐远，如果。

我望着她，眼带笑意。

你的幸运，我来不及
