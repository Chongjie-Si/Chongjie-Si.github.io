回程

Posted on 2023-06-25 In docs Views: Waline: Word count in article: 400 Reading time ≈ 1 mins.

坐在久违的绿皮火车里，我的视线被窗外缓缓驶过的风景所吸引。左手轻轻搭在老旧绿色布料包裹着的硬座上，似乎还能触碰到座椅的骨架；右手托着腮，搭在玻璃窗边的窗台上，好像还能感受着风从指间流逝的速度。

坐在三人座的最里面，我的身边是和我一同出行的朋友。此时，他正带着耳机，听着音乐。我隐隐约约能够听见他的耳机里传来的歌曲，朦朦胧胧的听感更显的神秘。列车驶入隧道，我发现我的对面是一对年长的夫妇，黝黑的皮肤倒映着太阳的影子。目光交汇的那一刹那，他们毫无预兆的对我微微一笑，带着庄稼的气息与麦田的金黄。我回以微笑，微微出神，耳边传来火车进隧道的呼呼声与铁轨摩擦的声音。我低下了头，看着餐桌前铁质的银盘，随着列车的起伏而上下颠簸，微微作响。我伸出指尖轻轻搭在了边缘，盘身传来的震动透过指尖传递给我，是否身体也会随之颤抖呢？

列车驶出了隧道，阳光已然不再。星空下的铁轨向远方延伸，伴随着火车轰轰的声音，又一个好梦诞生了。
