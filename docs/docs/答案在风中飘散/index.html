答案在风中飘散

Posted on 2022-08-01 In docs Views: Waline: Word count in article: 2.5k Reading time ≈ 5 mins.

列于“世界四大博物馆”之首的卢浮宫虽有几百年的历史，但在今天看来，几百年的风吹雨打并没有消减一丝一毫他的魅力，倒金字塔形的入口依旧闪烁着贝聿铭先生智慧的光亮。这座久负盛名的博物馆可谓无奇不有，但也正因其无所不含，才会引人深思。

刻有汉谟拉比法典的石碑被安放在一个可谓不起眼的角落里供有缘之士欣赏，但这座石碑蕴含的价值岂是区区一块土地所能容纳？古巴比伦王颁布的千年法典今天却被安放在这里——卢浮宫，法国人的地盘，这也难怪为何蒙娜丽莎这个百岁小卒竟比这位千年长者更为人所知。

那么今天的巴比伦又是何种景象？残损，破败，千年的墙壁也已崩塌，最具价值的石碑也被带走。曾经东征西掠的巴比伦王国，今天也只剩下那一条千年古道默默的叹息着。见证了太多屈辱的历史，不禁回想起曾经的穷兵黩武，一块块零落的石头就这么颤抖着，在火红的枫叶上写满了春天的谎言。

不仅仅是巴比伦，耶路撒冷这么一座宗教圣城更是遭遇了多次灾难。一次次的被摧毁，又一次次的重建，每一层古迹之下又遮掩着一层古迹。就这么七次被毁，七次重建，直到今天，这里仍是一个问题地带。

历史学家们从巴布伦、耶路撒冷以及上京龙泉府等一座座城市中不断的总结教训，总结出来的结果也只是要珍爱和平反对战争，并将其编写在历史课本中。一代代的学子于是又从课本中认识了这些城市，又得到了这些所谓的教训。但真正的教训真是如此吗？

是战争将巴比伦文明摧毁，是战争掳去了巴比伦最珍贵的那些文物，但也是战争创造了波斯文明，也是战争缔造了一个更加强大的帝国。如果说法国人不该带走石碑，那么斯坦因呢？他所盗走的经卷早已不知流落何方，而天一阁的书贩也已将千年古迹烧毁干净。战争的确应该反对，但弱肉强食毕竟是古代，或者说二十世纪之前世界最主流的竞争方式。战争在带来破坏的同时，也在不知不觉中促进了文明的融合。因此纵观人类文明史来说，战争无疑是一种文明交流的媒介。但如果对于像一战二战之类毫无理由却又惨无人道的战役，“支持”二字则毫无任何理由。

但历史学家们总结的前半句话则毫无问题：珍爱和平。与战争促进文明交流的方式相比，和平往来则更具有人性化和国际视野化。和平毋庸置疑是千百年来人们一直追求的目标，直至当下依然没有实现。人与人之间的和平，国与国之间的和平，已被历史证明是一个千古难题。而人际战役、国际战争爆发的导火索，往往都是所谓的利益不调和。

苏轼在宋代可谓是风云人物，醉酒微吟的一首首词很快便家喻户晓。然而这位文豪真正受到了文豪的待遇吗？答案是否定的，不仅如此，他还遭遇了乌台诗案。一句句即兴而发的诗词，没想到经别人那么一解释，居然成了反叛的怒吼。他做梦也不会想到，自己最终会毁于自己所热爱的诗词中。也正是因为他的文采使得当代文人星光黯淡，甚至整个被埋没，才会遭到李定、王珪等人的嫉妒与迫害。

但从苏轼一人的身上不难发现，人性的弱点是人际和平的最大敌手。那么，和平的答案究竟在哪里？

国是由人组成，而国际冲突则免不了是由膨胀的人性弱点所导致。当年李白让高力士脱靴，让杨国忠研墨，可谓风光一时，而朝廷之所以同意他的种种要求，是因为大唐需要李白用胡人的语言给大荣祚回信。由此看来，大荣祚统领的上京龙泉府是多么受朝廷重视。然而就是当时这么强盛的地方，也免不了因为利益冲突而被毁灭。新进的赢家发现这里高度发达的文明无法被自己所掌控，于是一把火烧了这座城市，烧了那段狼烟的历史。今天，上京龙泉府唯一留存的一眼古井冷眼看着社会，他像一位老者，默默无声却又铿锵有力地诉说着战争带来的破坏以及对和平的渴望。

从上京龙泉府的毁灭中可以发现，战争又是因为人性的弱点所致，就连乾隆大帝也好大喜功的在避暑山庄的石碑上刻写了一次又一次战绩。不难想象，他究竟打了多少次仗，又带去了多少的泪水与哭声。那么，和平的答案究竟在哪里？

当我们在为南宋北宋的逃离而感慨，当我们在为成吉思汗的征掠而惊叹，当我们在为亚历山大的东征而淡然，当我们在为莫卧儿帝国的行哇光而惜惋，当我们成为战争的施加者、受害者亦或是旁观者，当和平终于成为人类的共识时，我们又不得不重新考虑：反对战争。

不，反对战争不是我们的最终目的，我们更应该反思战争。也许在过去，战争是文明交流的手段，但在今天，在这个社会高速发展，国际社会俨然有序的时代，反对战争则显得格外重要。因为在今天，战争丝毫不会促进文化交流，相反，他在破坏古迹的同时，也会使社会倒退。

我们应反思战争。人类从远古时期的冷兵器战争，到近现代的核武器战争；人类从古代对文明的漠视到近现代对文明的尊重；人类从过去的季度低能，到近现代生活的高度智慧，难道只因为一次世界大战就会倒退回过去吗》即使倒退，那么这样的反思又有意义吗？

答案是不容置疑的肯定。二战的阴影直到今天依旧笼罩在国际社会的上空。战后的德国、日本社会倒退了二十年，而同盟国的经济发展、社会秩序也不免遭受严重打击。这样的战争怎能不使人们去反思，如何在此避免战争。难道一定要等到人们再次回到冷兵器时代，再次回到对文明漠视的时代，再次回到低能之时，才知道去反思战争吗？那时候，一切文明也都不将存在，人类又回到了几千年前，一切重新开始。或许多年后，他们的后代又开始战争，又打到了几千年后开始反思，如此反反复复无穷无尽。

这就是人性的弱点。

然而这种悖论不可能成立，德国首相已在墓碑前跪下双膝，联合国也已成立，海牙军事法院又开始执行作用。

反思战争带来的后果，总结战争带来的教训。我们终究还是应该去敬拜靖国神社，有关机构更应该去篡改教科书去教育下一代根本不存在南京大屠杀、旅顺大屠杀等轻描淡写的事件，而酒店的管理者同样应在宾馆房间内放入战争的读本，重写历史的读本。

反思战争，同样也需要国家首相的大力参与和支持。国家首相应该告诉民众本国并没有犯下任何罪行，更应带领随从官员参拜靖国神社。

殊不知，在遥远的新加坡，有一位叫做二叶亭四迷的日本作家，这位真正反战作家的陵地居然与战死的日本人埋在一起，而且，他的墓碑前，立着的是日本大佐的铭牌。几十年来，他也许会隐隐作痛，抱头痛哭道：

“和平，你该如何解答？”

我想，答案已在风中飘散。
