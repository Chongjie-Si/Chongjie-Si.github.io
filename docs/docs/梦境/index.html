梦境

Posted on 2021-05-29 In docs Views: Waline: Word count in article: 4k Reading time ≈ 7 mins.

“毕竟几人真得鹿，不知终日梦为鱼。生活就像戏剧，清醒的人永远分不清梦境与现实。”

有人常说，这个宇宙中存在着平行世界。在那里，生活着和你完全一样的你，你的朋友，你的家人。那里有和你所生活的世界中一模一样的花花草草，一模一样的高楼大厦。而联系着两个世界的唯一途径，便是梦。

羽木也是这么想的。

羽木做了个梦。在做这个梦之前他绝对不会想到这个梦有多么的离奇，正如一件事情在未结束之前永远也猜不到结局一般。和往常一样，他在床上看了半个多小时的手机。黑暗的侵蚀下唯有手机的屏幕映射出微弱的亮光，吸引着小飞虫们前来舞蹈。

门外的蛙声阵阵，雨后的空气也变得清新起来。

羽木翻了个身，知乎上的热点完全吸引不了他的兴趣。他平躺在床上，两眼望着天花板。近六百度的近视使努力想看清天花板的纹路的他始终看得模模糊糊，但也不知道是否是因为天黑的缘故。毕竟在黑暗之中你我终将都化为黑暗，谁也看不见谁，只能听到甚是遥远的呼唤。

他想起来一个小时之前，自己站在阳台上手扶着齐腰的栏杆，望着眼前一片荒凉的田野，听着蛙叫声此起彼伏。不远处还有些灯光，那是工地上的工人居住的地方。那边交响的气氛一定更浓厚吧，他默默的想。他仿佛看见了一年之后这里拔地而起的高楼直插天空，那时候可就没现在这般热闹，但也许比现在还要热闹，至少是表面上的。抬头看看天空，黑泛紫的夜幕下还能清晰的看到云丝飘渺，一丝丝的云围成了一个巨大的锅盖，他发现他就在这口锅的中心。

或许世界就是一口巨大的锅，而今天恰好锅盖被掀开，让我得以见到更远的世界吧，他想。

看了看手机上似乎还在走动的时间，他闭上眼睛，随即又睁开。

他想起来一天之前看的一些论文，繁杂的数学公式与逻辑扑面而来，压得他喘不过气。这是一项仿佛一眼能看得到头的工作，而世间哪一种事物不是一眼能看到头呢。从开始到结束，从诞生到消弭，世间万物都仿佛在遵从着一个特定的规律，日出日落，月圆月缺。人也是这样，昨天的你，今天的你，明天的你。虽然这种规律如此模糊，以至于人类研究了几千年都没法用一个公式去描述，但是，我们却依然可以对它产生一种朦胧的感觉，那就是时间。

时间就是一种规律，你无法解释，它就在那里看着你或许慢慢老去，或许慢慢年轻。

因果也是一种规律，因有缘起，果有缘灭，一饮一啄，自有天意。

他忽然想起来自己曾经做过的一个梦，梦里的世界是那么的真实。他梦到了一个女性，他们手牵着手，从一个地方走到另一个地方。他们做的每一件事都印证着他们是一对情侣。只是，当他醒来，他却怎么也记不住那个女性的模样。不过他可以清晰地记得他们一起做过的每一件事，虽然他一件也没做过，但至少可以确定的一点是，梦里的男主长得和他一样，他也能清楚的体会到相应的感受。

“那应该就是我了吧，我也许是在看自己主演的电影。”

后来的生活一切如旧，忙碌的工作迫使他成为一个真正意义上的社畜，每天在写字楼里进进出出的身影加剧了疲惫感与无力感。

直到后来有一天，他又梦见了同样的故事。这一次醒来之后，他依然记不得女主的相貌，但依然记得他们做的事情。

想到这里，躺在黑暗中的他笑了笑，人就是这么滑稽可笑。谁能想到自己做的梦居然还有可能是续集。这种事情只可能出现在文学作品里，联想到前段时间热播的《送你一朵小红花》，他笑的更灿烂了。或许是因为自己也确实该找一个女朋友了吧，他想。

他以前追求过一个女生，也就追求过一个女生。期间经历的事情逐渐使他身疲力尽，没有信心与希望再继续坚持下去。很多事情必须在经历之后才能感受到其存在的意义，这件事之后，他对于情爱便无关痛痒，可有可无。也许是这样。

他又向左侧侧过身去。他想起睡觉时如果向左边侧身，心脏会受到一定程度的压迫。他不由得摸了摸自己的胸口，指尖微微能感受到有规律的振动。人真是一个奇怪的玩意，他想，因为有着自己独特的经历与感受，所以我才是我。如果真的有孟婆，真的有孟婆汤，那么是不是真的会有转世一说呢。毕竟在下一世，上一世的记忆全部被抹除，你又是一个全新的个体了，又从出生开始建立起独一无二的经历与感受。想到这里，他试图努力回想前世的记忆，却不禁噗嗤一声笑出来。

但如果有这么一种可能，如果你的经历和感受，你所经历的一切，都能被准确无误的复制出来，那你还是你吗，他打了个寒颤。回想起前段时间刚丢失的身份证，他顿时想到以后一定要好好保管，那是唯一能证明自己是自己的东西了。

人还真是可悲啊，连自己都没法证明自己是自己，还需要借助外物。可是不是说君子善假于物吗。

他闭上眼睛，一阵猛烈的困意袭来。

夏天的天空总是一碧如洗，晴朗时还能看到云朵飘过。他下了床拿起牙杯牙刷，站在洗舆台前看了看镜子里的自己。眼睛一夜的分泌物还挂在眼角，蓬松的头发乱的像分叉的树枝。他捏了捏自己的脸，仿佛感觉不到疼痛。或许是因为睡的太沉，神志依旧不清醒。透过镜子，他看到墙角处窗户外的天空一片漆黑。或许是昨晚雨下的太大的缘故吧，所以尽管已经是早晨九点多，天依旧像没睡醒一般。

换好衣服走出门，他想起昨晚做的梦，嘴角不由的露出微笑。他梦到了自己的家人。自己已经很久没有回家了，所以梦到他们也是很正常的事情吧。是应该回去看看了。他撑起了伞，伸手招了一辆出租车。很有默契的，他坐在了第二排靠里面的位置。他很默契的不说话，就静静的坐着，正如出租车司机默契的不说话，抽着烟，一只手夹着烟搭在车窗外，一只手抚摸着方向盘，如同抚爱他最爱的玩具。当两人的视线通过后视镜交织时，他笑了笑，司机也笑了笑。

车就停在路边，静静地不说话，像是在思索。天空依旧不明亮，明明是上午时分却压抑的像夜晚。

终于，另一侧的门被打开了。羽木抬头看了眼，眼前的这个女性仿佛在哪里见过，个子不高，瘦瘦小小的，皮肤白皙，戴着黑框眼镜，手里提着一个白色金扣的小包，侧身上了车。两人都心照不宣的看着对方。这时，他下意识的用手搂着她的肩膀，嘴唇轻轻地印在她的脸颊。一切仿佛是如此自然，两人仿佛是多年的情侣一般。她挽着他的手臂，面带微笑甜甜的问道，“中午我们去哪吃呀”。他毫不犹豫地回答，可以去楼下的KFC，也可以去分米鸡，最近新开的一家店。

司机师傅带上墨镜，轻踩离合换挡，车辆缓缓的启动。他就这么看着她。他从未想到一切发生的如此突然而又自然，他努力想看清眼前的人，这个自己亲吻过的人，这个自己牵着的人，却怎么也看不清。这种感觉，就像是当你快要看清的时候突然一盆冷水浇了上来，让原本不那么模糊的视线瞬间变得模糊。

车缓缓地驶向家，他看到他们的孩子推开门张开双臂向他们奔跑过来。他蹲下去，一把抱住眼前的孩子缓缓起身，脸上洋溢着幸福的微笑。当他放下孩子，看着孩子转身进屋时，转头看向身边的人，意外的发现她已经满头白发。朝如青丝暮成雪。

他猛然间坐起身来，眼睛挤了挤，左手去寻摸着枕边的眼镜，右手拿起手机。已经是早晨九点多了。

原来是个梦，他想到。自我解嘲的笑了笑，自己现在还是孤单一人，又何来一姑娘能够与自己牵手相拥。原来只是个梦。他想到，却又不禁有些失落。

他下了床拿起牙杯牙刷，站在洗舆台前看了看镜子里的自己。眼睛一夜的分泌物还挂在眼角，蓬松的头发乱的像分叉的树枝。他捏了捏自己的脸，仿佛感觉不到疼痛。或许是因为睡的太沉，神志依旧不清醒。透过镜子，他看到墙角处窗户外的天空一片漆黑。或许是昨晚雨下的太大的缘故吧，所以尽管已经是早晨九点多，天依旧像没睡醒一般。

他依旧感觉很是疲惫，忍不住打了两个哈欠。坐在电瓶车的座椅上，他才感觉到一种真实感。坐垫发出吱吱的声音，他感到裤子一片冰凉，原来是昨天的雨水依旧顽强地附着在坐垫上。回头看了看后座，笑了笑，他心想这可不是梦，也不会有女生会坐在上面。

就像受到一种使命的召唤感一样，他骑着车直奔某地而去。他想起来前天领导让他去拿一个快递，昨天没有拿被领导友善的提醒了一下，今天必须去拿。他骑着车，头脑昏昏沉沉，眼睛几乎要睁不开。看来昨晚这一觉睡的并不是多好。

他一边迷迷糊糊的避开着街上的行人，一边想起梦里的场景。好像跟她说可以去KFC吃午饭，他扭头看了看身后的KFC，心里想到。此时的街上人并不多，他自由地穿梭在街上，自由的奔向下一个远方。

他想起梦里的那个人，他好像见过，也许是之前梦中的那个人？他摇了摇头。他清楚的记得在梦中自己努力想要看清她的模样，那时候的自己好像是清醒的，是有意识的，就和现实世界的自己一般。

可是如此努力追寻，最终不还是什么也没得到，什么也不知道吗？

和快递站的工作人员聊了两句，他拿起并不属于自己的包裹转身离开。一种荒唐感席卷他的全身，这种并不属于自己份内的事情最终也还是要自己去做，然而属于自己的事情却没人愿意帮忙。他想到拖欠的房租，想到房东大妈凝成一根绳的眉毛，不由笑了笑。

要是生活在梦里，也许并不是什么坏事。自己苦苦追求的一切都能在梦境中得到，但梦醒之后，却发现一切都是子虚乌有，随风而去。

他突然想起梦中的自己也做了个梦，梦到了自己许久未见的家人，好像是该回去看一看。从来到这陌生的城市以后，所见的一切都是如此灯红酒绿，花天酒地，自己不过是物欲横流世界里的一粒尘埃，无福消受也没有能力享受。而自己的家却慢慢的被遗忘，自己只想着怎么能过上像那些人一样的生活。好像还有一个孩子，看来梦中的生活是多么的幸福。在外拼搏这么久，在这个陌生的城市这么久，却从未有些许的归属感与幸福感。他还记得，在梦中他与她白头偕老。

如果给你个机会放弃现在的一切，让你拥有梦中的那种生活，你愿意吗。

他有点动心。这不正是自己想要的生活吗。

但他还是选择现在的这种生活，因为一旦选择梦境，那么他迄今为止所得的名声、所拥有的财产，在梦境里都将是一场空，正如梦境里他多次梦到的那个女性角色，在梦清醒的时候也终将是一场空。

现在的生活也很幸福。他笑了笑，骑车的速度更快了。

两边的风呼啸着吹过，他拍了拍自己的脸，也许是因为神智不清醒，他并未能感觉到疼痛。

他彷佛看到楼下的她，正牵着他的手，向KFC走去。

The End
