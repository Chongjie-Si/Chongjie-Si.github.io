你不要做一个小偷

天色少有的变为绛紫色，红红的一片略微有些冷意，我不禁打了个冷战。脚下的路三年来从未改变，只是路上的人已经变了颜色。我突然有一种难言的感觉。

我喜欢在这傍晚时分站在宿舍门口的阳台上眺望北边学校暂未开发的树林，树林的另一端是青教公寓，那里作为疫情以来学生隔离的场所，现在却也看来是一番世外桃源的光景，我不禁回忆起一个月前，我只是因为小感冒，被学校医生测体温误认为发热，送去隔离的场面。

那天，李辉和约翰抱着我的住宿用品一同前往青教，李辉一直不停的感叹：

“我都说了我是傻瓜，当年感冒被以为发热隔离，劝你们感冒不要去校医院拿药，你们倒好，前赴后继，果然被隔离了，快笑死了。”我倒是觉得还好，至少有人送饭，一人一室三厅，独立卫浴厨房，条件待遇一流，也就放宽了心。

“你真的，先揍我一顿！”

此时，楼道尽头的呐喊声把我的思绪从那段美好的隔离时光中拽了回来，我定睛一瞧，柴酒李辉在走廊尽头的拐角商量着什么。柴酒看见了我，说：“让我先和李辉说几句话，晚上咱们一起去食堂吃饭吧。”我点了点头走回了宿舍，心中疑虑便跃上心头。柴酒的事不是已经尘埃落定了吗，现在又会有什么跟他有关的事情？我不知道，也懒得想。因为我早就知道，身边的故事，远比轻小说要精彩，不会像《春物》里写尽追求真物的遗憾，显得具有悲情的浪漫。哪怕我不是预言家，也早已可以对生活这个RNN网络贴好所有的狗血的基因标签。

“走吧，我们说完了，咱们去吃饭！”

不一会儿，走廊尽头的声音消弭，柴酒敲了敲我的宿舍，走进门对我说。跟着走进来的李辉看上去有些焦虑，他感觉不太放心的看着我，对柴酒说：

“别告诉九猫了，反正明天都知道了。”

我耸了耸肩，表示知不知道无所谓，吃不吃瓜，眼下已经没有未来的路更加坚实了，我热爱的兄弟们，无论怎样我都会维护和坚持吧，除了她哭的那个夜晚，那也是后话了。

去哪里吃饭，对于快要毕业的学生来说，永远是这个世界里最难抉择的问题，在这个朴实无华的学校里，也只有三个以花为名的食堂分区。想必起名的人也曾煞费苦心，可是工科学子也会在有意无意中渐淡他们的文学情怀，这又何尝不是一种挫败的成功呢？那就去梅园吧，毕竟路远话长，可以细细道来。饭好不好吃，还得自己说了算。

李辉一直感慨着离谱与惊讶，我说就你那破事，还能有多离谱。李辉自信的告诉我，说，“这远比你能想到的要更离谱！”，我笑而不言。而柴酒也履行承诺，不说明任何有关这件事的内容。

事物总会因为神秘而吸引人们的目光，我确实想知道这个“离谱”又会刷新多少我的认知，不过如果真的是无用的见识，我宁可不知便是。对于母胎单身的李辉，我倒是希望他能尽快找到一个合适的女孩子脱单，但李辉总让我感觉，眼光高，自己喜欢的类型又说不清楚，但的确愿意去尝试各种类型吧。我笑着摇摇头，既然离谱，那肯定八成是跟余文脱不了干系了。

餐桌上，我笑着对李辉说，我都猜出来了。李辉不信，我就说一定是跟你一起打游戏的余文。李辉先是点了点头，又突然猛地摇头说，“不对不对！”

“你这啥意思啊，既点头，又摇头？”

“反正你猜的不对，不过确实有关。”

“不可能呀，那会是啥？”

“等明天吧，明天所有人都知道了。”

“我懂了，你是把兄弟当成‘所有人’了，对吧。”

“哎，真不能现在说。”

确实吧，如果真的是涉及到一个人的黑历史，一经传出，就会逼的人把虚伪性无限放大，并且通过畸形的方式去匡扶每个人自己坚持的所谓“正义”罢了。人性经不起测试，这一点我深有体会。“那就等明天吧，如果你真的脱单了，记得请兄弟们吃饭。”我放下筷子笑着说道。而此时的李辉也不知道是该答应好还是若无其事的不答应好。

柴酒也看起来毫不在意的样子，对着我笑了，只不过好像有些复杂？难道是我看错了吗？我揉了揉眼睛。
今晚，风儿不关心人类，我抬头看见金星凌月的夜空，忙碌的人群也似乎以夜为中心停下来脚步，用他们的手机替他们去观赏这难得的月偏食景象。说不清是深秋还是初冬了，反正叶子还没来得及飘散，天地仿佛温柔的手，把所有人都拽进一个甜蜜的漩涡，也没人会问及，漩涡的中心，会是什么……

第二天，舍友约翰从床上蹦了起来，大喊着离谱离谱把我叫醒。我从梦中缓过神来，却又被约翰告知我的消息打入新的梦里：李辉和吴雨官宣了！

“这不就是ntr吗？柴酒会怎么想？”约翰激动的吼道。

我依旧处于震惊中，说不出话来。

我想起了柴酒以前煞费苦心追求自己喜欢的吴雨，想起了他请我和李辉帮忙出谋划策……一瞬间，一切未解开的谜团瞬间解开了，而那些生活中的线索也串接起来形成了完美的链条，透过这个链条，我更清楚的认识到了李辉与吴雨这两人的形象。我相信，在知道这个消息的瞬间，柴酒一定也会串起来更多的东西。
没用了，我又不是《人生一串》的文案编辑，无法把那些五味杂陈串接起来，也无法用自己的震惊款待客人。

我一直觉得我跟吴雨的关系很不错，较为了解这个其实也很脆弱的女孩。可今天，她却变得如此陌生了。祝福的话挂在嘴边，但却又不知如何提起。官宣的朋友圈下面，已经垒起了数字99999的长龙，但愿所追求之物真的长久吧。浪漫情怀至死不渝的来说，这么快的闪击难以理解，然而我与柴酒最大的不同就是，我仍然是个上帝视角的旁人，最多也只能见证这一切的始末，山川复归江流，日月演替交加，无处感叹！
是时候收起我的好奇心了，我知道我不是凑热闹的闲人，更不是能看穿一切的伪圣人。

李辉，你赢了，论离谱，我是万万想不到这一步的。

The End
