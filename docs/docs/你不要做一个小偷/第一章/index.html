你不要做一个小偷

“诶，你有空吗，我跟你说个事。”

我低头看了眼手机跳动的消息，手里的动作已经没有停下。

“接下来，就是见证奇迹的时刻。”

故作玄虚的我看着我的两个室友，手中的牌夹轻轻地摇晃，一张牌就在他们眼前变成了另一张。他们不可思议的表情让我十分的满意，两张脸八只眼睛充满了大大的问号。

我露出得意的笑容，那表情好像就是在说“是我干的，没错”。都说玩魔术要具有很强的信念感，看来我演的很像那么一回事。

“你直接发消息不就好了。”

“诶不行，这事必须得当面说。”

没想到这小子居然秒回我。

“我在玩魔术呢。”

“你在宿舍等着，我马上过来。”

一会儿功夫，“咣咣咣”的敲门声急不可耐的响起，打开门后，李辉一把把我拉出来。

“啥事啥事，有话慢慢说。”我有些不悦，手中的牌受惯性掉落在地上。

“诶，这事太魔幻了，我们找个角落。”

李辉拉着我走到宿舍楼道一个犄角旮旯的地方后，犹犹豫豫的说：

“要不你先揍我一顿吧！”

我懵了。

在这么一个富强民主文明和谐的时代，居然还有人这么欠揍。都说如果当事人请求别人揍他的话，揍他的人是不需要承担责任的。我摸了摸鼻子，手居然有些痒了。难不成今天就能体验到打人的感觉了吗！我的内心逐渐开始激动。不过，最终还是逐渐冷静了下来。

“你脑子有泡？”我凑近闻了闻，“酒也没喝多啊。”

“诶不是，你真的，先揍我一顿！”

越说，李辉情绪还越激动，恨不得自己痛扁自己。

“啥事啊，你做了啥对不起我的事，先自己扇自己两巴掌吧。”

一边开玩笑的说着，一边听着“啪啪”两声，我的脑袋中一边倒着带放着电影。在他没说之前，我感觉我已经猜到了一些。

好像有极大的可能性，与她有关。

“你真不打我？”

“你先说啥事，我都不知道什么事我就揍你，到底是我有病还是你有病？”

“行，你说的，你不揍我的啊，那我说了。”李辉用手指我，大舒一口气，“不许揍我。”

“吴雨和我表白了。”

时间好像静止了一两秒。

“哦，就这？”我轻描淡写的说道。

“不是，你就这反应？”李辉捶胸顿足的说道，“我都想了好久了，不知道怎么跟你开口说。”

“这有啥，你情我愿的事。你答应了没？”

“早知道你是这个反应我就不告诉你了，我还以为你肯定要胖揍我一顿”，李辉搓了搓鼻子，“我还跑去问李老师了，她让我和你聊聊。我答不答应，这不得问问你。”

“问我有啥子用，你自己咋想的。”

“那就答应呗。”

听到李辉的回复，我大受震撼：“不是，你就这？你喜欢人不？”

“反正不讨厌。”

果然啊，女追男，隔层纱。

“诶不是，这个消息有点突然，我缓缓，”我想了想，继续说道，“怎么就表白了她？”

“事情是这样的，”李辉剁了剁脚，锤了我一拳，“早知道你这么淡定我就不告诉你了。”

我看他一脸悔恨的样子，好像做了什么亏心事被发现一样，不耐烦的催了催他继续说下去。

“我这阵子在追余文。我也不怎么懂女孩子想什么，就请吴雨帮帮忙，平常会给她发消息问一问。然后前不久我约了余文一起出去吃饭……”

“等会等会，余文这么高冷的女生你都泡到了，你真牛逼。”一边说着，我一边给他比划了一个大拇指，“我去年就和你说过她很好看，你还说不是。现在是不是真香？”

“屁嘞，你觉得她高冷是因为你不了解她，她可宅了，贼喜欢打英雄联盟。”李辉翻了个白眼，让我感觉他像是和余文很熟的样子。“然后那天晚上，我间接的表白了。也不算表白吧，就是暗示了一下她？我也不知道她是真不知道，还是说装不知道，反正也没有给我明确的回复。”

“你都没有表白还指望有回复？”

“但是给我的感觉是余文知道我在追她啊。而且这么久了，我们一直相处的很愉快，我觉得自己还是很有希望的——先不说这个，你听我说，回来之后吴雨发消息问我表白成功了没，我说没有，她说，那我冲了。我当时直接黑人问号脸。然后，她就表白了。”

“这么牛逼。”我摸了摸脑袋，没忍住噗嗤一声笑了出来。

“她说她挺喜欢我的，我当时就不会了。这该怎么办啊……好烦啊。我想了想，和吴雨说让我先思考一个星期。这不，转眼明天就是最后一天了，我想了一周都没敢告诉你，早知道你是这个反应我就不说了啊！”

“你怎么还在纠结这个，你要真想让我揍你，你就直接乖乖站好，明天就让你浮肿。”

“那我答应她了？”

“这不随你，和我又没关系。”

“真的？你不介意？”

“不介意，与我无关，不要打扰我。”

回到宿舍，舍友一脸期待的看着我。我瞅着这三，一个个坐在小板凳上，猥琐的目光赤裸裸的打在我身上，让我感觉浑身不自在。

“散了吧徒儿们，没什么好听的故事，都是些鸡毛蒜皮的小事。”我摸了摸鼻子，接着说，“不过，也许明天会有事情发生。”

深秋时分，天气转凉，侧卧在床上的我缩在暖和的被窝里。

好像去年的这个时候也是这么冷。

我也是缩在暖和的被窝中。

一年前。

“你说我要不要表白？”我给九猫发消息问。

“要不试试吧，不给自己留遗憾嘛。”九猫的消息很快便弹了出来。

想想也是，毕竟喜欢她也喜欢了很久。终于要迈出这一步了。我忐忑的拿出了手机，微亮的屏幕光印在我的脸上，写满了紧张。我熟练却又不熟练的找到了她的头像，打开了和她的聊天界面，伸出却又缩回了手指。我该说什么呢？我该怎么开口呢？这是一场豪赌啊，一旦输了就再也没法回头了。终于，我下定决心，在聊天框中输入道：

“你睡了吗？”

我忐忑的等待着对方的回复。时间过的可真慢，一分一秒都像是煎熬。

“必然没有，作业这么多，我不配在这个时间睡觉。”还是熟悉的风格，是她会说的话。我紧张的敲击着键盘，手指略微有些颤抖。

“好……我想说个事，我觉得不做不说我会后悔——”

“——我喜欢你很久了，我以为时间能冲淡我的感情，可是我发现，时间对我来说……可能是停止的。”

“我说不好华丽的词藻，写不好动人的情书，我只有一个真心。”

“不知道我有没有这个机会。”

“如果我给你再次造成困扰，对不起。”

表白之前唯唯诺诺，不知道说什么怎么说，而真到这一刻，我发现想说的话自然而然就到了嘴边，变换成
键盘下的文字。

半晌，我都没等到她的回复。

我都没收到吴雨的回复。

终于，聊天框中显示对方正在输入。

“我。”

只有这么一个字的回复。我本就忐忑不安，一种莫名的兴奋与紧张交织在一起，让我的身体、我的手都在
颤抖着。我的心已经提到了嗓子眼，生怕她说出一句拒绝。

“我不知道。”

“我要怎么做啊……”

悬在半空的心瞬间落了下来。

“我真的很喜欢你，满脑子都是你。我今晚突然有勇气，我怕我不说出来会后悔。我没谈过恋爱我也不知道怎么组织语言，但如果我能有这个机会，我一定会用我的方式，给你想要的幸福。”

“我不知道。”吴雨回复我。

“我觉得你很厉害，我很欣赏你，但这不是喜欢。我觉得如果在没有喜欢你的时候就答应你，对你也不负责任。”

这是拒绝吗？好像也不算是。但这也不是答应。我连忙给九猫发消息求救：

“这我该怎么回复？她没有答应也没有拒绝呀。”

“明显有戏，你就说，‘感谢你这么晚真诚的回复’。”

顺手看了看时间，已是午夜时刻，时间果真不等人。

“感谢你这么晚真诚的回复。”一个字都不变，我直接发给了吴雨。

“我其实特别怕打扰到你，所以我总是小心翼翼。我的喜欢不是一时冲动，是我思考了很久的答案。我相
信，如果是真心，相逢的人会再重逢。”

“不早了，早点休息。晚安。”

闭上眼睛，我已经做好了听天由命的打算。就在准备关闭手机屏幕的那一刹那，吴雨的消息又出现在聊天界面中。

“我不知道，我心里真的没有底。我觉得没有感情基础的喜欢就是耍流氓。你又不了解我，为什么就喜欢
我呢。你知道我喜欢吃什么喜欢什么颜色吗……”

“算了，不为难你了，我也不知道我喜欢吃什么。”

好像有机会，又好像没机会。我最害怕的就是这种薛定谔的事情。

“我不知道你喜欢吃什么，但是我知道你不喝冷的。只要是你说过的，我都记得。”

半晌，吴雨的消息再次亮起：

“明天说行吗，我还有两个报告没写完。”

那好吧，我不再追问，希望给她，也能给我一个冷静的机会。“晚安。”

“晚安。”思绪拉回现实，我和我的室友们道了晚安，期待明天的到来。

The End
