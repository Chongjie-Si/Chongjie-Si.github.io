你不要做一个小偷

人和人的初见，总是那么美好，格外新鲜。

记得我第一次遇见她，还是在一个晚会的彩排上。那时的她扎着双马尾，眼睛周围亮晶晶的，很是好看。

记得我第一次遇见她，外面还下着淅淅沥沥的小雨，参加演出的我的裤脚边也不免沾上些泥。

记得我第一次遇见她，她坐在远远的那边，我坐在远远的这边，中间间隔着的陌生感就像是一块磁铁狠狠的吸引着我，吸引着我向她靠近。

记得我第一次遇见她，外面寒冬凛冽，里面却温暖如春，哼唱着七里香的我更是感觉到内心盘旋着的温度。

人和人的初见，总是那么美好。我早已记不清那天台上的我说了什么表演了什么，当我闭上眼睛再想起那天，那个晚上，我的脑袋中只有那个双马尾，和那双亮亮的眼睛。

适逢其会，猝不及防，你会等待还是离开？

我通过我的朋友要到了她的联系方式，忐忑不安的打开聊天框，犹豫半天也只在输入框中笨拙的用九键打下了“你好呀”三个字，然后便漫长的等待对方的“正在输入”。

后来我意识到，我总是心急的想迅速看到事情的结果，就像那时的我只在乎她会输入什么，表白之后的我会得到她什么样的回复，以及后来种种事情都希望迅速的得到解释。如果能重来一次，那种忐忑不安，那种期待的心情，那种一开始的新鲜，该是一种多么棒的体验。

可是如果真的有机会，我会选择重新来过一次吗？

我想我不会。

我特别喜欢玩一款游戏。刚开始进入游戏的世界，一切都是新鲜的，花花草草，山水天下；可时间愈久，一种孤寂感便逐渐蔓延，以致于现在有些时候我都不敢打开这个游戏。

我有些时候会躺在床上闭上眼睛想着，要是能把我关于这款游戏的记忆删除该有多好。可是随后我便否认了这个荒诞的想法。因为，即便刚开始的新鲜与探索的欲望能给我带来极大的快乐与幸福感，我也不想再次去体验去享受那种慢慢累积的寂寞感。

感情也是这样。

如果有一天，你让我随便说几句，关于这些，我想我可能一句都说不出来。

不过，也许我会写到：

“有些时候我们其实挺像小偷，只不过偷走的不是捏在手里的钞票，不是捧在手心里的至宝，而是藏在别人内心深处的情感的钥匙。”

“你偷走了钥匙，还牢牢的锁住了别人的门，把自己包装成上帝一般故作温柔的给他留了一道窒息的缝隙，徒留他在阴暗的空间里被漆黑吞噬。”

“巴塞罗纳早已没有了流浪，这个世界上又多了一个流浪的人。”
