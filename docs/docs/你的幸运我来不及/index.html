你的幸运，我来不及

Posted on 2021-09-08 In docs Views: Waline: Word count in article: 3.2k Reading time ≈ 3 mins.

我喜欢走夜路，走那种很长很长的夜路，尤其是独自一人在黑夜里摸索，直到遇见了你。

我喜欢和你一起走夜路，走那种并不是很晚的夕阳，走那种并不是很黑的光明，走那种并不是很暗的灯光，走那种很长很长的路。

我喜欢在路上与你交谈，谈天说地；我不喜欢在脚下设立目标，标准某地；我喜欢在时空中与你并肩，肩并肩向前；我不喜欢在夜幕里寻找流星，星光满地。

脚下的目标一旦明确，不禁会忘却身边的人，一心只想着赶路；夜幕的星光固然美丽，正因如此，才可能会遮掩身边的美，让我后悔不已。

那就放慢脚步，和你一起穿梭在车水马龙间，穿行在亭台楼阁中，穿越了人山人海，穿起了一切的平凡与不平凡，一切的繁华与冷落，一切的真实与虚无，一切的爱与感伤。

路很短，我们只走了很短的一截，我甚至才度过短短的时光。

路很短，我们只并肩前行了一小会，我甚至没能触碰你滚烫的脸颊。

路很短，我们只不平凡了一秒钟的时光，我甚至没能感觉樱花飘落的速度。

路很短，我们只流星了一块夜幕，我甚至没能感受到你衣摆的弧度。

路很短。

路很长。

突然手机的屏幕亮了一下，我伸出早已疲倦的手，勉强挺起躺卧在沙发上的身子，打开了手机。

“我们算是朋友吗？”

“嗯。”

“那样很好。”

要是我的回答并不是如此，又会发生什么样的情况呢？我不知道，也不想知道。我知道的，只是今晚，我们走了很久。

一

我坐在钢琴前，痴痴地看着黑白的琴键，手指不听指挥的硬是按了下去，一种沉默的棉花音打破了图书馆静谧的气氛。

“你要弹钢琴吗？”

“不不不，我不会，只是很好奇。”

我突然间起身抬头看了看眼前的你。

“我想去前面的琴行转一转。”

“那我们一起去吧。”

“那你得跟着我走啦。”

张嘉佳的一段话在某一瞬间闪现在我的脑海中：故事的开头总是这样，适逢其会，猝不及防。康德说，滑稽是理想与现实间的巨大落差。适逢其会么？应该很早就认识了吧，只是，只是由于某些原因吧，失去的太多，交流的太少。猝不及防么？很早很早就想这样走了，只是缺少了主人公。滑稽么？我已经没有心思去想康德怎么说了。我从不甘心做配角，但在这个时刻，这片夜空下，我以头顶的繁星与脚下的土地起誓，我真的真的只愿做一个配角。

暗黄的灯光还在城市中摇曳不定，街边的小摊还在时不时响起吆喝的声音。可以就这么走下去吗？没有目的地，没有梦想，没有特别想要的与抛弃的，整个世界中只有你。

“抱歉呢，这家琴行好像关门了，害你走了这么远。”

“没关系，我很开心。”

开心么？我真的真的不想停下脚步，一直走，一直走，走到天涯海角，和你一起，模模糊糊又朦朦胧胧地走下去。

二

我站在楼下仰望夜空，突然觉得自己就像一片落叶，像一颗流星从一颗星滑到了另一颗星。也许走的太匆忙，打翻了我手中所有的时间。错了，又错了，还是错的。

“月亮很美吧。”我顺手拍了张照片发了过去。

“明天下雨，今晚会有月亮么？”

“月亮很难受。”

月亮难受地躲在厚厚的云层里，不愿出面。或许，月亮在流泪。

月亮会哭，但普鲁斯特会笑。似水流年啊，追忆无穷无尽。温暖的时光很短，却有可能一辈子记忆。

真的是一片黑暗，只有酒店依旧亮着残酷的灯。新华书店的门已经关闭，门前的漫咖啡馆里的人们也早已散去，只留下杂乱而伤感的桌椅。

三

“我们向哪走呢？”

“我们石头剪刀布吧，你赢了向前，我赢了左转。”

“石头剪刀布。”

“好，那我们向前走吧。”你调皮的眨了眨眼，捋了捋耳边飘起的长发。

“我喜欢出布，因为这能代表我海纳百川的包容之心，哈哈。”

“我喜欢出石头。”我顿了顿，却没有再说下去。

你停下脚步，撅起小嘴，傻傻地看着欲言又止的我，“为什么呢？”

傻瓜，我怎么可能会告诉你，因为我喜欢让你赢。

街边的发廊响起了简单爱。“我想就这样牵着你的手不放开，爱能不能够永远单纯没有悲哀，你靠着我的肩膀，你在我胸口睡着……”那我可以牵你的手么？你向远处走去，我在远处等你。

“我们再来一次吧。”

“嗯，石头剪刀布。”

“诶？为什么你会出布，你不是喜欢出石头的么？我还想出剪让你赢来着。”

“因为即使世界与你为敌，我都会包容你。”

若有可能，就请时间暂停吧，停在那一秒。我多么想侧身转向你，将你紧紧抱住。但理性将我拉回现实，告诉我不能这么做。

或许通过黑洞，人们终将发现白洞，发现白洞那边的新世界；或许通过虫洞，人们终将能够使时间流逝，能够以光速飞回千年前，飞到千年后。

可是我并不在乎，我只想与你寻找砌在墙里的传说，你与我被遗忘的姓名。

可是我并不在乎，我只在乎满地的太阳，我只在乎满天的泪滴。

四

我卧在沙发上，推送了一条消息。

“其实今晚这样，挺好”

又出现了一条新消息。

“其实今晚和你在一起，挺好。”

“没有杂念与目标。”

“机会很珍贵吧。”我叹了口气。

“我们算是朋友吗？”我不相信自己的眼睛，为什么你会问这么一个问题。你想要的答案是什么，你想要的答案我可以给么？

“嗯。”

“那样很好。”

五

“很好吗？”我走在路上，向你问道。

“对呀，虽然不是很清楚，但这条小街的感觉真的很好。”

气氛十分静谧，居民楼闪烁着忽明忽暗的灯光。人们纷纷迈着疲惫的步伐，回到各自的居所。

“你看他们的鲜花，终将枯萎。”你指着前方男人手中滴水的玫瑰，笑着小声告诉我。玫瑰失去了原色，花瓣失去了光泽。但只要送花的人心里火热，心里如春，即使枯萎也同样能拥有生机。

是的，玫瑰终将枯萎，花瓣终将凋谢。但我们的故事不会停止，继续，不停止，还会继续。靠着树干坐下，头顶满树韶光，树叶的罅隙里斜斜透着记忆，落满一地思念。醒来拍拍裤管，向不知名的地方走去。

故事戛然而止。

六

我站在楼下，趁着夜幕降临漆黑之时，顺手拍了张照片。浓厚的云层遮住了典雅的月光。

“月亮很美吧。”我把照片发给你。

“明天下雨，今晚会有月亮吗？”

“月亮很难受。”一阵无声的静寂。

我走上楼去站在你的门前，却迟迟不敢敲门，只得在楼道间废弃的沙发上坐着。

“其实今晚这样，挺好。”

“其实今晚和你在一起，挺好。”

“没有杂念与目标。”

“机会很珍贵吧。”我长长地叹了口气。

“我们算是朋友吗？”我怔了怔，不知如何回答。在那么一刻，我想回答：不，我们已经超过了朋友的关系。但我还是忍了下来，用颤抖的手指轻轻地敲出:

“嗯。”

“那样很好。”

很好吗？不，一点都不好，我不甘心！为什么不能鼓起勇气，大声地说出：

“不，我们已经超过了朋友的关系！”

退后，我们都忘了前进；停止，我们都忘了继续。

门，开了。

我拿出了写好的日记，慢慢地翻动着，你的目光也随之游离。

“我会哭，但普鲁斯特会笑。”

“是吗？月亮很难受。”

“笨蛋，我说的是晕车很难受。”你笑了笑。

“街边的魔术店还有刘谦的签名呢。”

“嗯，你还答应要教我变魔术呢。”你加了一句。

……

七

我喜欢走夜路，走那种很长很长的夜路，直到遇见了你。

你静静地看着我，站在马路的对面，我突然有一种冲动，想飞奔到你那边。只是，车闸缓缓拉升，红灯缓缓亮起，川流不息的车辆横亘在我们之间。

没有新海诚般的错过，没有张嘉佳般的告白，只有平静的点了点头，然后永不回眸的走了下去。

这是与我相知相识的你么？这是与我欢笑一路的你么？只是那个少年曾经的梦么？

这一切，好像都是梦。

我永远无法苏醒。

“文章以对话为主体不是很好哦，可能没有内容。”

“但是对话最能反映内容呢。”

“是这样子么？”

“是这样子吧。”

“但就像走夜路，如果只低着头，而忘记了景色，那又有什么意义呢？”

“如果我只注意了景色，却错过了你，那又有什么意义呢？”

你歪了歪头，若有所思。

从此，我不再走夜路。

岁月是那么的漫长，漫长到迷失自己，不明是非，愈合了伤痕累累，却仍旧不顾一切地走进伤痛；漫长到让人忘记自己是怎么走到这里的。但总有一天，总有一个人会在路口与你相遇，而后向新的地方义无反顾地走下去。

有些人适合陪伴在身边，一起厮守年华。有些人适合活在心里，时刻回想都能温暖时光，惊艳回忆。

也许当年的时光，就该搭配当年的我们，发生着那样的故事。再相见，哪怕物是人在，我们也不属于那个夜晚；再相见，哪怕桃花依旧，当年的欢乐也抓不住回不来。

我们已经记不起什么是惆怅。

“我们算是朋友吗？”

“不，我们已经超过了朋友的关系。”

“嗯，那样很好。我很幸运。”

你的幸运，我来不及。

The End
