思想观的变化录

Posted on 2021-11-13 In docs Views: Waline: Word count in article: 4.9k Reading time ≈ 9 mins.

近来一直想写篇这样的文章，一来算是整理自己读书带来的思想与价值之转变，二来是给自己过去的单一的思想和现在稍微成熟些的思想做个总结与反思。思想的变化总是与书和事分不开，所以我打算从两方面给自己做一个记录。

年少时思想总是很单纯，读书便为读书，心无杂念。相信很多人儿时都曾被问过这样一个问题，长大了想做些什么。这个问题问得好啊，问的既远，同时又寄托了家长的殷切期望。现在过年时偶尔也会有哥哥姐姐问着自家的小朋友，小朋友支支吾吾的说了句，我想当一个老师，因为可以管小朋友。在一片哄笑中他的妈妈或者老一辈的亲戚会告诉他，以后呀要好好学习，将来赚大钱。

其实相比“学而优则仕”这种传统观念来说，我以为现代观念已经有了长足之进步。不过学习读书在多数人眼里始终是与赚钱划等号，这可能与“读书改变命运”的思想观念密不可分。

印象中我小的时候也被问到过相似的问题，只不过答案已经记不清，或许从来就没有过答案，只是和别的小朋友说个一样的应付一下便完事。所以我现在认为当时的自己有无读书的目标是一件非常模棱两可的事。但我可以确定的事，无论儿时的我有没有说过将来想做什么，我以后将做的事都不可能是儿时口中玩笑般的美好憧憬。

其实直到上高中时，我都没有想明白为什么要读书为什么要上学。那时的我只知道，初中读完升入高中，高中读完进入大学，大学之后硕士博士，彷佛读书拿学位是一件顺其自然人人应做的事。成绩与排名带来的满足感和自信感也逐渐麻痹了我的心智，使我沉浸在可能算是“小我”的世界里。这里就像是一个舒适区，我不想轻易走出，因此思想上容易懒惰懈怠，也就懒得去想除了学习之外的事情。我这里指的学习，只是指应付高考而学习的六门功课。不过这也有可能是一个高考生最好的状态了吧？

后来有一天，我无意中想到曾经的那个问题：为什么要读书，为什么高一念完了念高二。这些问题困扰了我很久，也正是在那段时间我读了很多的书，从我以前的日记里我还能找到我对这些问题的记录与想法。最终，我形成了我自己的看法。读书的目的，是为了帮助更多的人，是为了有能力帮助更多的人。读书是获取知识的一种手段，拥有了知识即掌握了最重要的武器。至于高一高二之分，我觉得甚没必要，这只不过是为了阶段性教学能拥有一套较为完整的计划。知识毕竟不分高一高二，正如学无先后，达者为师。

可能正是处于那个意气风发的年纪，想问题也一根筋，认死理。不过我对读书学习最初的认识已经建立起来，现在开了上帝视角的我以为这种观念并没有偏离正轨。

进入大学之后，我频繁接受马列主义以及爱国主义的熏陶，一度立志为国做贡献。时至今日我还能够记得我与好友坐在食堂二楼的椅子上，告诉他，我今天所学，是为了国家将来所用，所以今日的我，定当努力学习。刚入校的我，每做任何一件事，每学任何一门课，每参加任何一次会议，我都会想这对当下的我和未来的我有什么帮助，而不是去想这会让我得到多高的分数。正是因为有着这种想法，我上半年的生活相当充实且正能量，成绩自然而然也不差。我始终以为成绩、分数、排名是在正确价值观指导下进行学习所带来的副产品，而不是追求的目标与学习的目的。

真希望后来的我也能一直能坚守这份初心。欲戴皇冠，必承其重，刚开学的我，什么外来之物都没有，无姓无名。一场考试之后，排名带给你的除了头上的光环，也有喘不过气的压力。我意识到要想继续做稳第一把交椅，就必须付出更多的努力。所以毫无疑问，下半年的收获很丰富，但生活非常的辛苦与劳累。我后来意识到，这是因为我学习的目的变了，我变得浮躁并且更加注重所谓的排名与名气。结合之前的经历，我顿时就感受到，名和利这两者就没一个是好东西。在此二者的长期驱使下有谁还能坚定的守住本心呢？无非是时间与多少的问题。从那时起我便坚定不从政的职业态度，即使我知道从政并不意味着贪污与腐败，但是这条路上会有数不尽的诱惑，甚至会有主动上门的诱惑和不得不收的诱惑。

进入大学之后，我能够很明显的感觉到自己的视野变得开阔，同时也会因为种种事情一次又一次的冲击着我看待读书观点的高楼大厦。后来有一天，我突然得知我的朋友的父亲去世。想到一个星期前还送我和朋友回学校的他，一个星期之后却不在了，我有些恍惚，有些不真实，我感觉我坚信的读书报国论受到了最强悍的一次冲击。

我们大部分人从小生活在一个提倡美化的环境中，提倡学习的目的是为了报效社会报效祖国，而对参杂个人因素的目的很多情况下都嗤之以鼻。不论学习的目的具体是什么，我想每个人都会有那么一段日子是真心把回报国家社会作为内心真实想法而努力，不过现实是绝大部分人最终还是会被生活所打倒，为茶米油盐而忙碌终生。

我想表达的和目标远不远大没关系，我想说的是，这种大规模的采样结果绝非偶然。当时我感触颇深，有病的人帮不了，有需要帮助的人帮不了，连想着去资助都要考虑各方面的压力与因素，学习究竟是为了什么。心里想着那么多宏伟的目标，可对于我来说今后能否给自己糊弄过去都是个问题，还怎么去谈这些空洞的目标。

以前看到水滴筹总是会捐一些，无论多少，可是现在也许都不会点开；以前看见生活困难的人总想着怎么样能尽自己的一份力帮助他们，可是现在也许也只会感叹一句生活不易。或许还有很多个这样的以前这样的现在，或许归于一句话就是看得多了，也许就麻木了。就算帮得了一两个，也帮不了那么多，心有余而力不足。

或许觉得能帮一个是一个，或许说学习能让你有能力去帮助更多的人。只不过对于这种话，我已经不再相信了，我感觉我甚至可能大部分人未来的十年基本已经确定，读研，也许读博，工作，没了。我甚至能够想象十年后的我会想些什么思考些什么。现实总是残酷的，我个人的意志和理想就是在两年半的大学时光里不断被消磨，曾经的满腔热血壮志凌云也不断的回归到现实生活，毕竟谁也无法逃离它，只活在自己天马行空的世界里。

记得曾经和一个家长交流时说到，做好自己的本职工作，不给社会添麻烦，在有能力的范围内关心别人，就算是为社会做贡献。当时觉得是最低标准，现在再想想觉得真的能做到其实还挺难。光是能力范围内就很难界定。

当我30岁40岁时，假设我能活到那么久并且我拥有独立生存或者养家糊口的能力时，我又有多少能力帮助别人，或者说以什么形式帮助呢。物质？又能提供多少物质帮助？精神？接受十几年甚至几十年教育的人就只能拍拍别人的肩膀说声加油或者节哀？这就是我纠结和难过的点啊。

我已经不止一次质疑我为什么要努力学习，只不过从来就没有答案。

我知道自己的能量首先得强大，才能坚守住初心，我也在努力的做，成绩算不上好但也绝对不差，可是一想到就算我做的再怎么出色，看到需要帮助的人我还是帮不上。生活从来不容易，只有相对轻松这一说法。也许我在别人眼中也在费力的乞讨着生活，但我怎么样不重要，重要的是，就像小红花里所描述的那些，我只能看着老奶奶孤独地等着她被拐走的孙子有一天能回来，只能看着抗癌战士一个个癌细胞被带走，只能看着学生在他这个年纪承担着生活的重担，连帮助时还怕触及内心而小心翼翼，只能看着他们费力的与生活斗争。

我很难过，也很抱歉。

从这时起，我的思想发生巨大的转变。我渴望在书籍里能够找到解决我类似心病的解药，我也渴望从师长从同辈间得到解答。我看了很多书也问了很多人，只不过从未得到答案。客观情况一直真实存在且发生，我深感无力阻止，更无力帮助，这是一种无奈的无力感。我的老师告诉我，有很多人在为他们而奋斗，我深以为然，然而我眼前看到的听到的却一直在告诉我，你还是没办法解决他们的困难，正如一屋不扫何以扫天下，眼前的困难都解决不掉，如何去谈更大更深层次的困难呢。

我怀揣着一种混沌的价值观浑浑噩噩的过了一年。这一年里我愈发感觉到世间事物与人们行为的可笑，也愈发的迷茫与不安。我和同辈多次交流思想，也从未有顿悟之感。终于，当我再次读完道德经之后，我选择了老庄思想作为我逃避现实的一个精神地。虚空的虚空，一切都是虚空，我深以为然。我逐渐学习着看淡对周遭人的评价与议论，看轻周围事件对我的潜在影响，我只希望能够和光同尘，知白守黑。那段时间我研读了相当多的老庄哲学以及空性哲学。

这样的状态维持了整整一年。这一年我不争不抢但也不惧，所有机会全部拱手让出，所有岗位凡是不指名道姓需要我皆不申请。也是在这一年我开始有点消极的思想，长期的这种所谓“无为”的处事风格让我觉得凡事都没有意义与价值，因为意义全部是人所赋予的，而人本身是否有意义尚未定论，又怎么能随意给物下定意义呢。

什么时候开始意识到不对劲呢，我想大约是最近几天吧。最近高考放榜让我想起了三年前的这个时候。突然思绪如潮水般涌上心间，不由得感慨万千，内心触动良多，满满的遗憾感随之而来。也正是这个时候，我突然意识到，我一直所接受并已经实践的老庄哲学空性哲学，它只是我的一个避风港，一个逃避现实的场所，一个安慰自己的精神乐园。这和吸食大麻又有什么区别呢。又联想到之前很多件告诉自己无关紧要的事，其实内心多少不还是有些波澜吗？我以为真正的放下与不争不抢，是那种完全不在乎，而不是内心会有起伏甚至是遗憾。

老庄思想不是一种积极的处世态度，我经历了一个从抗拒到接受的过程。时至今日，虽然我已从老庄哲学的安乐椅中站了出来，但不得不承认的是，空性哲学在我的身上留下了深刻的烙印，我的处事已经深受老庄哲学的影响。现在的我偶遇机会仍会拱手让出，因为“别人更需要”这种观念已经在我脑袋里扎根。这也许是一种崇高的思想，只不过生错了年代；被人借了三个月的碎银子不还，也就不要，因为不想因此撕破脸，更觉着这种身外之物与人相比甚是无所谓。

回顾短短20年思想变化的过程，我认为有三个重要的转折点。我自认为我具有强烈的悲悯心与同情心，这可能是引导我思想变化的最根本的原因。年少时不懂事，以为学习就是简单的读书争名次，后来逐渐接触到更加深层次的道理，明白了要想改变那些生活在水生火热之中的人们的生活，要想让人们能吃的上饭看得起病受的了教育有地方住，就要努力学习，这是我思想的第一个转折点。我以为这是最宏观的方向，我们每个人在做的事都是在为这个方向更进一步而努力。后来我终于感受到自己的渺小，自己的无能为力，自己没法给普天下所有需要帮助的人给予帮助，哪怕是一句安慰的话语都显得力不从心，因此我选择老庄哲学作为精神的避难所，这是我思想的第二个转折。第三个转折的时间就是最近了，我终于发现我所信仰依靠的哲学彷佛是也只是我的一个安乐花园，只是我不忍心面对现实众多苦难从而选择的一个逃避的场所。几年前的我因为高考改变了思想，谁又能想到几年后的我同样也是因为高考再一次转变了思想，一轮一回，冥冥之中，自有天意。

其实每次思想发生重大转变前夕，我都会非常的迷茫，三年前的我是这样，现在的我也是这样。我从未想到有这么一天我会否定我之前的行为，我会突然间的认为过去种种行为写满了“逃避”二字。每逢此情此景，一种颓废的荒唐感便会袭来全身。但也正是因为经历过荒唐感之后带来的那种玄之又玄的体会，我才相信是有“开悟”一说，我才能够相信我的思想变得更深层次。至于是进步还是退步，是先进还是落伍，是积极还是消极，是正面还是逃避，只有等以后再回头自我反思之后，才能下结论。

我现在正身陷于颓废的荒唐感之中，但我认为能够否定过去的思想行为便是种改变，虽然不一定能有所改观，毕竟行为习惯已经慢慢固定，思维模式亦然。我想起100多年前的人们探索救国道路的情景，从师夷长技以制夷，到百日维新，到新文化运动，再到马列主义，正确的道路就是在这种一步步地修正与改造中被发现的，所谓不破不立是矣。人的思想观也是，至少我已经接触过不同的主义与哲学，并不是说他们都不合适我，而是他们出现在了并不适合他们长期生存繁衍的、与应该形成这些思想观点的年龄不相符的我的脑袋里。在错误的时间错误的地方误打误撞遇见了也许是不合适的思想，多数人一辈子都是这样。所以一辈子上下求索，潜心问道，如是。

不知何时能拨云见日 ，甲光向日金陵开。我想我必会寻找到新的思想作为指引我的旗帜，而这一次，我相信我的思考将会更加成熟。
